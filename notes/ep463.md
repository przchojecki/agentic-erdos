# EP-463 partial attempt

## Route
Reformulated with `F(n)=\min_{m>n}(m-p(m))` and scanned finite windows to estimate both `n-F(n)` and a proxy for how far composites can extend beyond `n` while still satisfying `m-p(m)<n`.

## Evidence from this batch
- `data/ep463_Fn_gap_scan.json` (`n` up to `10^6`):
  - `n-F(n)` observed values: `30, 80, 122, 134, 186, 658` at checkpoints.
  - Positive finite-window proxy `g(n)` values also observed (`41` up to `733`).

## Hard point
Finite-window growth signals do not prove the required asymptotic existence of a diverging function `f(n)` that works for all sufficiently large `n`.

## Status
Heuristic positive signal, no proof.

## Deeper Proof Program (A => B => C)

### Target
Prove existence of diverging function \(f(n)\) controlling gap between \(n\) and
\[
F(n)=\min_{m>n}(m-p(m)).
\]

### A. Concrete Blocking Lemma (Most Critical)

Show that for every large \(n\), there exists \(m>n\) with
\[
p(m)\ge m-n+f(n),
\]
where \(f(n)\to\infty\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (equivalent gap form)
\[
n-F(n)=\max_{m>n}\bigl(p(m)-(m-n)\bigr)_+.
\]

**Proof.**
From \(F(n)=\min_{m>n}(m-p(m))\), subtract from \(n\) and rewrite as supremum of gains over \(n\). \(\square\)

#### Lemma B2 (witness transfer)
If some \(m>n\) satisfies \(p(m)\ge m-n+L\), then \(n-F(n)\ge L\).

**Proof.**
Then \(m-p(m)\le n-L\), so the minimum \(F(n)\le n-L\). \(\square\)

#### Lemma B3 (A implies divergence)
If A holds with \(f(n)\to\infty\), then \(n-F(n)\to\infty\).

**Proof.**
Apply B2 with \(L=f(n)\). \(\square\)

### C. Main unresolved point

The key obstacle is proving uniformly large least-prime-factor spikes sufficiently close above every large \(n\).
