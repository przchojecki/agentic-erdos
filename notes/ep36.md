## LaTeX Proofs for A => B => C Skeleton

Let \(F_N\) be the extremal overlap parameter at size \(N\), and \(c^\star\) the
candidate constant.

### Proposition E1 (Stability + template optimization => limit constant)
Assume:
1. (Upper stability) if \(F_N\le c^\star+\varepsilon\), then the configuration is
   \(o_\varepsilon(N)\)-close to a template family \(\mathcal T\);
2. (Template sharpness) \(\inf_{T\in\mathcal T} F_N(T)\to c^\star\).

Then
\[
F_N\to c^\star.
\]

\[
\textit{Proof.}
\]
Take near-extremizers \(X_N\) with \(F_N(X_N)\le \inf F_N+o(1)\). By (1), for
small \(\varepsilon\), \(X_N\) is close to some \(T_N\in\mathcal T\), hence
\[
F_N(X_N)\ge F_N(T_N)-o(1)\ge c^\star-o(1).
\]
By (2), there exist templates with value \(\le c^\star+o(1)\), so
\[
\inf F_N\le c^\star+o(1).
\]
Combine lower/upper bounds to get \(\inf F_N=c^\star+o(1)\), i.e. \(F_N\to c^\star\).
\(\square\)

### Proposition E2 (Constant extraction once template family is finite-dimensional)
If \(\mathcal T\) is parametrized by compact parameter set and objective is
continuous, then the optimizing constant \(c^\star\) exists.

\[
\textit{Proof.}
\]
Direct Weierstrass theorem.
\(\square\)

So open work is the stability theorem in E1(1), not the implication chain.
