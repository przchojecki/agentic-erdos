## Deeper Proof Program (A => B => C)

### Target
For graphs with edge density threshold in the EP-80 regime, force a linear-size
book \(B_t\) (many triangles sharing one edge) with explicit \(t\ge c n\).

### Step A: Stability decomposition
Show any near-extremal graph is close to a complete multipartite template plus a
small error set.

### Step B: Heavy-edge extraction
Prove that unless the error set is too large, some edge \(uv\) has
\[
|N(u)\cap N(v)|\ge c n,
\]
giving a linear book.

### Step C: Contradiction in the exceptional case
If all common-neighborhood sizes are \(o(n)\), derive a global triangle deficit
that contradicts the assumed density regime.

### Concrete Blocking Lemma (Most Critical)
Need a quantitative triangle-count-to-book-size transfer at the exact threshold
window; known supersaturation bounds are too coarse there.

## LaTeX Proofs for A => B => C Skeleton

### Proposition L1 (Edge-common-neighborhood criterion)
If a graph \(G\) on \(n\) vertices has an edge \(uv\) with
\[
|N(u)\cap N(v)|\ge t,
\]
then \(G\) contains a book of size at least \(t\).

\[
\textit{Proof.}
\]
For each \(w\in N(u)\cap N(v)\), triangle \(uvw\) exists. All these triangles
share edge \(uv\), yielding a book with \(t\) pages.
\(\square\)

### Proposition L2 (Reduction to a neighborhood lower bound)
To prove a linear book lower bound \(t\ge c n\), it is enough to show that every
graph in the target density class has some edge \(uv\) with
\[
|N(u)\cap N(v)|\ge c n.
\]

\[
\textit{Proof.}
\]
Immediate from Proposition L1.
\(\square\)
