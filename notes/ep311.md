# EP-311 partial

## Statement
`delta(N)` is the minimal nonzero value of
`|1 - sum_{n in A} 1/n|` over `A subseteq {1,...,N}`.
Is `delta(N)=exp(-(c+o(1))N)` for some `c in (0,1)`?

## Attempt in this batch
I computed exact `delta(N)` for small `N` via meet-in-the-middle on scaled
integer subset sums (`lcm(1..N)` denominator).

Data file:
- `data/ep311_delta_exact_small.json`

## Result
For `N=10..30`, exact values were obtained; empirical
`-log(delta(N))/N` stays around `0.54` to `0.65` in this range.

This supports exponential decay on finite scales.

## Hard point
Small-`N` exact values do not determine existence/uniqueness of an asymptotic
constant `c`.

## Deeper Proof Program (A => B => C)

### Target
Prove
\[
\delta(N)=\exp(-(c+o(1))N)
\]
for some constant \(c\in(0,1)\), or characterize limsup/liminf constants.

### Step A: Lattice subset-sum geometry
Embed scaled reciprocal sums into subset sums on integer lattice
\(\{0,1\}^N\) with common denominator \(L_N=\mathrm{lcm}(1,\dots,N)\).

### Step B: Exponential spacing bounds
Prove matching upper/lower bounds for minimum nonzero distance of subset sums to
\(L_N\).

### Step C: Constant extraction
Show normalized \(-\log\delta(N)/N\) has a limit.

### Concrete Blocking Lemma (Most Critical)
Need a sharp anti-concentration bound for near-cancellations of reciprocal subset
sums at scale \(\exp(-\Theta(N))\).

## Lemma Chain (with Proofs)

### Lemma 1 (Existence of positive minimum)
For fixed \(N\), set
\[
X_N=\left\{\left|1-\sum_{n\in A}\frac1n\right|:A\subseteq\{1,\dots,N\},\ 
\sum_{n\in A}\frac1n\ne1\right\}.
\]
Then \(\delta(N)=\min X_N>0\).

\[
\textit{Proof.}
\]
\(X_N\) is finite because there are finitely many subsets. All elements are
strictly positive by definition, so the minimum is positive.
\(\square\)

### Lemma 2 (Scaled integer formulation)
Let \(L_N=\mathrm{lcm}(1,\dots,N)\). Then
\[
\delta(N)=\frac{1}{L_N}\min_{A}\left|L_N-\sum_{n\in A}\frac{L_N}{n}\right|,
\]
where minimum is over subsets not giving exact equality.

\[
\textit{Proof.}
\]
Multiply each absolute difference by \(L_N\). Since \(L_N/n\in\mathbb Z\), inner
quantity is integer, and scaling is exact.
\(\square\)

### Lemma 3 (Submultiplicative envelope reduction)
If one proves for some constants \(c_1,c_2>0\):
\[
e^{-c_2N}\le \delta(N)\le e^{-c_1N}
\]
and a quasi-submultiplicative relation
\[
\delta(N+M)\le \delta(N)\delta(M)e^{o(N+M)},
\]
then \(-\log\delta(N)/N\) has limsup/liminf trapped in \([c_1,c_2]\), and limit
exists if matching quasi-supermultiplicativity also holds.

\[
\textit{Proof.}
\]
Take logs and apply generalized Fekete arguments to \(\log(1/\delta(N))\) with
additive \(o(N)\) errors.
\(\square\)
