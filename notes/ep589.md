# EP-589

## Problem Statement
For \(n\)-point sets in the plane with no four collinear points, determine the largest guaranteed size of a subset with no three collinear points.

## Literature / Context
- Existing bounds leave a wide gap (roughly between \(n^{1/2}\log n\)-type lower and sublinear upper scales).
- True asymptotic order is open.

## Our Approaches / What Is Proven
- Constructed no-4-collinear instances via iterative repair.
- From each instance, extracted large no-3-collinear subsets using a triple-hypergraph deletion heuristic.
- This provides finite lower-bound signals, not a proof of asymptotic order.

## Computation-Guided Observations
- Deep run outputs best subset ratios:
  - \(n=30: 0.93n\),
  - \(n=40: 0.875n\),
  - \(n=50: 0.82n\),
  - \(n=60: 0.767n\).
- Ratios decrease with \(n\), but remain large in this finite regime.

## Next Proof Targets
- Replace heuristic deletion by provable container/hypergraph bounds for the collinearity triple system.
- Quantify how no-4-collinear constraints limit triple density to force larger no-3-collinear subsets.
