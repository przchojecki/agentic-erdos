# EP-961 deeper attempt

## Statement
`f(k)` is one plus the maximum possible run length of consecutive `k`-smooth integers.
Estimate `f(k)`.

## Computation in this attempt
I scanned up to `X=6,000,000` and measured largest observed runs for selected `k`.

Data file:
- `data/ep961_smooth_run_scan.json`

Examples (observed run length `R`, so finite lower bound `f(k)>=R+1`):
- `k=100`: `R=13`, so `f(100)>=14`
- `k=500`: `R=19`, so `f(500)>=20`
- `k=1000`: `R=33`, so `f(1000)>=34`

## Interpretation
This provides finite lower bounds and scale data. No exact asymptotic proof is obtained.
