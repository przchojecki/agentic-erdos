## Deeper Proof Program (A => B => C)

### Target
Improve upper bounds for the maximum number \(u(n)\) of unit distances among
\(n\) planar points, ideally toward \(n^{1+o(1)}\) or better.

### Step A: Incidence partition
Partition point pairs by local directional/rich-line structure and treat each
part by tailored incidence inequalities.

### Step B: Polynomial partition + additive structure
Use partitioning to isolate cells, then show heavy cell interactions force
additive structure that limits unit-distance multiplicity.

### Step C: Global recombination
Sum cell and boundary contributions with balanced parameters to beat current
exponents.

### Concrete Blocking Lemma (Most Critical)
Need a robust anti-concentration theorem for unit vectors between large point
subsets after partitioning; current losses dominate.

## LaTeX Proofs for A => B => C Skeleton

### Proposition S1 (Incidence graph formulation)
Let \(P\) be an \(n\)-point set. Form bipartite graph \(G\subseteq P\times P\)
with \((p,q)\in E(G)\) iff \(|p-q|=1\), \(p\neq q\). Then
\[
u(n)=\frac12|E(G)|.
\]

\[
\textit{Proof.}
\]
Each unordered unit-distance pair corresponds to exactly two oriented edges.
\(\square\)

### Proposition S2 (Reduction via edge decomposition)
If \(E(G)=E_1\sqcup E_2\sqcup E_3\) and each class satisfies
\[
|E_i|\le B_i(n),
\]
then
\[
u(n)\le \frac12(B_1(n)+B_2(n)+B_3(n)).
\]
Hence any nontrivial improvement in the dominant \(B_i\) improves \(u(n)\).

\[
\textit{Proof.}
\]
Immediate from S1 and disjointness of classes.
\(\square\)
