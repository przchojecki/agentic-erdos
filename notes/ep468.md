# EP-468 deep attempt with standalone computation

## Statement
For each `n`, let `D_n` be partial sums of the increasing divisor sequence
`1<d_1<d_2<\cdots` of `n`.
Study `|D_n\\setminus \cup_{m<n} D_m|` and the inverse profile
`f(N)=\min\{n:N\in D_n\}`.

## Deeper Proof Program (A => B => C)

### Target
Prove asymptotic representability profile for \(D_n\) prefix-divisor sums and inverse function
\[
f(N)=\min\{n:\ N\in D_n\},
\]
in particular \(f(N)=o(N)\).

### A. Concrete Blocking Lemma (Most Critical)

Show a high-density covering lemma: union \(\bigcup_{n\le X} D_n\) covers all but \(o(X)\) integers up to \(X\), with quantitative novelty bound per \(n\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (inverse equivalence)
\[
f(N)\le X \iff N\in \bigcup_{n\le X} D_n.
\]

**Proof.**
By definition of minimal witnessing index. \(\square\)

#### Lemma B2 (coverage-to-growth transfer)
If uncovered count up to \(X\) is \(o(X)\), then for density-1 set of \(N\), \(f(N)\le X\) with \(X=(1+o(1))N\), and sharper coverage gives \(f(N)=o(N)\).

**Proof.**
Apply B1 and monotonicity with chosen \(X\)-scales; sparse uncovered set forces early representation for almost all \(N\). \(\square\)

#### Lemma B3 (A implies \(f(N)=o(N)\))
If A gives coverage at scales \(X=\eta N\) for every fixed \(\eta>0\) beyond thresholds, then \(f(N)\le \eta N\) eventually for represented \(N\), hence \(f(N)=o(N)\).

**Proof.**
Let \(\eta\downarrow0\) along sequence and use B1. \(\square\)

### C. Main unresolved point

The bottleneck is proving strong enough cumulative coverage from sparse per-\(n\) novelty increments.
