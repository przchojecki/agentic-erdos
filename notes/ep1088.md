# EP-1088 partial

## Statement
Estimate `f_d(n)`, the least `m` such that every `m` points in `R^d` contain
`n` points with all pairwise distances distinct.

## Attempt in this batch
I focused on `n=3`, where the property means existence of a scalene triangle.
I used the cross-polytope construction (`2d` points: `+-e_i`) and checked its
distance spectrum exactly.

Data file:
- `data/ep1088_ep1089_construction_scan.json`

## Result
Cross-polytope has exactly two distance values and no scalene triangle, so:
- `f_d(3) >= 2d+1` (explicit lower bound).

Random calibration at size `2d+1` had scalene-triple fraction `1.0` in tested
dimensions, indicating the construction is highly special.

## Hard point
This gives only a baseline lower bound for `n=3`.
The main open part is sharp growth for general `n` and large `d`.

## Deeper Proof Program (A => B => C)

### Target
Estimate \(f_d(n)\): minimum \(m\) such that every \(m\)-point set in \(\mathbb R^d\) contains \(n\) points with all pairwise distances distinct.

### A. Concrete Blocking Lemma (Most Critical)

Prove a Ramsey-type geometric statement: any sufficiently large \(m\)-point set in \(\mathbb R^d\) contains either
1) a large nearly equidistant structured block, or
2) an \(n\)-subset with pairwise distinct distances.

Quantitative control of case (1) size would yield explicit bounds on \(f_d(n)\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (distance-coloring translation)
Color each pair \(\{x,y\}\) by its distance value. Then an \(n\)-subset with all pairwise distances distinct is a rainbow \(K_n\) in this edge-colored complete graph.

**Proof.**
Pairwise distinct distances on a subset means exactly that all \(\binom n2\) edges have different colors. \(\square\)

#### Lemma B2 (lower bound from two-distance constructions)
If there exists an \(M\)-point set with at most two distance values, then
\[
f_d(3)\ge M+1.
\]

**Proof.**
Any 3-point subset uses at most two distances, so no rainbow triangle exists in that \(M\)-set. Hence at least one extra point is needed to force one. \(\square\)

#### Lemma B3 (general transfer)
If maximal size of a set using at most \(t\) distances in \(\mathbb R^d\) is \(M_t(d)\), then
\[
f_d(n)\ge M_{\binom n2-1}(d)+1.
\]

**Proof.**
A set with at most \(\binom n2-1\) global distance values cannot contain an \(n\)-subset with \(\binom n2\) distinct distances. \(\square\)

### C. Main unresolved point

The central difficulty is sharp upper bounds: geometric constraints can force many repeated distances, but current anti-Ramsey tools in Euclidean settings are far from optimal.
