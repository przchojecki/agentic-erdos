# EP-790 partial

## Statement
Let `l(n)` be the guaranteed size of a strong sum-free subset in every `n`-set
of integers (no relation `a_1 = a_2 + ... + a_r` with distinct elements in set).
Estimate `l(n)`.

## Attempt in this batch
I ran an exact surrogate model:
choose `A` from `[0..M]`, `|A|=n`, and compute the largest strong sum-free
subset of `A` by exhaustive search.

Data file:
- `data/ep790_interval_model_scan.json`

## Result
For tested `n`:
- `n=5`: model upper bound `l(n) <= 3`
- `n=6`: model upper bound `l(n) <= 4`
- `n=7`: model upper bound `l(n) <= 4`
- `n=8`: model upper bound `l(n) <= 4`

This supports sublinear behavior, but only in a restricted finite model.

## Hard point
The true problem ranges over all integer sets, not only subsets of short
intervals and not only small `n`.
