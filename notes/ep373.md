## Deeper Proof Program (A => B => C)

### Target
Classify solutions to
\[
n! = \prod_{j=1}^k a_j!,\qquad 2\le a_j < n,
\]
and in particular prove finiteness (or complete list) for fixed \(k\).

### A. Concrete Blocking Lemma (Most Critical)

For each fixed \(k\ge2\), prove an effective inequality
\[
\max_j a_j \le n - c_k \log n
\]
for all sufficiently large nontrivial solutions, with \(c_k>0\).

Combined with Stirling-expansion balancing, this would force an explicit contradiction for large \(n\), yielding finiteness.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (largest-factor normalization)
If \(a_k=\max_j a_j\), then \(a_k\le n-1\), and equality \(a_k=n-1\) forces
\[
n=\prod_{j<k} a_j!.
\]

**Proof.**
Each \(a_j<n\) by nontriviality. If \(a_k=n-1\), divide the identity by \((n-1)!\) to get \(n=\prod_{j<k}a_j!\). \(\square\)

#### Lemma B2 (prime-support constraint)
Let \(p\in (n/2,n]\) be prime. Then \(v_p(n!)=1\), hence exactly one factor \(a_j!\) has \(a_j\ge p\).

**Proof.**
Since \(p>n/2\), only one multiple of \(p\) lies in \([1,n]\), namely \(p\), so \(v_p(n!)=1\). In the product of factorials, this means exactly one factor contributes this single \(p\)-valuation. \(\square\)

#### Lemma B3 (effective finite search once A is known)
Assume Lemma A. Then for fixed \(k\), only finitely many \(n\) can satisfy the equation.

**Proof.**
Lemma A bounds all \(a_j\) away from \(n\) by at least \(c_k\log n\). Applying Stirling to both sides gives
\[
n\log n - n = \sum_j (a_j\log a_j-a_j)+O_k(\log n),
\]
and the right side misses enough top mass to violate equality for large \(n\). Hence only bounded \(n\) remain, thus finitely many solutions. \(\square\)

### C. Main unresolved point

The hard step is quantitative control of how close several \(a_j\) can simultaneously sit to \(n\) while matching all prime valuations of \(n!\). This is the exact point where known arguments lose uniformity.
