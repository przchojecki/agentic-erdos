# EP-156 partial attempt

## Route
Ran random-order greedy constructions of maximal Sidon sets in `[1..N]` and tracked size scaling.

## Evidence from this batch
- `data/ep156_maximal_sidon_random_scan.json` (`N=200..3200`, 600 trials each):
  - minimum found sizes: `11,15,20,27,36`;
  - ratios to `N^{1/3}` stay around `1.9` to `2.4`.

## Hard point
Finite/random constructions support the `N^{1/3}` scale but do not prove existence of maximal Sidon sets of size `O(N^{1/3})` for all large `N`.

## Status
Strong finite heuristic support; theorem-level existence question remains open.

## Deeper Proof Program (A => B => C)

### Target
Show existence of maximal Sidon subsets of \([1,N]\) of size \(O(N^{1/3})\), or
prove a larger necessary order.

### Step A: Saturation reformulation
A maximal Sidon set \(A\) is one where every \(x\notin A\) creates a repeated
sum when added. Encode this as a covering condition on forbidden sums.

### Step B: Counting lower envelope
Relate number of forbidden \(x\) values to \(|A|\) via pair-sum collision counts.

### Step C: Constructive upper envelope
Design explicit or probabilistic constructions meeting saturation with
\(|A|=O(N^{1/3})\).

### Concrete Blocking Lemma (Most Critical)
Need a saturation covering lemma showing each existing Sidon set of size
\(\gg N^{1/3}\) can be pruned/rearranged to a maximal set without large blow-up.

## LaTeX Proofs for A => B => C Skeleton

### Proposition BE1 (Maximality criterion)
A Sidon set \(A\subset[1,N]\) is maximal iff for every \(x\in[1,N]\setminus A\),
\(A\cup\{x\}\) is not Sidon.

\[
\textit{Proof.}
\]
This is exactly the definition of maximality under inclusion.
\(\square\)

### Proposition BE2 (Reduction to forbidden-value coverage)
If one proves that for every Sidon \(A\subset[1,N]\) with \(|A|=m\), the set of
forbidden \(x\) has size at most \(C m^3\), then maximality forces
\[
N \le |A| + C|A|^3,
\]
hence \(|A|\gg N^{1/3}\). Matching constructions at this scale resolve order.

\[
\textit{Proof.}
\]
By BE1, all \(x\notin A\) must be forbidden in a maximal set, so
\(N-|A|\le C|A|^3\).
\(\square\)
