## Deeper Proof Program (A => B => C)

### Target
For diameter-minimizing \(1\)-separated \(n\)-point sets in the plane, show the
number \(h(n)\) of incongruent minimizers satisfies \(h(n)\to\infty\).

### Step A: Stability class decomposition
Classify minimizers into finitely-parameterized geometric families up to
congruence.

### Step B: Independent deformation modes
Prove at least \(m(n)\to\infty\) binary/local deformation choices preserve
optimal diameter and separation constraints.

### Step C: Distinctness certification
Show different deformation codes yield non-congruent minimizers.

### Concrete Blocking Lemma (Most Critical)
Need a rigidity-with-defects theorem: near-boundary perturbations that preserve
optimality can be decoupled into many independent local choices.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AB1 (Code-counting lower bound)
Assume for each large \(n\) there is a family of optimal configurations
\(\{P_\sigma:\sigma\in\{0,1\}^{m(n)}\}\) such that \(P_\sigma\not\cong P_{\sigma'}\)
for \(\sigma\neq \sigma'\). Then
\[
h(n)\ge 2^{m(n)}.
\]

\[
\textit{Proof.}
\]
Each code gives a distinct congruence class of minimizers.
\(\square\)

### Proposition AB2 (Asymptotic consequence)
If \(m(n)\to\infty\), then \(h(n)\to\infty\).

\[
\textit{Proof.}
\]
From AB1, \(h(n)\ge 2^{m(n)}\), and \(2^{m(n)}\to\infty\).
\(\square\)
