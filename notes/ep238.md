# EP-238 partial resolution

## Statement
Given `c1,c2>0`, must every sufficiently large `x` have more than `c1 log x`
consecutive primes `<=x` whose pairwise gaps are all `>c2`?

## What is resolved
Background records Erdos proved this for any fixed `c2>0` when `c1` is chosen
sufficiently small.

## What remains open in this note
The uniform claim for arbitrary prescribed positive `c1,c2` is not settled by
the provided background.

## Status
- small-`c1` regime (for fixed `c2`): resolved.
- full parameter range: unresolved.

- In this finite range, many runs exceeded moderate logarithmic thresholds, with counts decreasing as `c_2` and `c_1` increased.
- Obstacle: the EP asks for universal asymptotics for all sufficiently large `x`; finite-range statistics cannot settle full parameter-uniform claims.

## Deeper Proof Program (A => B => C)

### Target
For fixed \(c_1,c_2>0\), decide when long runs of consecutive prime gaps
exceeding \(c_2\) force run length \(>c_1\log x\) for all large \(x\).

### Step A: Long-gap Bernoulli model with correction
Model events \((p_{n+1}-p_n)>c_2\) and quantify dependencies.

### Step B: Run-length large deviations
Prove lower/upper bounds for maximum run length among primes up to \(x\).

### Step C: Parameter map
Identify the admissible \((c_1,c_2)\)-region for guaranteed inequalities.

### Concrete Blocking Lemma (Most Critical)
Need uniform estimates for dependence between nearby long-gap events to justify
run-length asymptotics beyond independent heuristics.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CQ1 (Run-threshold equivalence)
Let \(R_x(c_2)\) be maximum length of consecutive prime gaps \(>c_2\) among
primes \(\le x\). Then the EP statement is equivalent to
\[
R_x(c_2)>c_1\log x
\]
for all large \(x\).

\[
\textit{Proof.}
\]
This is exactly the definition of the counted run condition.
\(\square\)

### Proposition CQ2 (Reduction)
If one proves
\[
R_x(c_2)\sim \kappa(c_2)\log x
\]
with explicit \(\kappa(c_2)\), then the inequality in CQ1 holds iff
\(c_1<\kappa(c_2)\).

\[
\textit{Proof.}
\]
Compare asymptotic coefficients.
\(\square\)
