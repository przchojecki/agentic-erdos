## Deeper Proof Program (A => B => C)

### Target
For each fixed \(k\), prove infinitude (or determine finiteness) of arithmetic
progressions of length \(k\) consisting entirely of primes.

### Step A: Linear forms majorant
Construct pseudorandom majorants for primes along AP templates.

### Step B: Relative Szemeredi/transference
Apply transference to obtain \(k\)-term APs in the prime set with quantitative
counts.

### Step C: Infinite repetition
Show count lower bounds remain positive for arbitrarily large scales.

### Concrete Blocking Lemma (Most Critical)
Need sufficiently uniform pseudorandomness estimates for the prime majorant at
complexity level matching \(k\)-term AP systems.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AT1 (Positive count implies existence)
If for infinitely many \(N\),
\[
\#\{(a,d): a,a+d,\dots,a+(k-1)d\le N,\ \text{all prime}\}>0,
\]
then there are infinitely many \(k\)-term prime APs.

\[
\textit{Proof.}
\]
Each such \(N\) contributes at least one progression with maximum term \(\le N\);
taking \(N\to\infty\) yields infinitely many distinct progressions.
\(\square\)

### Proposition AT2 (Reduction)
If one proves an asymptotic lower bound
\[
\#\text{k-AP primes up to }N \ge c_k \frac{N^2}{(\log N)^k}
\]
for all large \(N\), then EP-141 is resolved positively for that \(k\).

\[
\textit{Proof.}
\]
Right side is positive for large \(N\), then apply AT1.
\(\square\)
