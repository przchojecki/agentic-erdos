## Deeper Proof Program (A => B => C)

### Target
For fixed \(k\ge3\) and lacunary \(B\), decide when there exist bounded-gap sequences avoiding \(k\)-term progression/difference patterns induced by \(B\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a dichotomy criterion on \(B\): either a uniform modular obstruction exists forcing bounded-length avoidance sets, or no such obstruction exists and one can build arbitrarily long bounded-gap avoiders.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (finite-window obstruction inheritance)
If a residue-class obstruction modulo \(M\) blocks one admissible extension step, then periodic lifting blocks infinitely many steps.

**Proof.**
Constraint depends only on residues modulo \(M\). Once a residue is forbidden at one level, all integers in that class remain forbidden by periodicity. \(\square\)

#### Lemma B2 (bounded-gap to density proxy)
A sequence with gaps \(\le D\) up to \(N\) has lower density at least \(1/D\) on \([1,N]\) up to boundary terms.

**Proof.**
At most \(D\) integers per selected element in partition by forward intervals of length \(D\). \(\square\)

#### Lemma B3 (A implies qualitative classification)
If Lemma A gives necessary-and-sufficient modular obstruction criterion, then each \((k,B)\) case is decidable in principle.

**Proof.**
Check criterion; obstruction side gives impossibility, non-obstruction side gives constructive arbitrarily long (or infinite) bounded-gap avoiders. \(\square\)

### C. Main unresolved point

The main gap is proving that all long-range failures come from finite modular obstructions rather than genuinely aperiodic global phenomena.
