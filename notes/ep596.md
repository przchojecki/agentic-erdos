# EP-596

## Problem Statement
Characterize pairs \((G_1,G_2)\) exhibiting finite-color forcing but failure of the infinite-color analogue inside \(G_1\)-free graphs.

## Literature / Context
- Prior notes already record a concrete counterexample pair \((C_4,C_6)\), so nonexistence is false.
- Full classification over all pairs remains open.

## Our Approaches / What Is Proven
- Built a finite proxy focused on \((C_4,C_6)\):
  - construct dense \(C_4\)-free samples,
  - test whether 2-edge-colorings avoiding monochromatic \(C_6\) are found by local search.
- This only probes small finite behavior and cannot resolve the infinitary quantifiers.

## Computation-Guided Observations
- In tested dense \(C_4\)-free samples (\(n=14,16,18\)), avoiding monochromatic \(C_6\) in 2 colors was consistently found (probability \(1\) under current search budget).
- Finite evidence therefore shows no near-threshold forcing at these scales.

## Next Proof Targets
- Move from heuristic search to exact SAT/CP checks on selected extremal \(C_4\)-free instances.
- Try to identify finite structural witnesses that better reflect the known \((C_4,C_6)\) infinite-separation mechanism.
