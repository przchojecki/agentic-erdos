# EP-535 deeper attempt (r=3 finite proxy)

## Statement
Estimate maximal sets avoiding large equal-gcd sunflower-type structure.

## Exact finite proxy proved in this attempt
For the `r=3` proxy condition "no triple with equal pairwise gcd", I solved the exact
extremal size on `[1,n]` for `6<=n<=50` via hypergraph independence.

Data file:
- `data/ep535_r3_exact_small_hypergraph_scan.json`

Sample values:
- `n=32`: exact maximum `9`
- `n=50`: exact maximum `11`

## Interpretation
This is a rigorous finite proxy result; it does not resolve the full asymptotic problem for
general `r`.
