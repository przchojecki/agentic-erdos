# EP-1029 partial attempt

## Route
Tracked the normalization `R(k)/(k*2^{k/2})` against the known probabilistic lower bounds and classical upper bounds.

## What is resolved from background
- Current lower bounds have the same base scale `k*2^{k/2}` up to constant factors.
- The open claim asks that the normalized ratio tends to infinity, i.e. a superconstant gain over that scale.

## Hard point
No available method in this batch pushes beyond constant-factor improvements in the normalization.

## Status
Main divergence claim remains open.

## Rigorous baseline lower bound (probabilistic method)
Let \(R(k):=R(k,k)\).
For an integer \(n\), 2-color edges of \(K_n\) independently red/blue with probability \(1/2\).
Let \(X\) be the number of monochromatic \(K_k\)'s.

Then
\[
\mathbb E[X]
=2\binom{n}{k}2^{-\binom{k}{2}}
\le
2\left(\frac{en}{k}\right)^k 2^{-k(k-1)/2}.
\]

Choose
\[
n=\left\lfloor \frac{k}{4e}\,2^{k/2}\right\rfloor.
\]
Then
\[
\left(\frac{en}{k}\right)^k \le \left(2^{k/2-2}\right)^k=2^{k^2/2-2k},
\]
so
\[
\mathbb E[X]\le 2\cdot 2^{k^2/2-2k}\cdot 2^{-k(k-1)/2}
=2^{1-k}<1
\]
for \(k\ge2\).

Hence some 2-coloring has no monochromatic \(K_k\), so
\[
R(k)>\frac{k}{4e}\,2^{k/2}
\]
for all sufficiently large \(k\).

This proves a constant-factor \(k2^{k/2}\) lower bound.  
EP-1029 asks for a superconstant factor, i.e.
\[
\frac{R(k)}{k2^{k/2}}\to\infty,
\]
which is far beyond this baseline.
