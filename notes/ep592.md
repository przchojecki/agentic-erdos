# EP-592

## Problem Statement
Classify the corresponding infinitary partition relation in the unresolved regime (notably the remaining three-indecomposable-summand case in prior summaries).

## Literature / Context
- Prior notes indicate this problem is still open, with only a narrow structural case unresolved.
- Finite graph Ramsey tasks are only rough analogues, not faithful models of the full ordinal classification.

## Our Approaches / What Is Proven
- Built a deeper toy proxy: 3-color edge colorings avoiding monochromatic triangles, searched by local optimization.
- Purpose: stress-test threshold-like behavior near classical finite Ramsey ranges.

## Computation-Guided Observations
- In local search:
  - a zero-monochromatic-triangle coloring was found at \(n=12\) (rare in this heuristic),
  - none found for \(n=14,16,17\) in current attempt budgets, with best residual monochromatic-triangle counts increasing with \(n\).
- This is heuristic evidence only; it does not transfer to the ordinal open case.

## Next Proof Targets

- Track whether the remaining case can be reduced to a smaller canonical obstruction family.
