# EP-1083

## Problem Statement
For fixed $d\ge 3$, let $f_d(n)$ be the minimum number of distinct distances determined by any $n$-point subset of $\mathbb{R}^d$.
Estimate $f_d(n)$; in particular, is it true that
\[
f_d(n)=n^{\frac{2}{d}-o(1)}?
\]

## Literature
- Erdős: classical bounds
\[
n^{1/d}\ll_d f_d(n)\ll_d n^{2/d},
\]
with the upper bound from lattice constructions.
- For $d=3$: lower-bound improvements (Clarkson-Edelsbrunner-Guibas-Sharir-Welzl; Aronov-Pach-Sharir-Tardos; Solymosi-Vu and subsequent refinements).

## Our Approaches / What Is Proven
- Replaced batch stub with standalone large-scale lattice-box computation (`scripts/ep1083.mjs`).
- Computed exact distinct-distance counts for two families in $\mathbb{R}^3$:
  - cubes $(t,t,t)$,
  - slabs $(t,t,2t)$,
  across large $t$ values up to $500$.
- Added repeated exact passes per box (10 passes at deep setting) and checked deterministic stability.

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- $t\in\{100,150,200,250,300,350,400,450,500\}$,
- 2 families,
- 10 exact passes per box,
- total runtime about $15.16$ seconds.

Representative large-scale ratios:
- cube $(500,500,500)$:
  \[
  m=554119,\quad \frac{m}{n^{2/3}}\approx 2.216476;
  \]
- slab $(500,500,1000)$:
  \[
  m=1187036,\quad \frac{m}{n^{2/3}}\approx 2.9911432868.
  \]

All repeated passes were stable (identical counts).
This is finite construction-side evidence consistent with the lattice-type
\(n^{2/3}\)-scale in dimension $3$, but it is not an asymptotic proof.
