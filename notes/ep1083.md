# EP-1083

## Problem Statement
For fixed $d\ge 3$, let $f_d(n)$ be the minimum number of distinct distances determined by any $n$-point subset of $\mathbb{R}^d$.
Estimate $f_d(n)$; in particular, is it true that
\[
f_d(n)=n^{\frac{2}{d}-o(1)}?
\]

## Literature
- Erdős: classical bounds
\[
n^{1/d}\ll_d f_d(n)\ll_d n^{2/d},
\]
with the upper bound from lattice constructions.
- For $d=3$: lower-bound improvements (Clarkson-Edelsbrunner-Guibas-Sharir-Welzl; Aronov-Pach-Sharir-Tardos; Solymosi-Vu and subsequent refinements).

## Our Approaches / What Is Proven
- Replaced batch stub with standalone large-scale lattice-box computation (`scripts/ep1083.mjs`).
- Computed exact distinct-distance counts for two families in $\mathbb{R}^3$:
  - cubes $(t,t,t)$,
  - slabs $(t,t,2t)$,
  across large $t$ values up to $500$.
- Added repeated exact passes per box (10 passes at deep setting) and checked deterministic stability.

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- $t\in\{100,150,200,250,300,350,400,450,500\}$,
- 2 families,
- 10 exact passes per box,
- total runtime about $15.16$ seconds.

Representative large-scale ratios:
- cube $(500,500,500)$:
  \[
  m=554119,\quad \frac{m}{n^{2/3}}\approx 2.216476;
  \]
- slab $(500,500,1000)$:
  \[
  m=1187036,\quad \frac{m}{n^{2/3}}\approx 2.9911432868.
  \]

All repeated passes were stable (identical counts).
This is finite construction-side evidence consistent with the lattice-type
\(n^{2/3}\)-scale in dimension $3$, but it is not an asymptotic proof.

## Deeper Proof Program (A => B => C)

### Target
For fixed \(d\ge3\), determine the true order of
\[
f_d(n)=\min_{|X|=n} |\Delta(X)|,
\]
especially whether \(f_d(n)=n^{2/d-o(1)}\).

### A. Concrete Blocking Lemma (Most Critical)

Prove an incidence-energy lower bound giving
\[
|\Delta(X)| \gg n^{2/d-\varepsilon}
\]
for all large \(n\) and every \(X\subset \mathbb R^d\), with \(\varepsilon\to0\) via iteration.

This would match the lattice upper scale up to \(n^{o(1)}\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (distance-energy inequality)
Let \(r_X(t)\) be the number of ordered pairs in \(X^2\) at distance \(t\). Then
\[
\sum_t r_X(t)=n(n-1),\qquad
|\Delta(X)|\ge \frac{(n(n-1))^2}{\sum_t r_X(t)^2}.
\]

**Proof.**
First identity is counting ordered pairs by distance class. Second is Cauchy-Schwarz on the sequence \((r_X(t))_{t\in\Delta(X)}\). \(\square\)

#### Lemma B2 (target energy threshold)
If
\[
\sum_t r_X(t)^2 \ll n^{4-2/d+\varepsilon},
\]
then
\[
|\Delta(X)|\gg n^{2/d-\varepsilon}.
\]

**Proof.**
Insert the energy upper bound into Lemma B1. \(\square\)

#### Lemma B3 (A implies conjectural scale)
If Lemma A holds with arbitrarily small \(\varepsilon\), then \(f_d(n)=n^{2/d-o(1)}\).

**Proof.**
Upper bound \(f_d(n)\ll n^{2/d}\) is given by lattice constructions; combining with the lower bound from A and letting \(\varepsilon\downarrow0\) gives the claim. \(\square\)

### C. Main unresolved point

The bottleneck is controlling repeated-distance energy for worst-case point sets in \(\mathbb R^d\) at near-optimal exponent, beyond current incidence bounds.
