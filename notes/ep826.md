# EP-826

## Problem Statement
Are there infinitely many $n$ such that for all $k\ge 1$,
$$\tau(n+k)\ll k?$$

## Literature
- Listed as a stronger form of an earlier divisor-shift problem in the Erdos list.
- No complete proof/counterexample integrated in this repository yet.

## Our Approaches / What Is Proven
- Built a finite proxy scan with exact divisor counts up to $N_{\max}+K$.
- For each $n\le N_{\max}$, measured
  $$\max_{1\le k\le K}\frac{\tau(n+k)}{k}.$$
- Also measured a tail-only variant
  $$\max_{K_0\le k\le K}\frac{\tau(n+k)}{k},$$
  to reduce trivial small-$k$ domination.

## Computation-Guided Observations
- With $(N_{\max},K,K_0)=(40000,2000,50)$:
  - full metric best value is $2$ (dominated by $k=1$);
  - tail metric best value is about $0.1967$ at $n=11$.
- This highlights the difficulty of uniform-in-$k$ control: small shifts dominate finite constants.
