# EP-325

## Proof Attempts and Literature Notes

Let
\[
f_{k,3}(X):=\#\{n\le X:\ n=a^k+b^k+c^k\ \text{for some }a,b,c\in\mathbb Z_{\ge 0}\}.
\]
The question is whether
\[
f_{k,3}(X)\gg X^{3/k}
\]
for each fixed \(k\ge 3\).

### Local obstruction check (moduli search)
Deep residue scans show explicit non-representable congruence classes for several \(k\):
- \(k=3\): modulo \(9\), classes \(4,5\) are not sums of three nonnegative cubes.
- \(k=4\): strong 2-adic obstructions (e.g. many missing classes mod \(16,64\)).
- \(k=6\): multiple missing classes mod \(7,9,13,16,27,\dots\).

Interpretation:
- these obstructions rule out full density \(f_{k,3}(X)\sim X\) in general;
- they **do not** refute \(f_{k,3}(X)\gg X^{3/k}\), since \(X^{3/k}\) is much smaller than \(X\) for \(k>3\), and still linear for \(k=3\) with positive (not full) local density.

For \(k=3\), an extended modulus scan for \(2\le m\le 400\) gives minimum local
coverage ratio \(7/9\), attained at \(m\) divisible by \(9\). So congruence
obstructions alone leave a large admissible class set and cannot explain the much
smaller observed global density near \(0.10\); the main difficulty is global
distribution, not merely local admissibility.

## Proof route sharpened

### Energy-to-growth reduction (exact lemma)
Let \(A_N=\{0^k,1^k,\dots,N^k\}\), and define additive 3-fold energy
\[
E_{k,3}(N):=\#\{(a,b,c,a',b',c')\in [0,N]^6:\ a^k+b^k+c^k=a'^k+b'^k+c'^k\}.
\]
Then by Cauchy--Schwarz,
\[
|A_N+A_N+A_N|
\ge
\frac{|A_N|^6}{E_{k,3}(N)}.
\]
Since \(|A_N|=N+1\), any bound
\[
E_{k,3}(N)\ll N^{3+\delta}
\]
implies
\[
|A_N+A_N+A_N|\gg N^{3-\delta}.
\]
With \(X\asymp N^k\), this gives
\[
f_{k,3}(X)\gg X^{(3-\delta)/k}.
\]
Hence EP-325 would follow from proving \(E_{k,3}(N)\ll N^{3+o(1)}\).

This is the concrete bottleneck: controlling 6-variable collision counts for one-equation equal-sum structure.

### \(k=4\) analytic bottleneck (clean reduction)
Write \(r_{4,3}(n)=\#\{(a,b,c)\in\mathbb Z_{\ge0}^3:\ a^4+b^4+c^4=n\}\). Then
\[
\sum_{n\le X} r_{4,3}(n)\asymp X^{3/4},
\]
and by Cauchy--Schwarz
\[
f_{4,3}(X)\ge
\frac{\left(\sum_{n\le X} r_{4,3}(n)\right)^2}
{\sum_{n\le X} r_{4,3}(n)^2}
\asymp
\frac{X^{3/2}}{J_{3,4}(X^{1/4})},
\]
where \(J_{3,4}(B)\) counts
\[
a_1^4+a_2^4+a_3^4=b_1^4+b_2^4+b_3^4,\qquad a_i,b_i\le B.
\]
So proving \(J_{3,4}(B)\ll B^{3+\varepsilon}\) would yield
\[
f_{4,3}(X)\gg X^{3/4-\varepsilon}.
\]
This is the precise collision-count barrier.

### Correction to a common heuristic confusion
For \(k=4\), \(f_{4,3}(X)\) is at most the number of unordered triples
\((a,b,c)\) with \(a^4,b^4,c^4\le X\), hence
\[
f_{4,3}(X)\ll X^{3/4}.
\]
Therefore \(f_{4,3}(X)/X\to 0\) automatically. Local congruence obstructions
(mod \(16\), etc.) explain which classes are forbidden, but they are not the
primary reason the integer-density tends to zero; sparsity already follows from
the \(X^{3/4}\) scale.

## Deeper Proof Program (A => B => C)

### Target
Prove for fixed \(k\ge3\):
\[
f_{k,3}(X)\gg X^{3/k-\varepsilon}
\]
and ideally \(f_{k,3}(X)\gg X^{3/k}\).

### Step A: Representation-energy control
Set
\[
r(n)=\#\{(a,b,c)\in[0,N]^3:\ a^k+b^k+c^k=n\},\quad X\asymp N^k.
\]
Control \(E=\sum_n r(n)^2\).

### Step B: Cauchy lower bound
Use
\[
|S|\ge \frac{(\sum_n r(n))^2}{\sum_n r(n)^2}
\]
for \(S=\{a^k+b^k+c^k\}\).

### Step C: Translate \(N\)-scale to \(X\)-scale
From \(E\ll N^{3+\delta}\), deduce
\[
f_{k,3}(X)\gg X^{(3-\delta)/k}.
\]

### Concrete Blocking Lemma (Most Critical)
Need \(E\ll N^{3+o(1)}\) (or \(N^{3+\delta_k}\) with small \(\delta_k\)) for the
single-equation six-variable energy.

## Lemma Chain (with Proofs)

### Lemma 1 (Mass identity)
\[
\sum_n r(n)=(N+1)^3.
\]

\[
\textit{Proof.}
\]
Each triple \((a,b,c)\in[0,N]^3\) contributes exactly once to exactly one value
of \(n=a^k+b^k+c^k\).
\(\square\)

### Lemma 2 (Cauchy distinct-value lower bound)
Let \(M=|\{n:r(n)>0\}|\). Then
\[
M\ge \frac{(\sum_n r(n))^2}{\sum_n r(n)^2}.
\]

\[
\textit{Proof.}
\]
Cauchy-Schwarz:
\[
\left(\sum_n r(n)\right)^2
\le \left(\sum_n 1_{\{r(n)>0\}}\right)\left(\sum_n r(n)^2\right)
=M\sum_n r(n)^2.
\]
Rearrange.
\(\square\)

### Lemma 3 (Energy-to-exponent transfer)
If \(\sum_n r(n)^2\ll N^{3+\delta}\), then
\[
M\gg N^{3-\delta}.
\]

\[
\textit{Proof.}
\]
Insert Lemma 1 into Lemma 2:
\[
M\gg \frac{N^6}{N^{3+\delta}}=N^{3-\delta}.
\]
\(\square\)

### Lemma 4 (Scale conversion)
With \(X\asymp N^k\), Lemma 3 gives
\[
f_{k,3}(X)\gg X^{(3-\delta)/k}.
\]

\[
\textit{Proof.}
\]
Replace \(N\) by \(X^{1/k}\) in \(N^{3-\delta}\).
\(\square\)
