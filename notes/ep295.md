# EP-295 deeper attempt

## Statement
`k(N)` is the minimum number of unit fractions with denominators `>=N` summing to `1`.
Question: does `k(N)-(e-1)N -> infinity`?

## Rigorous partial result
If `1 = sum_{i=1}^k 1/n_i` with `N<=n_1<...<n_k`, then
`H_{N+k-1}-H_{N-1} >= 1`.
Hence `k(N) >= k_harm(N)`, where `k_harm(N)` is the minimal `k` with
`H_{N+k-1}-H_{N-1} >= 1`.

I computed this lower-bound curve up to `N=20000`.

Data file:
- `data/ep295_harmonic_lower_bound_scan.json`

Observed range of `k_harm(N) - (e-1)N` on `N<=20000`:
- min `~-0.8591`
- max `~0.1419`

## Interpretation
This gives a clean rigorous lower-bound mechanism, but by itself it does not prove divergence of
`k(N)-(e-1)N`.
