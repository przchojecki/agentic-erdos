# EP-1085

## Problem Statement
Let $f_d(n)$ be the maximum number of unit-distance pairs determined by an $n$-point set in $\mathbb{R}^d$.
Estimate $f_d(n)$.

## Literature
- Difficult low-dimensional cases are $d=2,3$.
- For $d=2$: current bounds are between a superlinear Erdős lower bound and the $O(n^{4/3})$ upper bound (Spencer-Szemerédi-Trotter).
- For $d=3$: known bounds are between roughly $n^{4/3}\log\log n$ and $n^{3/2}\beta(n)$.
- For $d\ge 4$, Lenz-type constructions and Erdős-Stone asymptotics yield a quadratic leading term
\[
\frac{p-1}{2p}n^2,
\]
with $p=\lfloor d/2\rfloor$.

## Our Approaches / What Is Proven
- Replaced stub with standalone deep construction script (`scripts/ep1085.mjs`).
- Computed finite construction profiles for:
  - 2D grids,
  - 3D grids,
  - 4D Lenz-type orthogonal-circle construction.
- For 4D, optimized split $n=a+b$ by exact formula and then performed brute-force pair-distance verification on large $n$ across repeated passes.

## Computation-Guided Observations
Deep run (`DEPTH=4`) included brute-force verification at
\[
n\in\{14000,18000,22000\}
\]
with $5$ passes each, totaling about
\[
2.509865\times 10^9
\]
pair checks.

Results:
- all brute-force counts matched exact formula,
- all repeated passes were stable,
- normalized ratio in 4D Lenz construction:
\[
\frac{f_4(n)}{n^2}\approx 0.25007,\ 0.25006,\ 0.25005
\]
for the tested $n$, consistent with approach to $1/4$.

For $d=2,3$, grid profiles were also computed as finite baselines; they show expected polynomial scaling patterns but do not improve known extremal exponents.

## Status
- Open in key low dimensions ($d=2,3$).
- Added deep standalone construction study with large explicit brute-force verification for the 4D Lenz family.

## Deeper Proof Program (A => B => C)

### Target
Determine the order of
\[
u_d(n):=\max_{|X|=n}\#\{(x,y)\in X^2:\ |x-y|=1\}
\]
for all fixed \(d\), especially \(d=2,3\).

### A. Concrete Blocking Lemma (Most Critical)

For \(d=3\), prove a uniform decomposition theorem: every near-extremal unit-distance graph is approximable by a bounded-complexity geometric template plus \(o(n^{3/2})\) error edges.

This would collapse the exponent gap between current lower and upper bounds.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (incidence graph viewpoint)
Unit pairs correspond to incidences between points and unit spheres centered at points:
\[
u_d(n)=I(P,\mathcal S)-n
\]
with \(|P|=|\mathcal S|=n\).

**Proof.**
\((x,y)\) is a unit pair iff \(x\) lies on the unit sphere centered at \(y\). Counting incidences includes \(x=y\) only when radius 0; here none, so the relation is exact up to ordered/unordered normalization conventions. \(\square\)

#### Lemma B2 (upper bound transfer)
Any incidence upper bound \(I(P,\mathcal S)\ll n^\alpha\) yields \(u_d(n)\ll n^\alpha\).

**Proof.**
Immediate from Lemma B1. \(\square\)

#### Lemma B3 (template stability implication)
If all near-extremizers are template-close with template optimum \(T_d(n)\), then
\[
u_d(n)=T_d(n)+o(T_d(n)).
\]

**Proof.**
Template closeness bounds the edge discrepancy by \(o(T_d(n))\), forcing asymptotic equality. \(\square\)

### C. Main unresolved point

The main gap is the absence of a structural stability theorem in low dimensions comparable to high-dimensional Lenz/Erdos-Stone behavior.
