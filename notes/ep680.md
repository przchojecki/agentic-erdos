# EP-680 partial attempt

## Route
Performed exhaustive finite scan for
`exists k: p(n+k) > k^2+1`
using least-prime-factor tables and full search over feasible `k` for each `n`.

## Evidence from this batch
- `data/ep680_lpf_threshold_scan.json` up to `n=10^6`:
  - only 7 failures found (`n=3,7,13,23,31,113,115`),
  - all other tested `n` satisfy the inequality.

## Hard point
Computation strongly suggests eventual truth but does not provide an unconditional asymptotic proof.

## Status
Very strong finite positive signal; theorem remains open.
