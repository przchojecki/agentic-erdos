# EP-1091

## Problem Statement
If $G$ is $K_4$-free with $\chi(G)=4$, must it contain an odd cycle with at least two diagonals?
More generally, if every subgraph on at most $r$ vertices is $3$-colorable, must there be an odd cycle with at least $f(r)$ diagonals for some $f(r)\to\infty$?

## Literature
- The first question is known to be true (Voss, 1982).
- The stronger quantitative variant remains open.
- The pentagonal wheel is a classical obstruction showing that forcing three diagonals is not automatic.

## Our Approaches / What Is Proven
- Implemented standalone deep graph-structure search (`scripts/ep1091.mjs`):
  - exact checks of $K_4$-freeness and $\chi\ge 4$,
  - odd-cycle diagonal measurement,
  - exact local-$r$ checks for $r\in\{5,6,7\}$,
  - randomized mutation search over 90 seconds starting from the Mycielski graph of $C_5$.

## Computation-Guided Observations
Deep run (`DEPTH=4`) found:
- pentagonal wheel $W_5$:
  - $K_4$-free, $\chi\ge4$,
  - max odd-cycle diagonals = 2,
  - local-$r$ condition fails at $r\ge 6$.
- Mycielski($C_5$):
  - $K_4$-free, $\chi\ge4$,
  - max odd-cycle diagonals = 9,
  - passes local-$r$ checks for $r=5,6,7$.
- best mutated graph found in 90s (2,351,783 iterations):
  - still $K_4$-free and $\chi\ge4$,
  - max odd-cycle diagonals reduced to 2,
  - local-$r$ fails at $r\ge 6$.

Interpretation:
- finite search supports the known first-question threshold (2 diagonals appears tight in small examples),
- but did not yield a small-diagonal example that also satisfies stronger local-$r$ constraints for larger $r$.

## Status
- First subquestion: resolved in literature.
- Quantitative $f(r)$ question: open.
- Added deep standalone structural search with explicit local-$r$ diagnostics.

## Added Proof Layer (No New Computation)

### First Subquestion (Closed)
Claim: If $G$ is $K_4$-free and $\chi(G)=4$, then $G$ contains an odd cycle with
at least two diagonals.

This is exactly the theorem proved by Voss (1982) cited in the background.
Hence the first subquestion is fully settled.

### Remaining Open Part
For the local condition
\[
\chi(H)\le 3\ \text{for every }|V(H)|\le r,
\]
the asked conclusion is existence of an odd cycle with at least $f(r)$ diagonals
with $f(r)\to\infty$.

### Blocking Lemma
- Need a quantitative local-to-global forcing theorem: from $r$-local
  3-colorability to a lower bound on odd-cycle chord complexity that diverges
  with $r$.

Current known examples show small fixed diagonal counts can persist when the
local condition is weakened, so the monotone divergence mechanism is the missing
step.

## Deeper Proof Program (A => B => C)

### Target
Construct \(f(r)\to\infty\) such that every graph with all \(\le r\)-vertex
subgraphs 3-colorable contains an odd cycle with at least \(f(r)\) diagonals.

### Step A: Local-Colorability to Forbidden-Core Bound
Show \(r\)-local 3-colorability excludes a finite family of dense odd-cycle
chord cores up to scale \(g(r)\to\infty\).

### Step B: Core Exclusion to Chord Inflation
Prove any \(K_4\)-free 4-chromatic graph avoiding those cores must have large
odd-cycle chord count in every critical odd cycle.

### Step C: Extract Divergence Function
Quantify Step B to obtain explicit \(f(r)\) with \(f(r)\to\infty\).

### Concrete Blocking Lemma (Most Critical)
Need a quantitative critical-graph structure theorem: in 4-critical \(K_4\)-free
graphs, bounded odd-cycle diagonal complexity forces a bounded-size witness
subgraph violating local 3-colorability.
