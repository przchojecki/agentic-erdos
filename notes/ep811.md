# EP-811

## Problem Statement
Suppose $n\equiv 1\pmod m$. Call an edge-coloring of $K_n$ with $m$ colors balanced if every vertex sees exactly $(n-1)/m$ incident edges in each color class. For which graphs $G$ is it true that, with $m=e(G)$, every sufficiently large balanced $m$-coloring of $K_n$ (with $n\equiv 1\pmod m$) contains a rainbow copy of $G$?

## Literature
- Attributed by Erdos to work with Pyber and Tuza.
- Discussed in Erdos-Tuza (1993) and subsequent Erdos problem lists.
- Classical challenge case mentioned in the notes: balanced 6-colorings of $K_{6t+1}$ and rainbow forcing for 6-edge graphs.

## Our Approaches / What Is Proven
- Built a structured balanced coloring model on $K_{13}$ using cyclic distance classes (6 colors, perfectly balanced at each vertex).
- Checked multiple 6-edge target patterns under injective search: $K_4$, $C_6$, $P_7$, and $K_{1,6}$.
- This is finite constructive evidence only; no asymptotic theorem is proved.

## Computation-Guided Observations
- The tested balanced coloring has exact per-vertex profile $(2,2,2,2,2,2)$.
- Rainbow copies were found for all tested 6-edge patterns.
- This explicit balanced family is not a counterexample to rainbow forcing for these patterns.
