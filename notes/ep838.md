# EP-838 partial

## Statement
`f(n)` is the minimum number of convex subsets determined by any `n` points in
`R^2` with no three collinear. Estimate `f(n)`.

## Attempt in this batch
I ran finite sampling on point sets and counted convex subsets exactly for each
sample (`n=10,12,14,16,18`), comparing random sets, double-chain constructions,
and full convex-position sets.

Data file:
- `data/ep838_convex_subset_scan.json`

## Result
For each tested `n`, double-chain had fewer convex subsets than the best random
sample in this run, and both were far below convex-position counts. Example:
- `n=18`: best random `4960`, double-chain `3727`, convex-position `261972`.

This gives explicit finite upper-bound witnesses on `f(n)` in this range.

## Hard point
Sampling only gives upper bounds and does not address the asymptotic constant in
`log f(n) / (log n)^2`.
