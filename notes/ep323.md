# EP-323 partial resolution

## Statement split
1. Is `f_{k,k}(x) >>_epsilon x^{1-epsilon}` for every `epsilon>0`?
2. For `m<k`, is `f_{k,m}(x) >> x^{m/k}` for all large `x`?

## What is resolved
Background records a full solution of the `k=2` diagonal case:

`f_{2,2}(x) ~ c x / sqrt(log x)`

(Landau, with `c>0`).

So for `k=2` the diagonal quantity is understood asymptotically.

## What remains open in this note
For `k>2`, background says even the coarse question `f_{k,k}(x)=o(x)` is not known.
Thus the stated lower-bound targets are unresolved in general. The `m<k` question is
also open in general in the provided record.

## Added Proof Layer (No New Computation)

### Closed Subcase
For \(k=2\), the diagonal counting function satisfies
\[
f_{2,2}(x)\sim c\,\frac{x}{\sqrt{\log x}}
\]
(Landau, cited in background), so the \(k=2\) diagonal problem is fully settled.

### Remaining Open Core
For \(k>2\), even coarse diagonal asymptotics are unresolved in this note's
record, and for \(m<k\) the uniform lower bound
\[
f_{k,m}(x)\gg x^{m/k}
\]
is not proved in general.

### Blocking Lemma(s)
- Need a minor-arc cancellation bound strong enough to keep collision losses
  below the natural \(x^{m/k}\) main scale.
- Need a uniform major-arc lower-envelope theorem for sums of \(m\) \(k\)-th
  powers that survives over all sufficiently large \(x\).

## Deeper Proof Program (A => B => C)

### Target
For \(m<k\), prove
\[
f_{k,m}(x)\gg x^{m/k},
\]
and clarify diagonal behavior for \(m=k>2\).

### Step A: Major-Arc Main Term
Establish explicit lower main term for representable numbers using singular
series/integral on dominant arcs, uniform in dyadic \(x\)-ranges.

### Step B: Collision/Exceptional Set Bound
Bound overlap multiplicities and minor-arc contribution so exceptional set size
is \(o(x^{m/k})\) in subdiagonal regime.

### Step C: Uniform Lower Envelope
Patch dyadic ranges to obtain all-large-\(x\) lower bound with stable constant.

### Concrete Blocking Lemma (Most Critical)
Need a hybrid minor-arc + determinant-method estimate:
\[
N_{\text{collisions}}(x)\le C x^{m/k-\eta}
\]
for some \(\eta>0\), uniformly for fixed \((k,m)\) with \(m<k\).
