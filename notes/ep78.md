## Deeper Proof Program (A => B => C)

### Target
Produce explicit 2-colorings showing
\[
R(k,k)\ge C^k
\]
for some absolute \(C>1\) via deterministic constructions.

### Step A: Explicit pseudorandom family
Construct deterministic colorings \(\mathcal G_k\) with strong discrepancy and
small clique-count moments.

### Step B: Certified clique suppression
Prove
\[
N_{\mathrm{mono}}(K_k;\mathcal G_k)\le \exp(o(k))\cdot \mathbb E_{\mathrm{rand}}
[N_{\mathrm{mono}}(K_k)],
\]
and force this below \(1\) at \(n=\lfloor C^k\rfloor\).

### Step C: Deterministic existence
Conclude \(N_{\mathrm{mono}}(K_k;\mathcal G_k)=0\), hence
\[
R(k,k)>C^k.
\]

### Concrete Blocking Lemma (Most Critical)
Need a high-moment bound for monochromatic \(K_k\) counts in explicit families
that is uniform in \(k\); current explicit constructions control only lower
moments or restricted \(k\)-ranges.

## LaTeX Proofs for A => B => C Skeleton

### Proposition K1 (Expectation transfer criterion)
Fix \(n=n(k)\). If an explicit family \(\mathcal G_k\) satisfies
\[
N_{\mathrm{mono}}(K_k;\mathcal G_k)\le M_k\cdot
\binom{n}{k}2^{1-\binom{k}{2}},
\]
with \(M_k=\exp(o(k))\), and if
\[
\binom{n}{k}2^{1-\binom{k}{2}}\le \exp(-\omega(k)),
\]
then for large \(k\),
\[
N_{\mathrm{mono}}(K_k;\mathcal G_k)=0.
\]

\[
\textit{Proof.}
\]
The right side is at most \(\exp(o(k)-\omega(k))=o(1)\). Since
\(N_{\mathrm{mono}}(K_k;\mathcal G_k)\) is an integer, it must be zero for all
sufficiently large \(k\).
\(\square\)

### Proposition K2 (Exponential constructive lower bound)
If Proposition K1 holds for \(n=\lfloor C^k\rfloor\), then
\[
R(k,k)\ge C^k
\]
for all large \(k\) (up to floor constants).

\[
\textit{Proof.}
\]
From Proposition K1, \(\mathcal G_k\) has no monochromatic \(K_k\) on
\(\lfloor C^k\rfloor\) vertices, so \(R(k,k)\) exceeds that order.
\(\square\)
