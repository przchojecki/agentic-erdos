# EP-1108 partial attempt

## Route
Used factorial-base valuation checks and finite exhaustive search on subset sums of `{1!,...,10!}`.

## Evidence from this batch
- `data/ep1108_factorial_subset_scan.json` scanned `1023` subset sums from the first `10` factorials.
- Found `22` powerful values and `17` perfect powers in this finite prefix.

## What is resolved
Background already gives a partial theorem (Brindza-Erdos 1991): for fixed term count `r`, if `n_1!+...+n_r!` is powerful then `n_1` is bounded in terms of `r`.

## Hard point
The actual problem allows arbitrary finite subsets; controlling `p`-adic valuations and carry interactions uniformly as the number of terms grows is the key obstruction.

## Status
Finite-prefix data and fixed-`r` partial theory recorded; global finiteness remains open.
