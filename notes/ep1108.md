# EP-1108 partial attempt

## Route
Used factorial-base valuation checks and finite exhaustive search on subset sums of `{1!,...,10!}`.

## Evidence from this batch
- `data/ep1108_factorial_subset_scan.json` scanned `1023` subset sums from the first `10` factorials.
- Found `22` powerful values and `17` perfect powers in this finite prefix.

## What is resolved
Background already gives a partial theorem (Brindza-Erdos 1991): for fixed term count `r`, if `n_1!+...+n_r!` is powerful then `n_1` is bounded in terms of `r`.

## Hard point
The actual problem allows arbitrary finite subsets; controlling `p`-adic valuations and carry interactions uniformly as the number of terms grows is the key obstruction.

## Status
Finite-prefix data and fixed-`r` partial theory recorded; global finiteness remains open.

## Deeper Proof Program (A => B => C)

### Target
Understand finiteness/infinitude of powerful (or perfect-power) values among sums of distinct factorials
\[
\sum_{m\in S} m!,\qquad S\subset\mathbb N\ \text{finite}.
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a uniform \(p\)-adic separation lemma: if
\[
\sum_{m\in S} m!
\]
is powerful, then the maximal index \(M=\max S\) is bounded by an explicit function of the number of "valuation-collision layers" (independent of \(|S|\)).

This would move beyond fixed-\(r\) bounds and imply global finiteness.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (top-factorial isolation)
Write \(M=\max S\). Then
\[
\sum_{m\in S} m! = M!\left(1+\sum_{m<M,\ m\in S}\frac1{(m+1)\cdots M}\right).
\]

**Proof.**
Factor out \(M!\) from each term \(m!\). \(\square\)

#### Lemma B2 (valuation rigidity at large primes)
For prime \(\ell\in(M/2,M]\), \(v_\ell(M!)=1\), so powerfulness forces cancellation constraints on the parenthesized rational factor in Lemma B1 modulo \(\ell\).

**Proof.**
Exactly one multiple of \(\ell\) lies in \([1,M]\), so \(v_\ell(M!)=1\). A powerful integer must have exponent either \(0\) or \(\ge2\) at \(\ell\), so the remaining factor must contribute an extra \(\ell\)-divisibility whenever \(\ell\) divides the whole sum. \(\square\)

#### Lemma B3 (A implies global finiteness)
If Lemma A bounds \(M\), then only finitely many subsets \(S\) can occur, hence finitely many powerful sums.

**Proof.**
There are finitely many subsets of \(\{1,\dots,M\}\). \(\square\)

### C. Main unresolved point

The bottleneck is controlling simultaneous valuation interactions over many primes near \(M\) without fixing the number of summands.
