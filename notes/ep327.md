# EP-327 partial resolution

## Statement
For `A subseteq {1,...,N}` with pairwise divisibility constraint
`a+b | ab` (or variant `a+b | 2ab`), can `|A|` be substantially above the odd
numbers, and in the second variant must `|A|=o(N)`?

## What is resolved
Background records an elementary density upper bound for the first constraint:
if `|A| >= (25/28+o(1))N`, then forbidden pairs must occur.

## What remains open in this note
This does not determine the true maximum density for the first constraint, and
the second variant remains open in the cited text.

## Status
- nontrivial upper bound for first constraint: known.
- exact density and second-variant asymptotics: unresolved.

## New Experiments
- 2026-03-05T14:37:26.973Z: enforced deep standalone compatibility search (exact small N + heuristic large N with RESTARTS=450000; runtime 25.04s).
- Finite signal: both variants produced very sparse best sets in tested ranges (exact max size 3 at N=70; heuristic best size 5 at N=1800, density about 0.00278), far below odd-density baseline.
- Proof direction update: computational witnesses are highly multiplicative/chain-structured, suggesting extremal families may be governed by divisibility lattices rather than dense parity-type constructions.

## Deeper Proof Program (A => B => C)

### Target
Determine maximal density for sets \(A\subseteq[1,N]\) under
\[
a+b\mid ab
\]
and the variant \(a+b\mid 2ab\), including whether the second gives \(|A|=o(N)\).

### Step A: gcd-normalized equation
For \(a=dx,\ b=dy,\ \gcd(x,y)=1\), rewrite divisibility constraints in terms of
\(x,y,d\) to expose rigid multiplicative structure.

### Step B: Pair-structure classification
Classify admissible pairs \((a,b)\) and show high-density sets must contain many
forbidden-type pair interactions.

### Step C: Global counting contradiction
Use graph/counting arguments to bound \(|A|\), and in variant 2 drive density to
zero if possible.

### Concrete Blocking Lemma (Most Critical)
Need a pair-count lemma: each \(a\in A\) can be compatible with only \(O_\varepsilon(a^\varepsilon)\)
many larger \(b\in A\) on average, yielding sublinear total size.

## Lemma Chain (with Proofs)

### Lemma 1 (Normalized divisibility form)
If \(a=dx,\ b=dy\) with \(\gcd(x,y)=1\), then
\[
a+b\mid ab \iff x+y\mid d.
\]

\[
\textit{Proof.}
\]
Compute:
\[
a+b=d(x+y),\quad ab=d^2xy.
\]
Condition \(d(x+y)\mid d^2xy\) is equivalent to \(x+y\mid dxy\). Since
\(\gcd(x+y,xy)=1\) for coprime \(x,y\), this is equivalent to \(x+y\mid d\).
\(\square\)

### Lemma 2 (Immediate sparsity consequence for fixed \(d\))
For fixed \(d\), admissible coprime pairs \((x,y)\) must satisfy \(x+y\mid d\),
hence \(x+y\le d\). Therefore only finitely many normalized pair types occur for
that \(d\).

\[
\textit{Proof.}
\]
From Lemma 1, divisor condition implies \(x+y\) is a positive divisor of \(d\).
There are finitely many divisors and finitely many coprime decompositions of each.
\(\square\)

### Lemma 3 (Reduction to compatibility graph bound)
Let \(G_A\) be graph on \(A\) with edges for compatible pairs. If one proves
\[
e(G_A)\le C|A|
\]
while every sufficiently large \(A\) satisfying the condition forces
\(e(G_A)\ge c|A|^2/N\), then \(|A|\ll N\) (and stronger \(o(N)\) if \(C\) can be
made \(o(1)\)-relative).

\[
\textit{Proof.}
\]
Combine inequalities:
\[
c|A|^2/N\le C|A|\implies |A|\le (C/c)N.
\]
If \(C=C(N)=o(1)\) in the normalized variant, obtain \(o(N)\).
\(\square\)
