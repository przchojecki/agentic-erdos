## LaTeX Proofs for A => B => C Skeleton

Let \(C_s(X)\) be maximal covered count in \(I_X=[X/2,X]\) using at most \(s\)
allowed square-shifts from the EP-33 model.

### Proposition D1 (Threshold implication)
If there exists \(s_0\) such that for all large \(X\),
\[
C_{s_0}(X)=|I_X|,
\]
then EP-33 has a positive answer with bounded shift budget \(s_0\).

\[
\textit{Proof.}
\]
The identity states exact full coverage of each large interval \(I_X\), which is
precisely the required eventual-coverage statement with fixed budget.
\(\square\)

### Proposition D2 (Lower-density obstruction)
If for every fixed \(s\) there exists \(\eta_s>0\) and infinitely many \(X\) with
\[
C_s(X)\le (1-\eta_s)|I_X|,
\]
then no fixed \(s\) can solve EP-33.

\[
\textit{Proof.}
\]
A fixed \(s\) giving eventual full coverage would force \(C_s(X)=|I_X|\) for all
sufficiently large \(X\), contradicting the infinitely many deficient \(X\).
\(\square\)

Thus the unresolved mathematical task is to prove either D1 (constructive) or D2
(obstruction) at theorem level.
