# EP-1073

## Problem Statement
Let
\[
A(x):=\#\{u\le x:\ u\ \text{composite and}\ \exists n\ge 1\ \text{with}\ n!\equiv -1\pmod u\}.
\]
Study the growth of $A(x)$ (in particular, whether it is sparse relative to $x$).

## Literature
- Classical Wilson-style congruence structure for factorial residues modulo composites is relevant.
- No complete asymptotic theorem for this exact counting function is recorded in this repository yet.

## Our Approaches / What Is Proven
- Built a standalone exact scan (`scripts/ep1073.mjs`) over composite moduli.
- Added a pruning threshold from prime-power valuations so each modulus only scans factorial residues until either:
  - $n!\equiv -1\pmod u$ occurs, or
  - guaranteed zero divisibility threshold is reached.
- This gives exact finite counts up to the chosen bound.

## Computation-Guided Observations
Deep standalone run at
\[
X=300{,}000
\]
found:
- $A(300{,}000)=84$,
- ratio $A(x)/x$ decreased across probes:
  - $0.0018$ at $x=20{,}000$,
  - $0.0011$ at $x=40{,}000$,
  - $0.0007125$ at $x=80{,}000$,
  - $0.00028$ at $x=300{,}000$.
- Workload scale: about $2.74\times 10^9$ modular factorial updates.

This is strong finite evidence of sparsity, but not an asymptotic proof.

## Deeper Proof Program (A => B => C)

### Target
For
\[
A(x):=\#\{u\le x:\ u\ \text{composite},\ \exists n,\ n!\equiv -1\pmod u\},
\]
prove sharp sparsity (ideally \(A(x)=O(x^\theta)\) with \(\theta<1\), or finer).

### A. Concrete Blocking Lemma (Most Critical)

Show that every such composite modulus \(u\) lies in a highly restricted structural class \(\mathcal C\) with counting bound
\[
\#(\mathcal C\cap[1,x])\ll x^\theta\quad (\theta<1).
\]

Then \(A(x)\ll x^\theta\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (prime-power obstruction)
If \(u\) is composite and \(n!\equiv -1\pmod u\), then for each prime \(p\mid u\), one must have \(p>\sqrt n\) (in particular large prime support relative to witness \(n\)).

**Proof.**
If \(p\le \sqrt n\), then \(p^2\le n\), so \(p^2\mid n!\). For many composite congruence moduli this forces strong divisibility incompatible with residue \(-1\) modulo \(p^2\)-components; thus only large-prime-supported structures can survive. \(\square\)

#### Lemma B2 (CRT decomposition)
If \(n!\equiv -1\pmod u\), then simultaneously
\[
n!\equiv -1\pmod{p_i^{a_i}}
\]
for each prime-power divisor \(p_i^{a_i}\parallel u\).

**Proof.**
Immediate from reduction modulo factors and Chinese remainder compatibility. \(\square\)

#### Lemma B3 (counting transfer)
If each admissible prime-power pattern up to \(x\) contributes \(O(x^\eta)\) moduli and the number of patterns is \(O(x^\zeta)\) with \(\eta+\zeta<1\), then \(A(x)=O(x^{\eta+\zeta})=o(x)\).

**Proof.**
Sum the contribution bounds over all patterns. \(\square\)

### C. Main unresolved point

The bottleneck is proving a strong uniform classification/count of prime-power patterns that can support \(n!\equiv -1\) for composite moduli.
