# EP-1130

## Problem Statement
For interpolation nodes $x_1<\cdots<x_n$ in $[-1,1]$, define
\[
\Upsilon(x_1,\dots,x_n)=\min_{0\le i\le n}\ \max_{x\in[x_i,x_{i+1}]}\sum_k |\ell_k(x)|,
\]
with $x_0=-1$, $x_{n+1}=1$, and Lagrange basis functions $\ell_k$.
Questions: is $\Upsilon\ll\log n$, and which nodes maximize $\Upsilon$?

## Literature
- This problem is resolved in the cited background:
  - de Boor-Pinkus proved Erdős’ maximizer characterization,
  - bounds imply
  \[
  \Upsilon\le \frac{2}{\pi}\log n + O(1).
  \]

## Our Approaches / What Is Proven
- Added standalone deep numerical verification (`scripts/ep1130.mjs`):
  - barycentric evaluation of the Lebesgue function,
  - direct computation of interval maxima $\lambda_i$ and $\Upsilon=\min_i\lambda_i$,
  - long-budget node optimization per $n$.

## Computation-Guided Observations
Deep run (`DEPTH=4`) for
\[
n\in\{8,10,12,14,16,18\}
\]
with 18s optimization budget per $n$ found best nodes matching Chebyshev-type nodes to numerical precision in every case.

Observed values satisfy
\[
\Upsilon/\log n\approx 0.89,\ 0.86,\ 0.85,\ 0.83,\ 0.82,\ 0.82,
\]
consistent with logarithmic growth.

Interval-max spread $\max_i\lambda_i-\min_i\lambda_i$ stays around $0.44$ in this finite regime; relative spread decreases with $n$.

These are numerical checks, not proofs, but align strongly with the solved theoretical picture.

## Added Theorem-Level Resolution (As Written)

EP-1130 asks two things:
1. whether $\Upsilon\ll\log n$;
2. which node sets maximize $\Upsilon$.

The cited de Boor--Pinkus results in the background address both:
- they prove Erdős' maximizer characterization for this functional;
- they obtain
\[
\Upsilon\le \frac{2}{\pi}\log n + O(1),
\]
which is exactly a logarithmic upper bound of the requested type.

Therefore both parts of the as-written problem are settled by known theorems.

So EP-1130 is resolved in the affirmative (with maximizers characterized in the
cited literature).
