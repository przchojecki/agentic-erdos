# EP-50 partial attempt

## Route
Used known singularity of the distribution function and added a finite empirical scan of `phi(n)/n` CDF slopes.

## Evidence from this batch

## What is resolved from background
Erdos proved the distribution function is purely singular.

## Hard point
Pure singularity does not by itself rule out isolated points where derivative exists and is positive, so the stated derivative question remains subtle.

## Status
Partially resolved structurally (singularity known), full derivative claim still open.

## Deeper Proof Program (A => B => C)

### Target
Settle whether the distribution function \(D(x)\) of \(\varphi(n)/n\) can have
points where \(D'(x)\) exists and is strictly positive.

### Step A: Local Mass Upper Bound
For every interval \(I=[x,x+h]\), prove
\[
\#\{n\le N:\varphi(n)/n\in I\}\le N\cdot \Psi(x,h)
\]
with \(\Psi(x,h)=o(h)\) uniformly as \(h\to0\).

### Step B: Uniform Differentiability Obstruction
Upgrade Step A to show
\[
\limsup_{h\to0^+}\frac{D(x+h)-D(x)}{h}=0
\]
for all \(x\) (or all but exceptional null set).

### Step C: Global Derivative Conclusion
Conclude no point has positive finite derivative, strengthening pure singularity
to the precise derivative statement.

### Concrete Blocking Lemma (Most Critical)
Need a sharp short-interval counting estimate for values of \(\varphi(n)/n\)
near a fixed target \(x\), with saving stronger than linear in interval length.
