## Deeper Proof Program (A => B => C)

### Target
Determine maximal size \(n\) (or infinitude) of planar sets with pairwise integer
distances under no-3-collinear and no-4-cocircular constraints.

### Step A: Algebraic encoding
Represent distance constraints by integer/rational solutions to quadratic systems
with general-position side constraints.

### Step B: Local-to-global rigidity
Show large feasible systems force forbidden collinearity/cocircularity patterns.

### Step C: Extremal conclusion
Derive finite upper bound for \(n\), or produce a construction family.

### Concrete Blocking Lemma (Most Critical)
Need a rigidity lemma that any sufficiently large integer-distance framework in
the plane contains a forbidden degenerate subconfiguration.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CJ1 (Constraint monotonicity)
If no configuration of size \(n_0\) exists, then none exists for \(n>n_0\).

\[
\textit{Proof.}
\]
Any \(n\)-point configuration contains an \(n_0\)-point subset preserving all
pairwise integer distances and general-position constraints.
\(\square\)

### Proposition CJ2 (Finite-threshold reduction)
If one proves nonexistence at a threshold \(n_0\), then maximal size equals
\(n_0-1\). If for all \(n\) a construction exists, the property is infinite.

\[
\textit{Proof.}
\]
Combine CJ1 with explicit constructions/nonexistence.
\(\square\)
