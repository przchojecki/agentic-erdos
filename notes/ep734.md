# EP-734

## Problem Statement
Construct large pairwise balanced designs where each block size \(t\) occurs only \(O(n^{1/2})\) times.

## Literature
- De Bruijn-Erd\H{o}s lower-scale reasoning implies some size class must appear at least on \(\Omega(\sqrt n)\) scale in general.
- The challenge is a matching construction with uniform \(O(\sqrt n)\) multiplicity control across sizes.

## Our Approaches / What Is Proven
- Built a finite heuristic proxy by edge-partitioning \(K_n\) into cliques (interpreted as blocks), with randomized greedy growth from uncovered edges.
- Measured the maximum multiplicity among block sizes in best runs across multiple restarts.

## Computation-Guided Observations
- Deep run for \(n=25\) to \(121\) (36 trials each) gave best-found max multiplicities much larger than \(\sqrt n\) (ratios about 7-33).
- The obstruction in this heuristic is strong accumulation of small blocks (especially size 2), indicating this naive decomposition is far from the target regime.

## Status
- Open.
- Added standalone finite-construction stress test showing current heuristic is insufficient for the conjectured \(O(\sqrt n)\) profile.
