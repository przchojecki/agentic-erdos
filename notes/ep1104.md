# EP-1104

## Problem Statement
Let $f(n)$ be the maximum chromatic number of a triangle-free graph on $n$ vertices.
Estimate $f(n)$.

## Literature
- Current asymptotic bracket (from cited results) is
\[
(1-o(1))\sqrt{\frac{n}{\log n}}\le f(n)\le (2+o(1))\sqrt{\frac{n}{\log n}}.
\]

## Our Approaches / What Is Proven
- Replaced stub with standalone deep random-search script (`scripts/ep1104.mjs`):
  - random triangle-free process generation,
  - DSATUR heuristic coloring to hunt high-$\chi$ samples,
  - exact/partial chromatic certification on best-found sample at each $n$.
- Certification step tries $k$-colorability for small $k$ with hard time limits, yielding exact values when solved and otherwise lower/upper bounds.

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- $n\in\{30,40,50,60\}$,
- 25s random-search budget per $n$,
- exact-colorability attempt timeout 1.8s per tested $k$.

Sample counts were large (e.g. about $503{,}666$ graphs at $n=30$).
Best heuristic values found were $6,6,7,7$ for $n=30,40,50,60$.

Certification outcomes:
- $n=30$: exact $\chi=4$ (heuristic overestimated to 6).
- $n=40,50,60$: proved non-3-colorable; 4-colorability checks timed out, so current certified interval is
  - $4\le \chi \le 6$ for $n=40$,
  - $4\le \chi \le 7$ for $n=50,60$.

Interpretation:
- finite search confirms dense triangle-free samples with chromatic number at least 4,
- but heuristic best values alone are not reliable without exact certification,
- current run does not tighten asymptotic constants.
