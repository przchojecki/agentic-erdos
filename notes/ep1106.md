# EP-1106

## Problem Statement
Let $p(n)$ be the partition function and let
\[
F(n)=\omega\!\left(\prod_{1\le k\le n} p(k)\right)
\]
be the number of distinct prime factors appearing among $p(1),\dots,p(n)$.
Questions:
1. Does $F(n)\to\infty$?
2. Is $F(n)>n$ for all sufficiently large $n$?

## Literature
- First question is resolved positively.
- Known lower bounds include at least logarithmic growth.
- Second stronger inequality remains open.

## Our Approaches / What Is Proven
- Added standalone deep modular scan (`scripts/ep1106.mjs`).
- For each prime $q\le Q$, computed first index $k\le N$ such that
\[
q\mid p(k)
\]
via Euler pentagonal recurrence modulo $q$.
- Also computed full residue-zero profiles up to $N$ for a substantial prime subset (no early exit) to deepen computation and capture recurrence behavior.

## Computation-Guided Observations
Deeper run used:
- $N=50{,}000$,
- $Q=12{,}000$,
- tested primes: $1{,}438$.

Results:
- among primes $\le 12{,}000$, exactly $1{,}433$ were detected dividing some $p(k)$ with $k\le 50{,}000$,
- giving rigorous lower bound
\[
F(50{,}000)\ge 1{,}433.
\]
- this remains far below $n$ at this scale (so finite data does not support the stronger eventual inequality).

Interpretation:
- strong finite evidence for robust growth of $F(n)$,
- but still only lower-bound evidence from bounded prime range and finite $n$.

## Status
- First question solved in literature.
- Second question open.
- Added deeper standalone modular recurrence computation producing large explicit lower-bound profiles.
