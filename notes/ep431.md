## Deeper Proof Program (A => B => C)

### Target
Decide whether infinite sets \(A,B\) can satisfy \(A+B\) equals primes up to finitely many exceptions.

### A. Concrete Blocking Lemma (Most Critical)

Prove a sumset-structure obstruction: if \(A+B\) captures all sufficiently large primes, then additive energy and modular distribution of \(A,B\) force contradictory density constraints (inverse-Goldbach barrier).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (modular necessity)
For each modulus \(q\), residue classes hit by \(A+B\) must include almost all prime-allowable classes modulo \(q\).

**Proof.**
All sufficiently large primes are in \(A+B\); primes occupy reduced residue classes mod \(q\), so \(A+B\) must cover these classes except finitely many exceptions. \(\square\)

#### Lemma B2 (sumset size lower bound)
If \(A,B\) are infinite with given counting functions, then \((A+B)\cap[1,x]\) size is constrained by additive combinatorial bounds from \(|A\cap[1,x]|,|B\cap[1,x]|\).

**Proof.**
Apply standard sumset inequalities (Cauchy-Davenport/Freiman-type in appropriate finite quotients and lifting). \(\square\)

#### Lemma B3 (A implies impossibility)
If A yields incompatible growth/density requirements from B1 and B2, no such infinite \(A,B\) exist.

**Proof.**
Contradiction between necessary coverage and achievable sumset structure. \(\square\)

### C. Main unresolved point

The bottleneck is obtaining a fully quantitative inverse theorem strong enough at prime-density scales, not just coarse heuristics.
