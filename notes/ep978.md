# EP-978

## Proof Attempts and Literature Notes

### Working statement
Study squarefreeness of
\[
F(n)=n^4+2,
\]
in particular whether infinitely many \(F(n)\) are squarefree.

### Exact indicator decomposition
Using
\[
\mu^2(m)=\sum_{d^2\mid m}\mu(d),
\]
the counting function
\[
Q_F(X):=\#\{n\le X:\ F(n)\ \text{squarefree}\}
\]
has exact form
\[
Q_F(X)=\sum_{n\le X}\sum_{d^2\mid F(n)}\mu(d)
      =\sum_{d\ge1}\mu(d)\,A_F(X;d^2),
\]
where
\[
A_F(X;q):=\#\{n\le X:\ F(n)\equiv 0\pmod q\}.
\]

### Local-density heuristic constant
Let
\[
\rho_F(q):=\#\{a\bmod q:\ F(a)\equiv0\pmod q\}.
\]
If one can justify
\[
A_F(X;d^2)=\frac{\rho_F(d^2)}{d^2}X+o(X)
\]
uniformly enough in \(d\), then
\[
Q_F(X)\sim C_F X,\qquad
C_F:=\prod_p\left(1-\frac{\rho_F(p^2)}{p^2}\right).
\]

So EP-978 is naturally a squarefree-sieve/local-solubility problem.

### Computation context
Finite scans in this project found squarefree frequency near `0.757` for `n<=3000`, consistent with a positive constant \(C_F\), but this is not a proof of infinitude.
