# EP-409 deep attempt with standalone computation

## Statement
For `T(n)=\phi(n)+1`, how many iterations are needed to hit a prime? Can
infinitely many `n` hit the same prime? What is the density of integers reaching
a fixed prime?

range using memoized dynamics.

## Deeper Proof Program (A => B => C)

### Target
Analyze dynamics of \(T(n)=\phi(n)+1\): hitting-time to primes, basin densities, and possible infinite basins for fixed terminal primes.

### A. Concrete Blocking Lemma (Most Critical)

Prove a basin-density transfer lemma: for each prime \(q\), preimage tree of \(q\) under \(T\) has natural density either \(0\) or an explicit value from convergent branch-weight product.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (prime terminal condition)
If iteration reaches a prime \(q\), subsequent iterate is \(\phi(q)+1=q\), so \(q\) is a fixed point.

**Proof.**
For prime \(q\), \(\phi(q)=q-1\). \(\square\)

#### Lemma B2 (preimage equation)
\[
T(n)=m\iff \phi(n)=m-1.
\]
Hence basin structure is governed by inverse totient fibers.

**Proof.**
Direct rearrangement of map definition. \(\square\)

#### Lemma B3 (A implies density statements)
If A gives convergent branch-weight formula for each \(q\), then densities of starting points converging to \(q\) follow.

**Proof.**
Sum branch contributions over preimage levels; convergence yields density value. \(\square\)

### C. Main unresolved point

The bottleneck is controlling inverse-totient branching multiplicities deeply enough to pass from finite-level counts to true natural densities.
