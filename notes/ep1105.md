# EP-1105

## Problem Statement
For anti-Ramsey numbers, ask whether the cycle/path formulas proposed by Erdős-Simonovits-Sós hold:
- for cycles $C_k$,
- for paths $P_k$.

## Literature
- This problem is marked solved on the source page (updated 2026) with modern progress on path/cycle anti-Ramsey formulas.
- In this project, computation is used as a consistency audit, not as proof.

## Our Approaches / What Is Proven
- Replaced stub with standalone bounded exhaustive search (`scripts/ep1105.mjs`) for small instances.
- Backtracking maximizes color count in edge-colorings of $K_n$ under no-rainbow-$G$ constraints.
- Reports exact values if search finishes; otherwise certified lower bounds from incumbent solutions.

## Computation-Guided Observations
Deep run (`DEPTH=4`) with 45s timeout per instance over 9 instances gave:

Exact values obtained:
- $\mathrm{AR}(4,C_3)=3$,
- $\mathrm{AR}(5,C_3)=4$,
- $\mathrm{AR}(6,C_3)=5$,
- $\mathrm{AR}(7,C_3)=6$,
- $\mathrm{AR}(6,P_5)=6$.

Timed-out instances (certified lower bounds):
- $\mathrm{AR}(7,P_5)\ge 7$,
- $\mathrm{AR}(8,P_5)\ge 7$,
- $\mathrm{AR}(7,C_4)\ge 8$,
- $\mathrm{AR}(8,C_4)\ge 8$.

This run explored very large trees (up to tens of millions of nodes in some cases) and is substantially deeper than prior quick checks.
