# EP-809

## Problem Statement
Local dataset text is malformed, but the preserved core indicates an odd-cycle/rainbow-coloring extremal function \(F_k(n)\) in a near-Tur\'an regime, with question whether \(F_k(n)\sim n^2/8\).

## Literature
- Background notes record a quadratic lower bound scale \(F_k(n)\gg n^2\).
- Exact asymptotic constant remains unresolved in the available notes.

## Our Approaches / What Is Proven
- Because source statement is malformed, we ran an explicit finite proxy:
  - near-Tur\'an edge-colorings,
  - sampled detection of rainbow odd cycles of lengths 3,5,7,
  - comparison of random colorings vs structured 2-color construction.

## Computation-Guided Observations
- Random \(k\)-color near-Tur\'an samples (here \(k=6\)) almost never avoided bounded rainbow odd cycles.
- Structured 2-color near-Tur\'an construction passed the sampled no-rainbow-odd-cycle test with edge density near the complete-graph scale in this proxy.
- This highlights high sensitivity to coloring structure and confirms the need for a precise formal statement before drawing extremal conclusions.
