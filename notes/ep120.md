## LaTeX Proofs for A => B => C Skeleton

### Proposition AI1 (Window reduction principle)
Let \(A\subset\mathbb R\) measurable with upper density \(\delta>0\). Then for a
sequence of intervals \(I_N\), the discretizations \(A_N=A\cap I_N\) satisfy
\[
|A_N|\ge (\delta-o(1))|I_N|.
\]

\[
\textit{Proof.}
\]
Direct from definition of upper density.
\(\square\)

### Proposition AI2 (Diagonal embedding upgrade)
Assume for each \(m\) there exists \(N_m\) such that every dense discretization
\(A_{N_m}\) contains an \(m\)-point similar copy of the target pattern. Then \(A\)
contains an infinite similar copy.

\[
\textit{Proof sketch.}
\]
Choose nested \(m\)-point copies with consistent first \(m-1\) points by compact
selection/diagonalization; pass to the limit.
\(\square\)
