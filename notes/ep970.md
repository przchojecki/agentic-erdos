# EP-970

## Proof Attempts and Literature Notes

### Working statement
Let `j(n)` be Jacobsthal's function: the least integer `m` such that every block of `m` consecutive integers contains an integer coprime to `n`.
For fixed `k`, define
\[
h(k):=\max_{\omega(n)=k} j(n),
\]
where \(\omega(n)\) is the number of distinct prime divisors.

### Computation context
Finite scans in this project show moderate growth on primorial/squarefree examples, but are not asymptotic proofs.

### Rigorous lower-bound lemma (primorial witness)
Let
\[
P_k:=\prod_{i=1}^k p_i
\]
(the `k`-th primorial). Then
\[
j(P_k)\ge p_{k+1}-1.
\]

Proof:
Consider the block
\[
2,3,\dots,p_{k+1}-1.
\]
Every integer in this block has a prime factor \(\le p_k\):
- if it is prime, it is one of \(p_1,\dots,p_k\);
- if it is composite, it has a prime divisor \(\le \sqrt{p_{k+1}-1}<p_{k+1}\), hence \(\le p_k\).
So each number in the block has gcd `>1` with `P_k`, i.e. none is coprime to `P_k`.
The block length is \(p_{k+1}-2\), so by definition of `j`,
\[
j(P_k) > p_{k+1}-2,
\]
hence \(j(P_k)\ge p_{k+1}-1\).

### Consequence for `h(k)`
Since \(\omega(P_k)=k\),
\[
h(k)\ge j(P_k)\ge p_{k+1}-1\sim k\log k.
\]
So at minimum the extremal growth is at least order `k log k`; the central open gap is between this and the conjectural/quadratic-scale upper behavior discussed in the literature.
