# EP-321 partial

## Statement
Let `R(N)` be the largest size of `A subseteq {1,...,N}` such that all subset sums
`sum_{n in S} 1/n` are distinct. Estimate `R(N)`.

## Attempt in this batch
I ran exact branch-and-bound for small `N`, converting reciprocals to exact integer
weights by scaling with `lcm(1,...,N)`.

Data file:
- `data/ep321_exact_small_scan.json`

## Result
Exact values were computed for `8 <= N <= 25`.
- `R(25)=18` (exact).
- Ratio `R(N)/(N/log N)` in this range is about `1.82` to `2.33`.
- Runtime grows sharply (e.g. `N=25` took about 37.8s).

## Hard point
Finite exact data supports `R(N)` on `N/log N` scale but does not resolve the
iterated-log correction terms in the background bounds.

## Deeper Proof Program (A => B => C)

### Target
Determine sharp asymptotics of \(R(N)\), including iterated-log factors around
the \(N/\log N\) scale.

### Step A: Collision-graph model
Encode equality of subset reciprocal sums as collisions in a weighted subset-sum
map; admissible sets correspond to injectivity.

### Step B: Upper bound via additive energy
Bound injective-family size by controlling number of near-collisions among
weighted subsets.

### Step C: Lower bound via sparse coding
Construct explicit families with provably distinct subset sums near the upper
scale.

### Concrete Blocking Lemma (Most Critical)
Need an anti-collision theorem for reciprocal weights \(\{1/n\}\) restricted to a
chosen denominator set, strong enough to save iterated-log factors.

## Lemma Chain (with Proofs)

### Lemma 1 (Injectivity criterion)
For \(A\subseteq\{1,\dots,N\}\), all subset sums
\(\sum_{n\in S}1/n\) are distinct iff
\[
\sum_{n\in S} \frac1n = \sum_{n\in T}\frac1n \implies S=T.
\]

\[
\textit{Proof.}
\]
This is exactly injectivity of the subset-sum map.
\(\square\)

### Lemma 2 (Scaling equivalence)
Let \(L_N=[1,\dots,N]\). Distinctness of reciprocal subset sums over \(A\) is
equivalent to distinctness of integer subset sums with weights \(L_N/n\), \(n\in
A\).

\[
\textit{Proof.}
\]
Multiply all subset sums by \(L_N\); scaling is injective.
\(\square\)

### Lemma 3 (Reduction to additive-combinatorial bound)
Any upper bound on size of a subset of \(\{L_N/n:n\le N\}\) with distinct subset
sums gives the same upper bound for \(R(N)\).

\[
\textit{Proof.}
\]
Apply Lemma 2 and definition of \(R(N)\) as maximal admissible size.
\(\square\)
