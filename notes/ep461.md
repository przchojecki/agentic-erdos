# EP-461 partial resolution

## Statement
For `f(n,t)` counting distinct `t`-smooth components in `[n+1,n+t]`, is
`f(n,t) >> t` uniformly?

## What is resolved
Background reports the lower bound `f(n,t) >> t/log t`.

## What remains open in this note
The full linear-in-`t` lower bound remains open in the cited material.

## Status
- `t/log t` lower bound: known.
- uniform linear lower bound: unresolved.

## New Experiments
- 2026-03-12T10:24:07.771Z: standalone deep interval scan (`scripts/ep461.mjs`)
  for `t` up to `1800` and many starting points `n<=80000`.

## One-by-One Cycle
- Replaced placeholder script with direct computation of
  `f(n,t)=|\{s_t(m):m\in[n+1,n+t]\}|`.
- Finite ratios `f(n,t)/t` stayed well above zero in all tested cases (minimum
  around `0.53-0.58` depending on `t`), with means roughly `0.58-0.72`.
- This gives strong finite support toward linear behavior, but does not prove a
  uniform asymptotic lower bound.

## Deeper Proof Program (A => B => C)

### Target
Upgrade known bound \(f(n,t)\gg t/\log t\) to uniform linear lower bound \(f(n,t)\gg t\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a local smooth-part anti-collision estimate: in any interval \([n+1,n+t]\), the number of pairs \((u,v)\) with equal \(t\)-smooth part is \(O(t)\), implying \(\gg t\) distinct values.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (collision counting identity)
If \(r_x:=\#\{m\in[n+1,n+t]: s_t(m)=x\}\), then
\[
\sum_x r_x=t,\qquad
\#\text{collisions}=\sum_x \binom{r_x}{2}.
\]

**Proof.**
Partition interval points by smooth-part value. \(\square\)

#### Lemma B2 (distinct-count formula)
\[
f(n,t)=\#\{x:r_x>0\}=t-\sum_x (r_x-1).
\]

**Proof.**
Each class contributes one distinct value; extras are duplicates. \(\square\)

#### Lemma B3 (A implies linear lower bound)
If A gives \(\sum_x\binom{r_x}{2}\le Ct\), then by convexity \(\sum_x(r_x-1)\le C't\) with \(C'<1\), hence \(f(n,t)\ge (1-C')t\).

**Proof.**
Bound pair collisions controls average class multiplicity; apply B2. \(\square\)

### C. Main unresolved point

The bottleneck is establishing uniform anti-collision bounds for smooth parts in short intervals independent of \(n\).
