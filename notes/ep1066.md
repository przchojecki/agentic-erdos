# EP-1066

## Problem Statement
Investigate the minimum possible independence number in unit-distance graphs on $n$ vertices (EP-1066 formulation), and its linear-scale constant behavior.

## Literature
- Current known bounds are linear in $n$ with a nontrivial constant gap; exact constant remains open.

## Our Approaches / What Is Proven
- Replaced the small batch proxy with a standalone deeper exact MIS pipeline:
  - large triangular-lattice patch model ($14\times14$),
  - many random induced subgraphs,
  - exact maximum independent set computation on each sampled graph,
  - distributional profile over sample set.
- Deep run size:
  - $n\in\{24,30,36,42\}$,
  - $480$ exact samples per $n$.

## Computation-Guided Observations
- Minimum sampled ratios in this induced-subgraph model were around:
  - $0.625$ for $n=24$,
  - $0.567$ for $n=30$,
  - $0.556$ for $n=36$,
  - $0.548$ for $n=42$.
- Means decrease with $n$ in this finite model but stay far above the sharpest known lower-bound constants for worst-case unit-distance graphs.
