### Target

tail coverage in the EP-32 formulation.

### Step A: Almost-Cover with Logarithmic Budget
Prove \(O(\log N)\)-sized complement sets can cover all but \(o(N)\) numbers in
the target interval model.

### Step B: Exceptional Set Structure
Show the residual uncovered set has strong arithmetic structure (few residue
classes / sparse Bohr-type support).

### Step C: Finite Repair
Use a bounded number of additional shifts/complements to eliminate the
structured exceptional set entirely.

### Concrete Blocking Lemma (Most Critical)

shifts implies eventual complete coverage after bounded structural correction.

## LaTeX Proofs for A => B => C Skeleton

Let \(I_X=[X/2,X]\cap\mathbb Z\) and \(U_X\subset I_X\) be uncovered points.

### Proposition C1 (Structured exceptional set + finite repair => full cover)
Assume for all large \(X\):
1. \(|U_X|=o(X)\),
2. \(U_X\subset \bigcup_{j=1}^m (b_j+q\mathbb Z)\) for fixed \(m,q\),
3. for each \(j\), one admissible extra shift eliminates \(U_X\cap (b_j+q\mathbb Z)\).

Then after adding at most \(m\) extra shifts, coverage on \(I_X\) is complete
for all large \(X\).

\[
\textit{Proof.}
\]
Condition (2) decomposes all uncovered points into at most \(m\) classes.
By (3), adding the class-specific shift removes each class entirely.
After all \(m\) repairs, no uncovered points remain.
\(\square\)

### Proposition C2 (Asymptotic full coverage from vanishing uncovered fraction)
If complete repair in Proposition C1 is available for all large \(X\), then the
EP-32 tail-coverage statement follows.

\[
\textit{Proof.}
\]
For sufficiently large \(X\), \(I_X\) is fully covered; monotone extension in
the construction then yields eventual tail coverage in the intended range.
\(\square\)

Open content is exactly the structural claim in (2) and admissible repair in (3).
