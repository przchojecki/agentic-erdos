## Deeper Proof Program (A => B => C)

### Target
Determine whether the sumset \(A+B\) (with digit-restricted base sets in the
problem) has positive asymptotic density, and if possible compute the density.

### Step A: Automaton/expansion encoding
Model admissible expansions and carry propagation by a finite-state transducer.

### Step B: Ergodic frequency analysis
Analyze long-prefix frequencies of accepted expansions to derive lower/upper
density bounds.

### Step C: Gap-distribution control
Show maximal forbidden gap growth is sublinear enough to imply positive lower
density (or construct sparse counter-patterns).

### Concrete Blocking Lemma (Most Critical)
Need a carry-sensitive mixing lemma for the transducer so local digit freedoms
produce global density uniformly.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AM1 (Density from block frequencies)
If for some block length \(L\), every residue class modulo \(q^L\) appears in
\(A+B\) with frequency at least \(\eta>0\) in long intervals, then
\[
\underline d(A+B)\ge \eta.
\]

\[
\textit{Proof sketch.}
\]
Partition long intervals into \(q^L\)-blocks and average class counts.
\(\square\)

### Proposition AM2 (Reduction to transducer mixing)
If the finite-state expansion transducer for \(A+B\) is strongly connected and
aperiodic with stationary mass bounded below on accepting states, then the
hypothesis of AM1 holds for some \(\eta>0\).

\[
\textit{Proof.}
\]
Markov-chain mixing gives uniform asymptotic visitation frequencies of accepting
states, which translate to accepted residue densities.
\(\square\)
