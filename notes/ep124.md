### Proposition AL1 (Admissibility necessity)
If an integer \(N\) is representable in the shifted model, then \(N\) lies in an
admissible residue class modulo \(M\) determined by generators and shift.

\[
\textit{Proof.}
\]
Take the representation modulo \(M\); only residues generated by the allowed
combination pattern can occur.
\(\square\)

### Proposition AL2 (Admissibility sufficiency reduction)
If for each admissible class \(r\!\!\pmod M\) there exists \(N_r\) such that all
\(N\equiv r\pmod M\), \(N\ge N_r\), are representable, then eventual complete
classification follows with threshold \(N_0=\max_r N_r\).

\[
\textit{Proof.}
\]
Check residue class of \(N\); apply the corresponding bound.
\(\square\)
