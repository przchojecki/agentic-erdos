# EP-108 partial

## Statement
For `r>=4, k>=2`, does there exist finite `f(k,r)` such that every graph with
chromatic number at least `f(k,r)` contains a subgraph of girth at least `r`
and chromatic number at least `k`?

- The related infinitary version is also recorded open.

## Hard point

requires stronger cycle-control while preserving large chromatic number.

## Deeper Proof Program (A => B => C)

### Target
For each \(k\ge2, r\ge4\), prove existence of finite \(f(k,r)\) such that
\(\chi(G)\ge f(k,r)\) implies \(G\) contains a subgraph \(H\) with
\(\chi(H)\ge k\) and girth at least \(r\).

### Step A: High-chromatic sparse core
Extract a subgraph with large average degree and controlled local cycle profile.

### Step B: Cycle-pruning with chromatic retention
Delete a carefully chosen edge/vertex set to kill all short cycles
\(<r\) while keeping chromatic number \(\ge k\).

### Step C: Parameter closure
Track losses to produce explicit finite \(f(k,r)\).

### Concrete Blocking Lemma (Most Critical)
Need a short-cycle hitting lemma with chromatic stability: removing all cycles
below length \(r\) should not collapse chromatic number too much.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AD1 (Chromatic retention criterion)
If \(G\) has a subgraph \(H\) with
\[
\chi(H)\ge k,\qquad \text{girth}(H)\ge r,
\]
then \(G\) satisfies the EP conclusion for \((k,r)\).

\[
\textit{Proof.}
\]
Immediate since \(H\subseteq G\) is exactly the required witness.
\(\square\)

### Proposition AD2 (Reduction to pruning lemma)
Suppose every graph \(G\) with \(\chi(G)\ge F\) contains \(H\subseteq G\) such
that deleting a short-cycle transversal \(X\) gives
\[
\chi(H-X)\ge k,\qquad \text{girth}(H-X)\ge r.
\]
Then \(f(k,r)\le F\).

\[
\textit{Proof.}
\]
Take \(f(k,r)=F\). Any \(G\) with \(\chi(G)\ge F\) yields witness \(H-X\), then
apply AD1.
\(\square\)
