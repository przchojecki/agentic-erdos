## LaTeX Proofs for A => B => C Skeleton

### Proposition BR1 (Finite-prefix approximation)
If a sequence has discrepancy profile \(h_N(d)\) on prefixes \([1,N]\), then
asymptotic bounds on \(h(d)\) can be derived from uniform control of \(h_N(d)\)
for \(N\to\infty\).

\[
\textit{Proof sketch.}
\]
Define \(h(d)\) via limsup/liminf over \(N\); uniform in-\(N\) estimates pass to
limits.
\(\square\)

### Proposition BR2 (Exponent reduction)
If one proves
\[
d^{\alpha-o(1)}\le h(d)\le d^{\beta+o(1)},
\]
then any improvement of \(\alpha\) upward or \(\beta\) downward narrows the
asymptotic window. Matching exponents resolves growth.

\[
\textit{Proof.}
\]
Take logs and divide by \(\log d\).
\(\square\)
