# EP-944

## Proof Attempts and Literature Notes

### Working statement
For \(k=4\): does every vertex-critical \(\chi=4\) graph contain at least one edge whose deletion drops chromatic number below \(4\)?

### Deep standalone computation (`ep944.mjs`)
- Method: heavy random graph search with exact DSATUR-based chromatic checks.
- For each sampled graph, tested:
  - \(\chi(G)=4\),
  - vertex-criticality (
    \(\chi(G-v)<4\) for all \(v\)),
  - number of critical single edges (
    edges \(e\) with \(\chi(G-e)<4\)).
- Run parameters: `n in {10,11,12,13,14}`, `100000` samples per `n` (total `500000`). Runtime \(\approx 18.6\)s.

### Results
- No sampled graph satisfied “vertex-critical \(\chi=4\) and zero critical single edges”.
- Vertex-critical \(\chi=4\) hits were rare:
  - \(n=10\): 15 hits, minimum critical-edge count `12`
  - \(n=11\): 3 hits, minimum critical-edge count `16`
  - \(n=12\): 2 hits, minimum critical-edge count `18`
  - \(n=13,14\): 0 hits

### Interpretation
This deeper search still found no counterexample candidate for the `k=4` case; finite evidence points toward rarity of such constructions, but does not prove impossibility.
