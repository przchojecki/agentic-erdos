# EP-1030

## Problem Statement
In the near-diagonal off-diagonal regime, ask whether there is a fixed $c>0$ such that
\[
\frac{R(k+1,k)}{R(k,k)}>1+c
\]
for all sufficiently large $k$.

## Literature
- Known additive lower bounds (e.g. linear one-step gaps) do not by themselves imply a fixed multiplicative separation.
- The multiplicative-gap formulation remains open.

## Our Approaches / What Is Proven
- Added a deeper standalone constructive ratio proxy:
  - exact random-coloring avoidance checks for
    - diagonal constraint $(K_k,K_k)$,
    - off-diagonal constraint $(K_{k+1},K_k)$,
  - finite lower-bound ratio comparison for $k=4,5,6$.

## Computation-Guided Observations
- Constructive lower-bound ratios in this run:
\[
\frac{\text{LB}(R(5,4))}{\text{LB}(R(4,4))}\approx 1.33,\quad
\frac{\text{LB}(R(6,5))}{\text{LB}(R(5,5))}\approx 1.20,\quad
\frac{\text{LB}(R(7,6))}{\text{LB}(R(6,6))}\approx 1.20.
\]
- This is finite evidence only; it does not establish the asymptotic fixed-gap theorem.

## Status
- Open.
- Added deep standalone exact-clique constructive ratio experiment.

## Deeper Proof Program (A => B => C)

### Target
Show there exists \(c>0\) such that for all large \(k\),
\[
R(k+1,k)\ge (1+c)R(k,k).
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a uniform off-diagonal increment bound:
\[
R(k+1,k)-R(k,k)\ge c\,R(k,k)
\]
for some absolute \(c>0\) and all large \(k\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (ratio-difference equivalence)
\[
\frac{R(k+1,k)}{R(k,k)}>1+c
\iff
R(k+1,k)-R(k,k)>c\,R(k,k).
\]

**Proof.**
Multiply by positive denominator \(R(k,k)\). \(\square\)

#### Lemma B2 (monotonic lower bound)
\[
R(k+1,k)\ge R(k,k).
\]

**Proof.**
Avoiding red \(K_{k+1}\) is weaker than avoiding red \(K_k\), with same blue condition. \(\square\)

#### Lemma B3 (A implies fixed multiplicative gap)
If A holds, then B1 immediately gives the desired fixed-gap ratio.

**Proof.**
Substitute A into B1. \(\square\)

### C. Main unresolved point

The hard step is obtaining a relative (not merely additive) one-step separation between adjacent diagonal/off-diagonal Ramsey regimes.
