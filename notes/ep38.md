# EP-38 partial resolution

## Statement
Ask for a set B that is not an additive basis but still guarantees a uniform positive
Schnirelmann-density increment for A union (A+b), with b chosen from B.

## What is resolved
Background gives density increment when B is an additive basis of finite order k.
It also notes existence of essential components that are not additive bases.

## What remains open in this note
Whether such non-basis B can satisfy the stronger uniform increment property remains
unresolved.

## Status
- additive-basis case known; related non-basis constructions exist.
- stated non-basis strengthening remains open.

## One-by-One Cycle
- Read statement/background and replaced placeholder script with a per-problem exhaustive model: for each candidate shift-set `B`, and each admissible finite set `A\subseteq[1,N]` with positive density floor, computed `\max_{b\in B} ( |(A\cup(A+b))\cap[1,N]|-|A\cap[1,N]| )/N` and minimized this over all such `A`.
- Result in this finite model: structured non-basis proxy families (even shifts, multiples of 3, powers of 2, prime shifts) all admit near-zero worst-case gains at tested scales, while allowing all shifts gives strictly stronger gains.
- Obstacle: converting finite-window adversarial sets into a rigorous infinite Schnirelmann obstruction (or proving positivity for a non-basis `B`) still requires a structural theorem beyond these computations.
