# EP-38 partial resolution

## Statement
Ask for a set B that is not an additive basis but still guarantees a uniform positive
Schnirelmann-density increment for A union (A+b), with b chosen from B.

## What is resolved
Background gives density increment when B is an additive basis of finite order k.
It also notes existence of essential components that are not additive bases.

## What remains open in this note
Whether such non-basis B can satisfy the stronger uniform increment property remains
unresolved.

## Status
- additive-basis case known; related non-basis constructions exist.
- stated non-basis strengthening remains open.

## Deeper Proof Program (A => B => C)

### Target
Find a non-additive-basis set \(B\subset\mathbb N\) and \(c>0\) such that for
every \(A\subset\mathbb N\) with \(\sigma(A)>0\), some \(b\in B\) satisfies
\[
\sigma(A\cup(A+b))\ge (1+c)\sigma(A).
\]

### Step A: Uniform Increment Hypothesis
Assume the above inequality with fixed \(c>0\) holds for all \(A\) of positive
Schnirelmann density.

### Step B: Iterative Density Amplification
Iterate shifts from \(B\): define \(A_0=A\), \(A_{t+1}=A_t\cup(A_t+b_t)\) with
best choice \(b_t\in B\). Then
\[
\sigma(A_t)\ge (1+c)^t\sigma(A_0)\wedge 1.
\]

### Step C: Finite-Term Coverage Consequence
For \(t\) large enough, \(\sigma(A_t)=1\), forcing eventual full coverage by a
finite iterated sum-difference expression in \(B\).

### Concrete Blocking Lemma (Most Critical)
Need an iteration-to-structure theorem: if Step A holds uniformly, then finite
iterates of \(B\)-shifts must act like a finite-order basis operator despite \(B\)
not itself being an additive basis.

## LaTeX Proofs for A => B => C Skeleton

### Proposition F1 (Iterative multiplicative growth)
Assume there is \(c>0\) such that for every set \(X\) with \(\sigma(X)>0\), one
can choose \(b\in B\) with
\[
\sigma(X\cup(X+b))\ge (1+c)\sigma(X).
\]
Then for the iterative process above,
\[
\sigma(A_t)\ge \min\{1,(1+c)^t\sigma(A_0)\}.
\]

\[
\textit{Proof.}
\]
Induct on \(t\). Base \(t=0\) is trivial. Inductive step uses the hypothesis on
\(X=A_t\):
\[
\sigma(A_{t+1})\ge (1+c)\sigma(A_t)\ge (1+c)\min\{1,(1+c)^t\sigma(A_0)\},
\]
and truncation at \(1\) gives the claim.
\(\square\)

### Proposition F2 (Finite-time saturation)
Under Proposition F1 assumptions, if
\[
t\ge \left\lceil \frac{\log(1/\sigma(A_0))}{\log(1+c)}\right\rceil,
\]
then \(\sigma(A_t)=1\).

\[
\textit{Proof.}
\]
By Proposition F1, \(\sigma(A_t)\ge (1+c)^t\sigma(A_0)\ge 1\). Since
\(\sigma(\cdot)\le1\), equality follows.
\(\square\)
