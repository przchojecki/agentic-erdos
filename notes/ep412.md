## Deeper Proof Program (A => B => C)

### Target
Decide whether every pair \(m,n\ge2\) has a collision \(\sigma_i(m)=\sigma_j(n)\).

### A. Concrete Blocking Lemma (Most Critical)

Prove basin dichotomy for \(\sigma\)-iteration graph: either all trajectories enter a single giant component, or there exists an invariant valuation/parity barrier producing infinitely many disjoint components.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (forward-map monotonicity)
For \(t\ge2\), \(\sigma(t)\ge t+1\), so each trajectory is strictly increasing.

**Proof.**
Divisors include \(1,t\), hence \(\sigma(t)\ge t+1\). \(\square\)

#### Lemma B2 (collision graph interpretation)
Two seeds collide iff their forward orbits intersect; this is equivalent to belonging to same connected component in directed functional graph.

**Proof.**
Functional graph has one outgoing edge per node; orbit intersection equals component co-membership. \(\square\)

#### Lemma B3 (A resolves EP-412)
If A yields single-component inevitability, answer is yes; if it yields invariant barriers and infinitely many components, answer is no.

**Proof.**
Translate basin/component structure to pair-collision quantifier. \(\square\)

### C. Main unresolved point

The key obstacle is identifying (or ruling out) global invariants preserved by \(\sigma\)-iteration that could prevent eventual orbit intersection.
