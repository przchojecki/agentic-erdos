# EP-787

## Problem Statement
Let \(g(n)\) be maximal such that every \(A\subset\mathbb R\), \(|A|=n\), contains \(B\subseteq A\), \(|B|\ge g(n)\), with
\[
b_1+b_2\notin A\quad\text{for all distinct }b_1,b_2\in B.
\]
Estimate \(g(n)\).

## Literature
- Bounds have improved, but the exact asymptotic order is open.

## Our Approaches / What Is Proven
- Built deeper exact finite proxy:
  - for each sampled \(A\), construct conflict graph on indices \(i\neq j\) when \(a_i+a_j\in A\),
  - compute exact maximum independent set size \(\alpha(A)\), yielding a lower-bound proxy for \(g(n)\).
- Compared structured families (interval, odd interval, geometric-like) plus random sets.

## Computation-Guided Observations
- For \(n=20,28,36,44\), worst sampled family remained the interval \(\{1,\dots,n\}\).
- Worst ratios \(\alpha(A)/n\) were about 0.55, 0.536, 0.528, 0.523, showing gradual decline.
- Random and non-interval structured samples were generally less adversarial.
