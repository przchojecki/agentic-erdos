# EP-969

## Proof Attempts and Literature Notes

### Working statement
Let
\[
Q(x)=\#\{n\le x:\ n \text{ squarefree}\},\qquad
E(x)=Q(x)-\frac{6}{\pi^2}x.
\]
The problem asks for the true order of growth of \(E(x)\).

### Computation context
Finite scans in this project (up to `x=2e6`) saw small \(|E(x)|\), but finite behavior does not settle asymptotics.

### Exact Möbius formula
Using
\[
1_{\mathrm{sqf}}(n)=\sum_{d^2\mid n}\mu(d),
\]
we get
\[
Q(x)=\sum_{d\le \sqrt x}\mu(d)\left\lfloor\frac{x}{d^2}\right\rfloor.
\]
Hence
\[
E(x)=
\sum_{d\le \sqrt x}\mu(d)\left(\left\lfloor\frac{x}{d^2}\right\rfloor-\frac{x}{d^2}\right)
-x\sum_{d>\sqrt x}\frac{\mu(d)}{d^2}.
\]

### Unconditional elementary bound
Since \(|\lfloor t\rfloor-t|\le 1\),
\[
\left|\sum_{d\le \sqrt x}\mu(d)\left(\left\lfloor\frac{x}{d^2}\right\rfloor-\frac{x}{d^2}\right)\right|
\le \lfloor\sqrt x\rfloor.
\]
Also
\[
x\left|\sum_{d>\sqrt x}\frac{\mu(d)}{d^2}\right|
\le x\sum_{d>\sqrt x}\frac1{d^2}
\le \sqrt x.
\]
Therefore
\[
|E(x)|\le 2\sqrt x
\]
for all \(x\ge 1\).

This is much weaker than conjectural bounds, but is a fully rigorous baseline.
