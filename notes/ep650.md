# EP-650

## Problem Statement
Let \(f(m)\) be such that for every \(A\subseteq\{1,\dots,N\}\) with \(|A|=m\), every interval of length \(2N\) contains at least \(f(m)\) distinct integers \(b_1,\dots,b_r\), each divisible by some distinct \(a_i\in A\).
Estimate \(f(m)\); in particular, is \(f(m)\ll m^{1/2}\)?

## Literature
- Erd\H{o}s and S\'ar\'anyi proved \(f(m)\gg m^{1/2}\).
- Upper-order behavior is open in the problem statement.

## Our Approaches / What Is Proven
- Implemented a standalone interval matching proxy: left side = \(A\), right side = integers in a window of length \(2N\), edge \(a\sim b\) if \(a\mid b\), score = minimum matching size over sampled windows.
- Performed local search over \(A\) for several \((N,m)\).

## Computation-Guided Observations
- In all tested regimes up to \((N,m)=(600,50)\), the proxy saturates at full matching \(=m\).
- Additional dense sanity checks (e.g. \(m\) close to \(N\)) also saturated under this proxy.
- This indicates the current proxy is too weak to detect the true obstruction behind the open problem.

## Status
- Open.
- Current computation is diagnostic of proxy weakness, not of the true asymptotic value of \(f(m)\).
