# EP-846

## Problem Statement
If an infinite planar point set $A$ has the property that every finite $n$-point subset contains a subset of size at least $\epsilon n$ with no three collinear, must $A$ be a finite union of sets each having no three collinear?

## Literature
- Erdos-Nesetril-Rodl problem.
- Closely related to finite-vs-global decomposition principles for geometric hypergraphs.

## Our Approaches / What Is Proven
- Built finite analog via 3-uniform collinearity hypergraph on selected point configurations.
- Computed exactly:
  - $\alpha_3$: largest no-three-collinear subset;
  - $\chi_3$: minimum number of classes partitioning points so each class is no-three-collinear.

## Computation-Guided Observations
- Structured line-rich examples (e.g. $5\times3$ grid / union of 3 lines) show $\alpha_3/n\approx 0.4$ with $\chi_3=3$.
- A generic curved example (parabola points) gives $\alpha_3=n$ and $\chi_3=1$.
- Finite data supports that local linear extraction can coexist with small finite partition number, but does not settle the infinite statement.
