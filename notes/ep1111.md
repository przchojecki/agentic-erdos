## Deeper Proof Program (A => B => C)

### Target
Quantify how large \(\min(\chi(A),\chi(B))\) must be for anticomplete pairs \((A,B)\) inside high-chromatic triangle-free graphs.

### A. Concrete Blocking Lemma (Most Critical)

Prove a local expansion lemma: in every triangle-free graph with \(\chi(G)\ge c\), either there is an anticomplete pair with
\[
\min(\chi(A),\chi(B))\ge \phi(c),
\]
or \(G\) contains a bounded-size forbidden configuration violating known strengthening hypotheses.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (anticomplete pair coloring inequality)
If \(A,B\) are anticomplete, then
\[
\chi(G[A\cup B])=\max(\chi(A),\chi(B)).
\]

**Proof.**
No edges between \(A,B\), so a coloring of \(A\cup B\) is obtained by independently coloring each side with the same palette size equal to the larger chromatic number; lower bound is obvious from induced subgraphs. \(\square\)

#### Lemma B2 (critical subgraph reduction)
To prove lower bounds depending on \(\chi(G)\), it suffices to consider \(\chi\)-critical triangle-free \(G\).

**Proof.**
Every graph contains an induced \(\chi(G)\)-critical subgraph; any anticomplete-pair guarantee there lifts to the original graph. \(\square\)

#### Lemma B3 (A implies asymptotic growth)
If Lemma A yields \(\phi(c)\to\infty\), then the desired function \(f(c)\) exists with \(f(c)\to\infty\).

**Proof.**
Set \(f=\phi\). The lemma provides the property for each \(c\). \(\square\)

### C. Main unresolved point

The hard part is establishing quantitative local expansion/structure in triangle-free \(\chi\)-critical graphs strong enough to force large chromaticity on both sides of an anticomplete split.
