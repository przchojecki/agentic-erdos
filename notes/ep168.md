# EP-168 partial

## Statement
Let `F(N)` be the largest size of a subset of `{1,...,N}` with no triple
`{n,2n,3n}`. Determine `lim F(N)/N` and whether this limit is irrational.

## Attempt in this batch
I solved the exact finite optimization for small `N` as a 3-uniform independent
set problem on edges `{n,2n,3n}` with branch-and-bound plus propagation.

Data file:
- `data/ep168_exact_small_scan.json`

## Result
Exact values for `20 <= N <= 75` (step 5):
- Most sampled points gave exact ratio `F(N)/N = 0.8`.
- One sample (`N=50`) gave `41/50 = 0.82`.

## Hard point
This finite exact frontier is consistent with a stable density near `0.8`,
but it does not determine the true limit or address irrationality.

## Deeper Proof Program (A => B => C)

### Target
Determine
\[
\lim_{N\to\infty}\frac{F(N)}{N}
\]
for sets avoiding \(\{n,2n,3n\}\), and decide rationality/irrationality.

### Step A: Multiplicative-chain decomposition
Partition \(\mathbb N\) into chains \(m\cdot\{1,2,4,\dots\}\) and coupled
\((1,2,3)\)-constraints.

### Step B: Periodic optimization on truncated systems
Solve finite-state density optimization for chain-coupled constraints and prove
subadditivity/superadditivity.

### Step C: Limit identification
Extract limit density via ergodic/subadditive convergence and analyze algebraic
equation defining optimizer to test irrationality.

### Concrete Blocking Lemma (Most Critical)
Need a transfer theorem from finite periodic optimizers to global integer
density with quantified boundary error.

## LaTeX Proofs for A => B => C Skeleton

### Proposition BL1 (Subadditive envelope)
If \(F(N)\) is maximum feasible size under local forbidden triples, then
\[
F(N+M)\le F(N)+F(M)+O(1).
\]

\[
\textit{Proof sketch.}
\]
Concatenate optimal blocks with bounded interface correction for triples crossing
the join.
\(\square\)

### Proposition BL2 (Limit existence reduction)
If BL1 and a matching superadditive lower envelope hold up to \(O(1)\), then
\[
\lim_{N\to\infty}\frac{F(N)}{N}
\]
exists (Fekete-type argument with bounded defects).

\[
\textit{Proof.}
\]
Apply generalized subadditive convergence theorem.
\(\square\)
