## Deeper Proof Program (A => B => C)

### Target
Show existence (or impossibility) of a seed producing an infinite prime sequence under
\[
q_{n+1}=\min\{q_n+q_i-1:\ 1\le i\le n,\ \text{prime}\}.
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a nontermination criterion: if seed-generated set of offsets \(\{q_i-1\}\) hits infinitely many admissible residue classes modulo every fixed modulus, then prime candidates persist indefinitely.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (candidate form)
Each step searches primes in arithmetic-progressive translate set
\[
\mathcal C_n=\{q_n+q_i-1:1\le i\le n\}.
\]

**Proof.**
Definition of recurrence. \(\square\)

#### Lemma B2 (termination condition)
Process terminates at \(n\) iff all elements of \(\mathcal C_n\) are composite.

**Proof.**
By minimal-prime selection rule. \(\square\)

#### Lemma B3 (A implies infinitude)
If A ensures at least one prime in \(\mathcal C_n\) for all \(n\), recurrence continues forever.

**Proof.**
Then B2 never occurs. \(\square\)

### C. Main unresolved point

The bottleneck is proving sufficiently uniform prime occurrence in the evolving candidate sets \(\mathcal C_n\), which are highly dependent on earlier steps.
