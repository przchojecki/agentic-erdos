# EP-258 partial resolution

## Statement
If `a_n` are positive integers with `a_n -> infinity`, is

`sum_n tau(n)/(a_1...a_n)`

irrational?

## What is resolved
Background says Erdos-Straus proved irrationality when `(a_n)` is monotone
nondecreasing.

## Additional proof route attempted here
I tried extending from monotone to broader non-monotone classes via block/effective-monotone
decompositions of the denominator products.

Main obstruction: after truncation/shift, the resulting tails involve `tau(n+offset)` in the
numerator, so the monotone theorem does not transfer directly in a black-box way.

## Status
- monotone `a_n`: resolved.
- general `a_n -> infinity`: unresolved.

## Deeper Proof Program (A => B => C)

### Target
Extend irrationality from monotone \(a_n\) to all positive integer sequences
\(a_n\to\infty\) in
\[
\sum_{n\ge1}\frac{\tau(n)}{a_1\cdots a_n}.
\]

### Step A: Product-growth normalization
Analyze denominator products \(Q_n=a_1\cdots a_n\) and extract subsequences with
controlled growth ratios.

### Step B: Shifted-divisor handling
Control the effect of non-monotone blocks that introduce \(\tau(n+\Delta)\)-type
tails in reduction attempts.

### Step C: Irrationality criterion
Prove no eventual periodic/rational-tail structure can persist under
\(a_n\to\infty\).

### Concrete Blocking Lemma (Most Critical)
Need a robust irrationality criterion that depends only on \(Q_{n+1}/Q_n\to\infty\)
along subsequences, without monotonicity of \(a_n\).

## LaTeX Proofs for A => B => C Skeleton

### Proposition DE1 (Tail control)
Let \(S=\sum_{n\ge1}\tau(n)/Q_n\), \(Q_n=a_1\cdots a_n\). Then
\[
0<S-S_N\le \sum_{n>N}\frac{n^\varepsilon}{Q_n}
\]
for any fixed \(\varepsilon>0\) and large \(n\), using \(\tau(n)\ll_\varepsilon
n^\varepsilon\).

\[
\textit{Proof.}
\]
Apply standard divisor-function bound termwise.
\(\square\)

### Proposition DE2 (Reduction)
If one proves infinitely many \(N\) with denominator-growth windows where no
rational with denominator \(\le Q_N^\theta\) approximates \(S\) within the DE1
tail scale, then \(S\) is irrational.

\[
\textit{Proof.}
\]
A rational \(S=p/q\) would force compatible approximations from partial sums at
all large \(N\), contradicting the windowed exclusion.
\(\square\)
