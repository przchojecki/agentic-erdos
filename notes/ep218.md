# EP-218 partial attempt

## Route
Computed prime-gap comparison frequencies for index-based prefixes up to `n=10^6`.

## Evidence from this batch
- `data/ep218_prime_gap_monotonicity_scan.json` shows empirical proportions
  `P(d_{n+1}>=d_n)` and `P(d_{n+1}<=d_n)` both near `0.516` at `n=10^6`.
- Equality events `d_{n+1}=d_n` occur many times in range (frequency about `0.032` at `n=10^6`).

## Hard point
Finite-frequency evidence does not prove exact limiting density `1/2` or infinitely many equalities.

## Status
Conjectural asymptotic statements remain open.

## Deeper Proof Program (A => B => C)

### Target
Prove limiting comparison frequencies for consecutive prime gaps (including the
conjectural \(1/2\) split and infinitely many equalities).

### Step A: Prime tuple transition model
Express events \(d_{n+1}\gtrless d_n\) via local patterns of consecutive primes.

### Step B: Averaged distribution control
Use distribution of prime constellations in short intervals to estimate event
densities.

### Step C: Limit extraction
Prove convergence of empirical frequencies and identify limit constants.

### Concrete Blocking Lemma (Most Critical)
Need a sufficiently uniform asymptotic for adjacent-gap pair distribution
\((d_n,d_{n+1})\) beyond current averaged prime-gap results.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CL1 (Frequency partition identity)
For each \(N\),
\[
P_N(d_{n+1}>d_n)+P_N(d_{n+1}<d_n)+P_N(d_{n+1}=d_n)=1.
\]

\[
\textit{Proof.}
\]
Trichotomy of real numbers for each index \(n\), then average.
\(\square\)

### Proposition CL2 (Reduction to equality density)
If one proves \(P_N(d_{n+1}=d_n)\to 0\), then
\[
P_N(d_{n+1}>d_n)\to \frac12
\iff
P_N(d_{n+1}<d_n)\to \frac12.
\]

\[
\textit{Proof.}
\]
Use CL1 and symmetry/equivalence of the two complementary limits.
\(\square\)
