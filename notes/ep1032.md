# EP-1032

## Problem Statement
For $4$-critical graphs, investigate whether one can force very large minimum degree (in particular linear-in-$n$ scale in the EP formulation).

## Literature
- Existing constructions discussed in background are far from establishing the strongest linear-scale target in full generality.

## Our Approaches / What Is Proven
- Added deterministic exact verification of a standard infinite family:
  - odd wheels $W_{2m+1}$,
  - exact checks that these examples are $4$-critical,
  - extracted minimum degree profile.
- Added deeper random exact search for additional small-$n$ $4$-critical graphs and their minimum degrees.

## Computation-Guided Observations
- Deterministic family checks verify odd wheels are $4$-critical for tested sizes.
- In these examples, minimum degree is exactly $3$, so
\[
\delta(G)/n\to 0
\]
along this family.
- Random search over $n=11$ to $20$ (large trial budget) did not produce better minimum-degree $4$-critical examples in this run.

## Status
- Open.
- Added standalone exact criticality verification and finite minimum-degree profile evidence.

## Deeper Proof Program (A => B => C)

### Target
Determine maximal possible minimum degree in \(4\)-critical graphs as a function of \(n\), and whether linear-scale forcing is possible.

### A. Concrete Blocking Lemma (Most Critical)

Show every \(4\)-critical graph \(G\) has a large sparse structural decomposition implying
\[
\delta(G)\le \eta n
\]
for some universal \(\eta<1\), or conversely build \(4\)-critical expanders with \(\delta(G)\ge c n\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (critical-degree floor)
If \(G\) is \(4\)-critical, then \(\delta(G)\ge 3\).

**Proof.**
Removing a vertex of degree \(\le2\) allows extending any 3-coloring of \(G-v\) back to \(G\), contradiction. \(\square\)

#### Lemma B2 (odd-wheel witness)
Odd wheels are \(4\)-critical with \(\delta=3\).

**Proof.**
Rim odd cycle requires three colors; hub adjacent to all rim vertices forces fourth color. Deleting any edge/vertex reduces to 3-colorable graph. Minimum degree is 3 (rim vertices). \(\square\)

#### Lemma B3 (A decides linear-scale possibility)
If A yields universal sublinear upper bound, linear target fails; if A yields explicit linear families, target holds.

**Proof.**
Direct implication from definition of minimum-degree scaling. \(\square\)

### C. Main unresolved point

The bottleneck is understanding global structure of \(4\)-critical graphs beyond known sparse families to either force or preclude linear minimum degree.
