## LaTeX Proofs for A => B => C Skeleton

### Proposition Z1 (Double-counting identity)
Let \(\mathcal L_4\) be set of 4-point lines in \(P\), \(|P|=n\). Then
\[
\sum_{p\in P} \deg_{\mathcal L_4}(p)=4|\mathcal L_4|.
\]

\[
\textit{Proof.}
\]
Count incidences \((p,\ell)\) with \(p\in\ell\), \(\ell\in\mathcal L_4\), by
points and by lines.
\(\square\)

### Proposition Z2 (Reduction to point-degree bound)
If one proves \(\deg_{\mathcal L_4}(p)\le g(n)\) for all \(p\), then
\[
|\mathcal L_4|\le \frac{n\,g(n)}4.
\]

\[
\textit{Proof.}
\]
Apply Z1 and bound each summand.
\(\square\)
