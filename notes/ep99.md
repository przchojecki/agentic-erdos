## Deeper Proof Program (A => B => C)

### Target
For sufficiently large \(n\), prove every diameter-minimizing \(1\)-separated
\(n\)-point set contains a unit equilateral triangle.

### Step A: Stability of near-optimal packings
Show minimizers are close to triangular-lattice truncations after scaling/rigid
motions.

### Step B: Pattern forcing in lattice-like sets
Prove any sufficiently large lattice-close set necessarily contains a unit
equilateral triple.

### Step C: Transfer through stability
Deduce the same for all true minimizers.

### Concrete Blocking Lemma (Most Critical)
Need a robust geometric stability theorem with error small enough to preserve
exact unit-length triangles.

## LaTeX Proofs for A => B => C Skeleton

### Proposition X1 (Template transfer)
Assume every minimizer \(P_n\) is \(\varepsilon_n\)-close to a template set
\(T_n\) that contains a unit equilateral triangle, with \(\varepsilon_n\to0\)
and nondegenerate angle margins. Then \(P_n\) contains a unit equilateral
triangle for all large \(n\).

\[
\textit{Proof sketch.}
\]
Choose the witness triple in \(T_n\) with margin \(\eta>0\) away from degenerate
configurations. For \(\varepsilon_n\ll \eta\), perturbation preserves equality of
the three side lengths at value \(1\) after the normalization used in minimizers.
\(\square\)

### Proposition X2 (Reduction)
If Step A provides the closeness in X1 and Step B supplies witness triangles in
all large templates, then EP-99 asymptotic form follows.

\[
\textit{Proof.}
\]
Immediate from X1.
\(\square\)
