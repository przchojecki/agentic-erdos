# EP-1089

## Problem Statement
Let $g_d(n)$ be the least integer such that every set of $g_d(n)$ points in $\mathbb{R}^d$ determines at least $n$ distinct distances.
Estimate $g_d(n)$; in particular, for fixed $n$, study behavior as $d\to\infty$.

## Literature
- Classical framework asks whether growth is on the order of $d^{n-1}$ for fixed $n$.
- For $n=3$, small-dimensional exact values are known in low dimensions (e.g. $g_3(3)=7$).

## Our Approaches / What Is Proven
- Focused on the $n=3$ case, where avoiding 3 distinct distances means constructing a set with at most 2 distinct pairwise distances.
- Built a richer candidate universe in $\mathbb{R}^d$:
  - shell points of norm $1$ and $2$ in $\{-1,0,1\}^d$.
- For each $d$, solved many exact max-clique subproblems (one per allowed 1- or 2-distance value set) using branch-and-bound to maximize candidate subset size.
- Any found subset of size $M$ gives a rigorous finite lower bound
\[
g_d(3)\ge M+1.
\]

## Computation-Guided Observations
Deep run (`DEPTH=4`) for
\[
d=3,\dots,10
\]
found best candidate-universe 2-distance sizes:
- $d=3$: $M=6$ (implies $g_d(3)\ge 7$),
- $d=6$: $M=15$,
- $d=7$: $M=21$,
- $d=8$: $M=28$,
- $d=9$: $M=36$,
- $d=10$: $M=38$.

Compared with cross-polytope baseline $2d$, this gives improvements for $d\ge 6$ in this candidate universe.
The winning allowed distance pair was consistently
\[
\{2,4\}
\]
(squared distances).

Important caveat:
- for larger $d$ some clique subproblems hit time/node caps, so these are valid lower bounds from current incumbents, not certified global optima even within the chosen universe.

## Deeper Proof Program (A => B => C)

### Target
For fixed \(n\), determine growth of \(g_d(n)\) as \(d\to\infty\), especially the conjectural \(d^{n-1}\)-type scale.

### A. Concrete Blocking Lemma (Most Critical)

For \(n=3\), classify maximal 2-distance sets in high dimension up to \(o(M)\) perturbation, with
\[
M \asymp d^2.
\]
This would pin down \(g_d(3)\) order and guide higher \(n\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (definition bridge)
\(g_d(3)-1\) equals the maximum size of a set in \(\mathbb R^d\) with at most two distinct distances.

**Proof.**
By definition, \(g_d(3)\) is the threshold forcing at least 3 distinct distances. So one less is the largest size avoiding that, i.e., using \(\le2\) distances. \(\square\)

#### Lemma B2 (construction-to-lower-bound transfer)
Any explicit \(M\)-point 2-distance configuration in \(\mathbb R^d\) gives
\[
g_d(3)\ge M+1.
\]

**Proof.**
Immediate from Lemma B1. \(\square\)

#### Lemma B3 (stability-to-asymptotics)
If every near-maximal 2-distance set is structurally close to a classified family with size \(T(d)\), then
\[
g_d(3)=T(d)+o(T(d)).
\]

**Proof.**
Near-maximality plus stability bounds size deviation by \(o(T(d))\); maximal size then matches \(T(d)\) asymptotically. \(\square\)

### C. Main unresolved point

The bottleneck is a high-dimensional stability/classification theorem for 2-distance sets at the exact extremal scale.
