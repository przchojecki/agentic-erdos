# EP-960

## Proof Attempts and Literature Notes

### Working statement
Let `f_{r,k}(n)` be the threshold number of ordinary lines forcing an `r`-subset whose all pair-lines are ordinary, among `n`-point sets with no `k` collinear points.

### Deep standalone finite experiment (`ep960.mjs`)
- Parameters: `r=6`, `k=4`, `n in {12,14,16,18,20}`.
- For each `n`, sampled `12,000` accepted point sets (rejecting samples with `>=4` collinear points).
- Mixed sampling: general-position random sets + triad-heavy sets (to create many non-ordinary lines).
- For each set: computed ordinary-line count and whether the ordinary-line graph contains a `K_6`.

### Results (empirical)
- Estimated forcing thresholds (in sampled family):
  - `n=12`: `55`
  - `n=14`: `77`
  - `n=16`: `82`
  - `n=18`: `109`
  - `n=20`: `0` (all sampled sets already had `K_6`)
- Highest ordinary-line counts seen without `K_6` were one below threshold where nontrivial (`54, 76, 81, 108`).

### Interpretation
These are finite sampled proxies, not proofs of true thresholds. They provide computational evidence that large ordinary-line counts strongly correlate with forced `K_6` in the ordinary-line graph for these small `n`.

### Turan reduction (proof-level upper bound template)
For a fixed point set \(P\), define its ordinary-line graph \(G_{\mathrm{ord}}(P)\):
- vertices are points of \(P\);
- an edge \(uv\) is present iff the line through \(u,v\) is ordinary in \(P\).

Then the number of ordinary lines equals \(|E(G_{\mathrm{ord}}(P))|\).
If \(|E(G_{\mathrm{ord}}(P))| > \left(1-\frac1{r-1}\right)\frac{n^2}{2}\), Turan's theorem implies \(G_{\mathrm{ord}}(P)\) contains \(K_r\), i.e. an \(r\)-subset whose every pair-line is ordinary.

Therefore,
\[
f_{r,k}(n)\ \le\ \left(1-\frac1{r-1}\right)\frac{n^2}{2}+1,
\]
independently of \(k\). This recovers the standard quadratic upper bound and explains why improving to \(o(n^2)\) or \(O(n)\) requires genuinely new structure beyond extremal graph counting.
