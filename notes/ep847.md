# EP-847

## Problem Statement
If every $n$-point subset of an infinite $A\subseteq\mathbb N$ contains an $\epsilon n$-sized 3-term-AP-free subset, must $A$ be a finite union of 3-term-AP-free sets?

## Literature
- Erdos-Nesetril-Rodl problem.
- Companion to geometric version EP-846.

## Our Approaches / What Is Proven
- Built finite exact analog on $[1..N]$ using 3-AP hypergraph.
- Computed exactly:
  - $\alpha_{3\text{AP}}(N)$, maximum 3AP-free subset size;
  - minimum color count partitioning $[1..N]$ into 3AP-free classes.

## Computation-Guided Observations
- For $N=9,12,15,18$, computed ratios $\alpha_{3\text{AP}}(N)/N$ in the range about $0.44$-$0.56$.
- In all tested cases, 3 colors suffice to partition into 3AP-free classes.
- Finite exact behavior is consistent with nontrivial local extraction plus bounded finite coloring, but does not imply the infinite theorem.
