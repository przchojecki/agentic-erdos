# EP-936

## Proof Attempts and Literature Notes

### Working statement
Investigate powerful values in the two families:
\[
2^n\pm 1,\qquad n!\pm 1.
\]

### Deep standalone computation (`ep936.mjs`)
- Method: exact BigInt factorization (Miller-Rabin + Pollard Rho), then direct test that every prime exponent is at least \(2\).
- Run ranges: \(2\le n\le 110\) for \(2^n\pm1\), and \(3\le n\le 36\) for \(n!\pm1\); runtime \(\approx 42.87\)s.
- Powerful hits found:
  - \(2^n\pm1\): only \(2^3+1=9=3^2\).
  - \(n!\pm1\): only \(4!+1=25=5^2\), \(5!+1=121=11^2\), \(7!+1=5041=71^2\).
- No unresolved factorizations in this run.

### Interpretation
At this depth, there are no new powerful values beyond the known tiny examples. This supports extreme sparsity, but does not by itself prove finiteness.
