# EP-1022 deeper attempt

## Statement
Find `c_t` (with `c_t->infinity`) such that local subset-density condition
`# {A in F: A subset X} < c_t |X|` for all `X` forces property B.

## Computation in this attempt (finite search, t=3)
I ran random search over non-property-B 3-uniform families on 9 vertices and measured
`rho=max_X e(X)/|X|`.

Data file:
- `data/ep1022_random_nonpropertyB_density_search.json`

Best found non-property-B family had `rho ~ 1.222`.

## Interpretation
This is finite lower-bound evidence on thresholds for small instances, not an
asymptotic determination of `c_t`.

## Exact threshold reformulation
For a \(t\)-uniform hypergraph \(H=(V,E)\), define
\[
\rho(H):=\max_{\varnothing\ne X\subseteq V}\frac{e_H(X)}{|X|},
\]
where \(e_H(X)=\#\{A\in E:\ A\subseteq X\}\).

Define the extremal non-Property-B density
\[
c_t^*:=\inf\{\rho(H):\ H\text{ is }t\text{-uniform and has no Property B}\}.
\]

Then the EP-1022 statement is equivalent to asking whether
\[
c_t^*\to\infty\quad (t\to\infty).
\]

Reason:
- If \(c<c_t^*\), every \(t\)-uniform \(H\) with \(\rho(H)<c\) must have Property B.
- Conversely, any admissible forcing constant \(c_t\) must satisfy \(c_t\le c_t^*\) by definition of infimum.

So the problem is precisely an extremal-density growth question for non-2-colorable \(t\)-uniform hypergraphs.

## Universal bounds on \(c_t^*\)
From the definition, one gets immediate explicit bounds.

### Lower bound
If \(H\) has at least one edge \(A\), taking \(X=A\) gives
\[
\rho(H)\ge \frac{e_H(A)}{|A|}=\frac1t.
\]
Hence
\[
c_t^*\ge \frac1t.
\]

### Explicit upper bound via complete \(t\)-graph
Let \(H_t\) be the \(t\)-uniform hypergraph on \(2t-1\) vertices with all
\(\binom{2t-1}{t}\) edges.
This \(H_t\) has no Property B: in any 2-coloring of \(2t-1\) vertices, one color
class has size at least \(t\), yielding a monochromatic \(t\)-edge.

For \(X=V(H_t)\),
\[
\rho(H_t)\ge \frac{|E(H_t)|}{|V(H_t)|}
\;=\;
\frac{\binom{2t-1}{t}}{2t-1}.
\]
Therefore
\[
c_t^*\le \rho(H_t)\le \frac{\binom{2t-1}{t}}{2t-1}.
\]

So currently we have
\[
\frac1t \le c_t^* \le \frac{\binom{2t-1}{t}}{2t-1},
\]
and EP-1022 asks whether \(c_t^*\to\infty\).
