## Deeper Proof Program (A => B => C)

### Target
Determine asymptotics of \(f(n)\), the largest size of a set with all consecutive
interval sums distinct.

### Step A: Conflict hypergraph formulation
Vertices are chosen indices; hyperedges encode equal consecutive-sum constraints.

### Step B: Upper bound via conflict counting
Bound maximal independent-set size in this hypergraph.

### Step C: Lower construction
Build near-optimal families with provably distinct interval sums.

### Concrete Blocking Lemma (Most Critical)
Need a sharp counting lemma for number of interval-sum collisions induced by a set
of size \(k\), with explicit dependence on \(n,k\).

## Lemma Chain (with Proofs)

### Lemma 1 (Collision characterization)
Distinctness of all consecutive sums fails iff there exist two intervals
\([i,j],[u,v]\) with
\[
\sum_{t=i}^j a_t=\sum_{t=u}^v a_t,\quad (i,j)\ne(u,v).
\]

\[
\textit{Proof.}
\]
This is direct restatement of distinctness condition.
\(\square\)

### Lemma 2 (Prefix-sum reformulation)
Let \(P_r=\sum_{t\le r} a_t\). Then
\[
\sum_{t=i}^j a_t=P_j-P_{i-1},
\]
so interval-sum collisions are equivalent to repeated differences \(P_x-P_y\).

\[
\textit{Proof.}
\]
Standard telescoping identity.
\(\square\)

### Lemma 3 (Reduction)
To bound \(f(n)\), it suffices to control how large a subset of prefix sums can be
chosen so that all pairwise differences are distinct.

\[
\textit{Proof.}
\]
By Lemma 2 the problem is exactly a distinct-differences (Golomb-ruler type)
constraint on prefix-sum indices.
\(\square\)
