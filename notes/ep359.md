## Deeper Proof Program (A => B => C)

### Target
Prove asymptotic growth for \(a_k\) (seed \(n=1\)) and establish the two limits in
EP-359.

### Step A: Forbidden-sum frontier analysis
At step \(k\), characterize representable consecutive sums and the first gap.

### Step B: Differential growth inequality
Bound \(a_{k+1}-a_k\) using density of representable intervals.

### Step C: Asymptotic integration
Solve resulting recurrence/inequality to obtain \(a_k\sim C\,k\log k/\log\log k\)
or comparable scale.

### Concrete Blocking Lemma (Most Critical)
Need a precise estimate for length and location of first nonrepresentable block as
function of current terms.

## Lemma Chain (with Proofs)

### Lemma 1 (Greedy minimality equation)
If \(R_k\) is the set of representable sums from consecutive earlier terms, then
\[
a_{k+1}=\min\{m>a_k:\ m\notin R_k\}.
\]

\[
\textit{Proof.}
\]
This is exactly the defining greedy rule.
\(\square\)

### Lemma 2 (Monotone growth)
\[
a_{k+1}>a_k,\quad\text{hence }a_k\to\infty.
\]

\[
\textit{Proof.}
\]
By Lemma 1, \(a_{k+1}\) is chosen strictly above \(a_k\); strictly increasing
integer sequence diverges.
\(\square\)

### Lemma 3 (Scale reduction)
If one proves constants \(c_1,c_2>0\) such that for large \(k\),
\[
c_1\frac{k\log k}{\log\log k}\le a_k\le c_2\frac{k\log k}{\log\log k},
\]
then both requested limit statements reduce to identifying exact constants and
second-order terms.

\[
\textit{Proof.}
\]
These inequalities pin principal scale; remaining task is sharpened asymptotics.
\(\square\)
