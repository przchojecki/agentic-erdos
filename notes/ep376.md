## Deeper Proof Program (A => B => C)

### Target
Study infinitude/finitude of
\[
\mathcal N_{105}:=\{n\ge1:\ \gcd\!\binom{2n}{n},105=1\}.
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a symbolic-dynamics criterion implying that the set of \(n\) whose base-\(p\) digits avoid carries for all \(p\in\{3,5,7\}\) is recognized by a finite automaton with spectral radius \(>1\) (for infinitude) or \(=1\) (for finiteness with eventual periodic constraints).

Either outcome would decisively resolve the qualitative question.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (Kummer carry criterion)
For prime \(p\),
\[
p\nmid \binom{2n}{n}\iff \text{adding }n+n\text{ in base }p\text{ has no carry}.
\]

**Proof.**
Kummer’s theorem states \(v_p\!\binom{2n}{n}\) equals the number of base-\(p\) carries when adding \(n\) and \(n\). So valuation \(0\) is equivalent to no carry. \(\square\)

#### Lemma B2 (digit-set formulation)
\[
n\in\mathcal N_{105}
\iff
\bigl(\text{no carry in base }3\bigr)\land\bigl(\text{no carry in base }5\bigr)\land\bigl(\text{no carry in base }7\bigr).
\]

**Proof.**
\(\gcd(\binom{2n}{n},105)=1\) is equivalent to nondivisibility by each prime \(3,5,7\). Apply Lemma B1 separately for each prime. \(\square\)

#### Lemma B3 (closure under CRT digit lifting)
If one can realize infinitely many compatible finite carry-free digit prefixes simultaneously mod \(3^a,5^b,7^c\), then \(\mathcal N_{105}\) is infinite.

**Proof.**
Use CRT to combine compatible residues into a residue class mod \(3^a5^b7^c\). A nested construction of longer compatible prefixes gives infinitely many integers satisfying all three no-carry constraints, hence in \(\mathcal N_{105}\) by Lemma B2. \(\square\)

### C. Main unresolved point

The key difficulty is global compatibility of no-carry constraints across three incommensurate bases. Local constructions in one base are easy; synchronized infinite constructions across \((3,5,7)\) remain hard.
