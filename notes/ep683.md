# EP-683 partial

## Statement
Does there exist `c>0` such that for all `1<=k<=n`,
`P(binomial(n,k)) >= min(n-k+1, k^(1+c))`?

## Attempt in this batch
I computed exact largest prime divisors of binomial coefficients up to `n=2000`
using factorial prime-valuation tables.

Data file:
- `data/ep683_binom_largest_prime_scan.json`

## Result
- The worst observed exponent margin
  `c_local = log(P(binomial(n,k))/k)/log k` in this range is about `0.099`.
- Current worst witness at scan end: `(n,k)=(1972,986)` with
  `P(binomial(n,k))=1951`.

So finite data supports positive `c`, but with small margin in this range.

## Hard point
Finite scans cannot establish a uniform asymptotic constant `c>0` for all large
`n,k`.
