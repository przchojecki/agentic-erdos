# EP-341 deep attempt (no partial theorem extracted)

## Statement
Starting from a finite seed set and greedily extending by forbidding pairwise sums,
is the difference sequence eventually periodic?

## Attempt route
Tried automaton-style state compression on recent differences and sumset frontier,
with small-seed simulation checks.

## Obstacle
Background gives only qualitative evidence that transient behavior can be very
long; no theorem-level structural criterion is cited.

## Status
- no proof or counterexample obtained in this attempt.

## New Experiments
- 2026-03-05T14:54:09.460Z: enforced deep standalone tail-period scan on greedy pairwise-sum-avoiding sequences (8 seeds, 28k terms each, runtime 22.68s).
- Finite signal: all tested seeds exhibited detectable short tail periods in differences (periods 1,2,5,11,12 in this scan) after long transients.
- Proof direction update: data supports an eventual-periodicity conjecture for broad seed families; a next route is to encode the greedy step as a finite-state process on a moving forbidden-sum frontier.

## Deeper Proof Program (A => B => C)

### Target
Prove eventual periodicity (or produce a counterexample) for the difference
sequence of the EP-341 greedy process.

### Step A: State compression
Represent greedy step by a finite "frontier state" capturing enough information to
determine next difference.

### Step B: Deterministic state evolution
Show evolution is a deterministic map on this state space.

### Step C: Pigeonhole periodicity
If state space is finite, eventual periodicity follows by repetition.

### Concrete Blocking Lemma (Most Critical)
Need a finite-memory theorem: after some point, all constraints affecting the next
step depend only on a bounded window of recent differences.

## Lemma Chain (with Proofs)

### Lemma 1 (Determinism of greedy extension)
Given prefix \(a_1<\cdots<a_n\), next term \(a_{n+1}\) is uniquely determined.

\[
\textit{Proof.}
\]
Greedy rule chooses the least admissible integer greater than \(a_n\); least
element of a nonempty subset of \(\mathbb N\) is unique.
\(\square\)

### Lemma 2 (Finite-state implies eventual periodicity)
If there exists a finite set of states \(\mathcal S\) and deterministic update
\(T:\mathcal S\to\mathcal S\) encoding the process tail, then resulting output
difference sequence is eventually periodic.

\[
\textit{Proof.}
\]
Orbit \(s_0,s_1,\dots\) in finite \(\mathcal S\) must repeat: \(s_i=s_j\) for some
\(i<j\). Determinism gives periodic tail with period \(j-i\).
\(\square\)

### Lemma 3 (Reduction)
EP-341 reduces to proving Lemma 2's finite-state encoding from process
definitions.

\[
\textit{Proof.}
\]
Once such encoding exists, eventual periodicity follows immediately by Lemma 2.
\(\square\)
