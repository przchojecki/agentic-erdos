## Deeper Proof Program (A => B => C)

### Target
Determine sharp asymptotics of \(k(n,m)\): the least \(k\) such that every
tournament on \(k\) vertices has an independent set of size \(n\) or a transitive
subtournament of size \(m\).

### Step A: Entropy lower constructions
Build explicit/random tournaments avoiding both structures up to large \(k\),
giving lower bounds on \(k(n,m)\).

### Step B: Container/regularity upper mechanism
Show any larger tournament must contain one forbidden structure via iterative
density increment.

### Step C: Match scales
Align lower and upper formulas to pin the correct growth in \(n,m\).

### Concrete Blocking Lemma (Most Critical)
Need a two-parameter container theorem simultaneously controlling large
independent sets and large transitive subtournaments.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AF1 (Monotonicity)
The function \(k(n,m)\) is nondecreasing in each argument:
\[
n_1\le n_2,\ m_1\le m_2 \implies k(n_1,m_1)\le k(n_2,m_2).
\]

\[
\textit{Proof.}
\]
If a tournament forces an independent set of size \(n_2\) or transitive set of
size \(m_2\), it certainly forces one of size \(n_1\) or \(m_1\).
\(\square\)

### Proposition AF2 (Reduction to matching bounds)
If functions \(L(n,m)\), \(U(n,m)\) satisfy
\[
L(n,m)\le k(n,m)\le U(n,m),
\]
and \(\log U-\log L=o(1)\cdot \log L\) in a regime, then \(k(n,m)\) is
asymptotically determined there up to \(1+o(1)\) multiplicative factor.

\[
\textit{Proof.}
\]
Take logarithms and compare relative error.
\(\square\)
