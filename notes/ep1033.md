# EP-1033 partial attempt

## Route
Ran local search on dense graphs near the Turán threshold to find small values of
`max_{triangle} (deg(u)+deg(v)+deg(w))`, giving finite upper-profile evidence for `h(n)`.

## Evidence from this batch
- `data/ep1033_triangle_degree_sum_search.json` found best ratios around `1.60n` for `n=24..48`.
- This is above `2(\sqrt{3}-1) n ≈ 1.464n` in scanned sizes, hence no finite contradiction to the conjectured lower target.

## Hard point
Search gives only finite upper-profile samples and cannot establish asymptotic lower bounds for all dense graphs.

## Status
No closure; finite experiments consistent with current bound window.

## Deeper Proof Program (A => B => C)

### Target
Sharpen asymptotics for the minimum possible
\[
h(n):=\min_{e(G)\ge n^2/4}\ \max_{\triangle uvw}\bigl(\deg u+\deg v+\deg w\bigr)
\]
in the EP-1033 formulation.

### A. Concrete Blocking Lemma (Most Critical)

Prove a density-to-heavy-triangle lemma:
every graph with \(e(G)\ge n^2/4\) contains a triangle with degree-sum at least
\[
\left(2(\sqrt3-1)+\varepsilon_n\right)n,
\]
where \(\varepsilon_n\to0\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (triangle existence at threshold plus)
If \(e(G)>n^2/4\), then \(G\) contains a triangle.

**Proof.**
Turán/Mantel theorem: triangle-free maximum is \(n^2/4\). \(\square\)

#### Lemma B2 (degree-sum averaging on triangles)
Let \(T\) be triangle set. Then
\[
\frac1{|T|}\sum_{uvw\in T}(\deg u+\deg v+\deg w)
\]
is controlled by weighted counts of edge-triangle incidences, enabling lower bounds from global edge density.

**Proof.**
Expand triangle-degree sum by summing vertex degrees over incident triangles and rewrite via edge/vertex incidence identities. \(\square\)

#### Lemma B3 (A implies lower bound for \(h(n)\))
If A holds uniformly, then
\[
h(n)\ge \left(2(\sqrt3-1)+o(1)\right)n.
\]

**Proof.**
Take minimum over all eligible graphs; A gives same lower bound for each graph. \(\square\)

### C. Main unresolved point

The key gap is converting global density information into sharp pointwise control on one triangle's degree sum at the conjectured constant.
