## Deeper Proof Program (A => B => C)

### Target
Determine whether \(V(x)/V'(x)\) has a limit and identify it, where
\[
V'(x)=\#\{\phi(m):m\le x\},\qquad V(x)=\#\{n\le x:\exists m,\phi(m)=n\}.
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a multiplicity-moment theorem for totient preimages:
\[
\sum_{n\le x}(A(n)-1)_+ = \lambda V'(x)(1+o(1)),
\]
where \(A(n)=\#\{m:\phi(m)=n\}\) on relevant range, yielding controlled excess \(V-V'\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (set inclusion)
\[
\{\phi(m):m\le x\}\subseteq \{n\le x:\exists m,\phi(m)=n\},
\]
hence \(V'(x)\le V(x)\).

**Proof.**
Any value attained by \(m\le x\) is a totient value \(\le x\), so counted by \(V(x)\). \(\square\)

#### Lemma B2 (difference as delayed-preimage count)
\[
V(x)-V'(x)
\]
counts totient values \(n\le x\) whose preimages lie entirely above \(x\).

**Proof.**
Subtract values already hit by \(m\le x\) from all totient values \(\le x\). \(\square\)

#### Lemma B3 (A implies ratio limit)
If A controls excess multiplicity proportionally to \(V'(x)\), then
\[
\frac{V(x)}{V'(x)}\to 1+\lambda.
\]

**Proof.**
Write \(V=V'+(V-V')\), divide by \(V'\), apply A asymptotic. \(\square\)

### C. Main unresolved point

The core gap is understanding distribution of large preimages for small totient values (the delayed-preimage phenomenon in B2).
