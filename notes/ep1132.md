# EP-1132

## Problem Statement
For an infinite node sequence $(x_k)_{k\ge1}\subset[-1,1]$, with Lebesgue function
\[
L_n(x)=\sum_{k=1}^n |\ell_k(x)|,
\]
ask for lower-limsup growth at the $(2/\pi)\log n$ scale:
for repeated $n$ and (in stronger versions) for almost every $x$.

## Literature
- Classical fixed-$n$ bound:
\[
\max_{x\in[-1,1]} L_n(x) \ge \frac{2}{\pi}\log n - O(1).
\]
- Bernstein-type results imply dense sets of points with limsup behavior at this scale.
- Full infinite-sequence almost-everywhere formulations remain subtle/open in the cited background.

## Our Approaches / What Is Proven
- Added deep standalone finite-prefix study (`scripts/ep1132.mjs`) for several infinite-sequence models:
  - `quasi_cos`,
  - `random_uniform`,
  - `leja_like_grid` (discrete Leja-like construction).
- For each model and many checkpoints $n\le 480$, computed on a dense grid:
  - $\max_x L_n(x)$,
  - ratio $\max_x L_n(x)/\log n$,
  - exceedance versus threshold $(2/\pi)\log n$.

## Computation-Guided Observations
Deep run (`DEPTH=5`, `Nmax=480`, grid `1601`) shows:
- all tested models exceed the $(2/\pi)\log n$ threshold strongly on many checkpoints;
- for `random_uniform` and `leja_like_grid`, finite-$n$ maxima can become extremely large, indicating very high Lebesgue growth for these constructions;
- `quasi_cos` also exceeds threshold robustly, though with less explosive behavior.

Caveat: this is finite-grid finite-$n$ evidence only; it does not settle the full infinite-sequence almost-everywhere statement. Very large constants are expected for poorly conditioned node families and should be interpreted as growth evidence, not as exact asymptotics.

## Deeper Proof Program (A => B => C)

### Target
For infinite node sequences \((x_k)\), force repeated/ae lower-limsup growth at scale \((2/\pi)\log n\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a uniform prefix-stability lemma: for infinitely many \(n\), every interval of \(x\)-mass at least \(\eta\) contains points where
\[
L_n(x)\ge \frac{2}{\pi}\log n - C_\eta.
\]

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (global max lower bound)
For each fixed \(n\),
\[
\max_{x\in[-1,1]}L_n(x)\ge \frac{2}{\pi}\log n - O(1).
\]

**Proof.**
This is the classical Erdős-Bernstein-type lower bound cited in the literature summary. \(\square\)

#### Lemma B2 (many-point transfer from max control)
If one has an \(L^1\) lower bound for \(L_n\) of the same logarithmic scale plus an \(L^\infty\) upper envelope \(O(\log n)\), then a positive-measure set of \(x\) must satisfy \(L_n(x)\gg\log n\).

**Proof.**
Chebyshev/Markov measure estimate: if too small on most points, integral would contradict lower \(L^1\) mass. \(\square\)

#### Lemma B3 (A implies limsup statement)
If Lemma A holds for infinitely many \(n\), then
\[
\limsup_{n\to\infty}\frac{L_n(x)}{\log n}\ge \frac{2}{\pi}
\]
on a positive-measure set (and with stronger uniformity, almost everywhere).

**Proof.**
Take subsequence \(n_j\) from A; pointwise lower bounds along \(n_j\) give limsup lower bound. \(\square\)

### C. Main unresolved point

The gap is promoting fixed-\(n\) maximal lower bounds to robust distribution-in-\(x\) statements for an arbitrary infinite node sequence.
