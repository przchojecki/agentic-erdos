# EP-959

## Proof Attempts and Literature Notes

### Working statement
For an `n`-point planar set and distance multiplicities \(f(d)\), study the top-gap
\(f(d_1)-f(d_2)\) between the two largest multiplicities.

### Deep standalone computation (`ep959.mjs`)
- Method: exact squared-distance multiplicity histograms for candidate families plus heavy random search.
- Families tested per `n`: `grid`, `two_lines`, `parabola`, and `2500` random samples.
- Deep run: `n in {120,180,240,300,360}`.

### Results
Best family was always `two_lines`, with gaps:
- `n=120`: gap `60`
- `n=180`: gap `90`
- `n=240`: gap `120`
- `n=300`: gap `150`
- `n=360`: gap `180`

Normalized values `gap/(n log n)` stay positive in range `~0.085..0.104`.

### Interpretation
At this scale, explicit two-line constructions dominate random search and produce robust positive normalized gaps, consistent with lower-bound direction.

### Rigorous construction lemma
Let \(n=2m\), and take
\[
P=\{(i,0):1\le i\le m\}\cup\{(i,1):1\le i\le m\}.
\]
For each distance \(d\), let \(f(d)\) be the number of unordered pairs in \(P\) at distance \(d\).

Define \(d_1=1\). Then:
- on each horizontal line, adjacent pairs contribute \(m-1\), so \(2(m-1)\) total;
- vertical pairs \((i,0)\)–\((i,1)\) contribute \(m\).

Hence
\[
f(d_1)=2(m-1)+m=3m-2.
\]

For \(d_2=\sqrt2\), diagonal cross-line pairs contribute
\[
(m-1)+(m-1)=2m-2.
\]
Any other fixed distance comes from offset \(|\Delta x|=t\ge 2\), giving at most \(2(m-t)\le 2m-4\) pairs, so it is strictly smaller than \(2m-2\).
Therefore
\[
\max_{d\ne d_1} f(d)=2m-2.
\]

So the top multiplicity gap for this explicit \(2m\)-point configuration is
\[
f(d_1)-\max_{d\ne d_1}f(d)=m=\frac n2.
\]
This gives a clean linear lower bound with explicit witness family.

### Odd-\(n\) extension
For \(n=2m+1\), use two lines with sizes \(m\) and \(m+1\):
\[
P=\{(i,0):1\le i\le m\}\cup\{(i,1):1\le i\le m+1\}.
\]
Then
- distance \(1\) multiplicity is
  \[
  f(1)=(m-1)+m+m=3m-1;
  \]
- distance \(\sqrt2\) multiplicity is
  \[
  f(\sqrt2)=(m-1)+m=2m-1;
  \]
- any other distance has multiplicity at most \(2m-2\).

Hence the gap is again exactly
\[
f(1)-\max_{d\ne 1}f(d)=m=\left\lfloor\frac n2\right\rfloor.
\]

So this explicit two-line family gives
\[
\max_d f(d)-\max_{d'\ne d} f(d')\ \ge\ \left\lfloor\frac n2\right\rfloor
\]
for every \(n\).
