# EP-176 deeper attempt

## Statement
`N(k,ell)` is the first `N` such that every `f:[1,N]->{-1,1}` has a `k`-term AP with
`|sum_{P} f| >= ell`.

## Exact finite results proved in this attempt
I used exact backtracking to determine small thresholds.

Data file:
- `data/ep176_ap_discrepancy_small_exact.json`

Exact values obtained:
- `N(3,2)=9`
- `N(4,2)=13`
- `N(5,2)=22`
- `N(6,2)=11`
- `N(6,3)=42`

## Interpretation
These are rigorous finite thresholds; the exponential-in-`k` regime from the main question
remains open.

## Deeper Proof Program (A => B => C)

### Target
Determine asymptotic growth of \(N(k,\ell)\) (especially \(N(k,2)\)) and decide
whether it is exponential in \(k\).

### Step A: Discrepancy lower constructions
Build \(\{-1,1\}\)-colorings on long intervals keeping all \(k\)-AP sums below
\(\ell\).

### Step B: Fourier/entropy upper forcing
Show sufficiently long intervals force a \(k\)-AP with discrepancy at least
\(\ell\).

### Step C: Asymptotic matching
Align lower and upper growth rates.

### Concrete Blocking Lemma (Most Critical)
Need a sharp large-deviation estimate for AP-sum distributions under optimized
colorings, uniform in \(k\).

## LaTeX Proofs for A => B => C Skeleton

### Proposition BQ1 (Monotonicity in \(N\))
For fixed \(k,\ell\), if every coloring of \([1,N]\) has a \(k\)-AP with
\(|\sum_P f|\ge \ell\), then the same holds for all \(N'>N\). Hence \(N(k,\ell)\)
is well-defined once existence is known.

\[
\textit{Proof.}
\]
Restrict any coloring on \([1,N']\) to \([1,N]\).
\(\square\)

### Proposition BQ2 (Exponential-scale reduction)
If one proves constants \(a_\ell,b_\ell>1\) with
\[
a_\ell^k \le N(k,\ell)\le b_\ell^k
\]
for large \(k\), then the growth is exponential in \(k\).

\[
\textit{Proof.}
\]
Take logarithms and divide by \(k\).
\(\square\)
