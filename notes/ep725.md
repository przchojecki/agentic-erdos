# EP-725

## Problem Statement
Give an asymptotic formula for the number of \(k\times n\) Latin rectangles.

## Literature
- Broad asymptotic ranges are known, with template behavior
\[
\#\mathrm{LR}(k,n) \asymp e^{-\binom{k}{2}} (n!)^k
\]
in substantial regimes of \(k\) versus \(n\).
- Full uniform asymptotic control across all \(k\)-ranges remains open.

## Our Approaches / What Is Proven
- Implemented exact counting for small \((n,k)\) via backtracking with column constraints.
- Compared exact values against the asymptotic template factor
\[
e^{-\binom{k}{2}} (n!)^k.
\]

## Computation-Guided Observations
- Deep exact run (up to \(n=6, k\le4\)) shows:
  - for \(k=2\), ratio exact/template is very close to 1,
  - for larger \(k\), finite-size deviations are larger (ratios around 0.31-0.84 in tested range).
- This is consistent with the template being asymptotic rather than exact at small scales.

## Status
- Largely resolved in broad regimes; fully uniform range still open.
- Added standalone exact finite benchmark counts against the main asymptotic template.

## Added Proof Layer (No New Computation)

### What Is Closed
Known theorems already provide the principal asymptotic shape in broad
\((k,n)\)-regimes:
\[
\#\mathrm{LR}(k,n)\asymp e^{-\binom{k}{2}}(n!)^k.
\]
So the main-order template is established in large parameter regions.

### What Remains Open
The unresolved part is a fully uniform asymptotic formula across all growth
relations between \(k\) and \(n\), with controlled error terms.

### Blocking Lemma
- Need a uniform dependency-control expansion (or cluster estimate) for column
  collision constraints that remains sharp simultaneously across all \(k\)-scales.

## Deeper Proof Program (A => B => C)

### Target
A single asymptotic formula with explicit error term valid uniformly for all
relevant \(k=k(n)\) ranges.

### Step A: Uniform Local Constraint Expansion
Express Latin-rectangle count as product of conditional matchings with a
uniform error model for column conflicts.

### Step B: Global Error Summation
Sum local errors over rows while keeping total multiplicative error
\(\exp(o(k^2))\) (or sharper) uniformly in \(k\).

### Step C: Regime Stitching
Match sparse-\(k\), medium-\(k\), and dense-\(k\) asymptotics with consistent
overlap formulas to yield one global statement.

### Concrete Blocking Lemma (Most Critical)
Need a coupling theorem for random near-Latin row extensions whose dependency
graph has spectral gap/control uniform in \(k\), preventing error blow-up in
intermediate \(k\)-windows.
