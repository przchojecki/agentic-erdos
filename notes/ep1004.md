# EP-1004 partial attempt

## Route
Scanned consecutive windows of totient values to find long runs where
`phi(n+1),...,phi(n+K)` are all distinct.

## Evidence from this batch
- `data/ep1004_phi_distinct_run_scan.json`:
  - best run lengths up to `271` by `x=4e6`.
  - these exceed `(log x)^c` for tested `c in {0.5,1,1.5,2}`.

## Hard point
The problem asks existence for every fixed `c>0` asymptotically; finite scans for selected `c` do not prove the full statement.

## Status
Large finite distinct runs found; full asymptotic theorem remains open.

## Collision-count identity for totient windows
For an interval \(I=[m+1,m+K]\), let
\[
C_I:=\#\{(a,b)\in I^2:\ a<b,\ \varphi(a)=\varphi(b)\}.
\]
Then
\[
\varphi(m+1),\dots,\varphi(m+K)\ \text{all distinct}
\iff
C_I=0.
\]

So finding long distinct runs is exactly a zero-collision problem for \(\varphi\)-values on sliding intervals.

### Counting-function reformulation
Define
\[
L(X):=\max\{K:\exists\,m\le X,\ \varphi(m+1),\dots,\varphi(m+K)\ \text{pairwise distinct}\}.
\]
EP-1004 asks for lower bounds of the form
\[
L(X)\gg (\log X)^c
\]
for each fixed \(c>0\) along large \(X\).

Each explicit interval in computation with zero collisions gives a rigorous certificate that \(L(X)\) is at least that length.

### Basic lower bound
For all \(X\ge1\),
\[
L(X)\ge 1.
\]
This is immediate from a one-term window.

### Monotonicity
\[
L(X+1)\ge L(X).
\]
Proof: any interval witnessing \(L(X)\) is still admissible at level \(X+1\).

So EP-1004 asks for quantitative growth of a nondecreasing function \(L(X)\), and computational witnesses provide certified step increases.

### Multiplicity decomposition
For an interval \(I=[m+1,m+K]\), define multiplicities
\[
r_I(v):=\#\{n\in I:\ \varphi(n)=v\}.
\]
Then
\[
\sum_v r_I(v)=K,\qquad
C_I=\sum_v \binom{r_I(v)}{2}.
\]
Hence
\[
C_I=0 \iff r_I(v)\in\{0,1\}\ \forall v
\]
(equivalently all \(\varphi\)-values in the window are distinct).

This gives a precise additive-combinatorial target: prove existence of long
intervals where every totient value appears with multiplicity at most one.

## Deeper Proof Program (A => B => C)

### Target
Show for every fixed \(c>0\), there exist arbitrarily large \(X\) with
\[
L(X)\gg (\log X)^c.
\]

### A. Concrete Blocking Lemma (Most Critical)

Prove a sparse-collision theorem: for many intervals \(I=[m+1,m+K]\) with
\(K\le (\log m)^C\), one has
\[
\sum_v \binom{r_I(v)}{2}=o(1),
\]
hence zero collisions for infinitely many such intervals.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (collision equivalence)
\[
C_I=0 \iff \varphi(m+1),\dots,\varphi(m+K)\ \text{are pairwise distinct}.
\]

**Proof.**
By definition, \(C_I\) counts equal-value pairs in the window. \(\square\)

#### Lemma B2 (multiplicity decomposition)
\[
C_I=\sum_v \binom{r_I(v)}{2}.
\]

**Proof.**
For each value \(v\), choose unordered pairs among \(r_I(v)\) occurrences. Summing over \(v\) counts all equal-value pairs exactly once. \(\square\)

#### Lemma B3 (A implies polylog lower bounds)
If A holds for \(K=(\log m)^c\), then \(L(m)\ge K\) for infinitely many \(m\).

**Proof.**
A gives infinitely many windows with \(C_I=0\); by Lemma B1 each provides a distinct-run witness of length \(K\). \(\square\)

### C. Main unresolved point

The key gap is obtaining sufficiently strong short-interval control on repeated totient values to force collision-free windows at polylogarithmic lengths.
