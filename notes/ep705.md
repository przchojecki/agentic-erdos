# EP-705

## Problem Statement
Let \(G\) be a finite unit-distance graph in \(\mathbb R^2\). Is there \(k\) such that girth\((G)\ge k\) implies \(\chi(G)\le 3\)?

## Literature
- This is now known to be false: there exist unit-distance graphs of arbitrarily large girth and chromatic number at least 4.

## Our Approaches / What Is Proven
- Verified exact properties of the classical 7-vertex Moser spindle witness:
  - exact chromatic number via backtracking coloring,
  - exact girth via BFS cycle search.
- This does not itself give large girth, but confirms the fundamental 4-chromatic unit-distance obstruction.

## Computation-Guided Observations
- For the Moser spindle: \(|V|=7\), \(|E|=11\), \(\chi=4\), girth \(=3\).
- Combined with modern large-girth constructions from the literature, this closes the original question negatively.
