# EP-1075

## Problem Statement
For $r\ge 3$, if an $r$-uniform hypergraph on $n$ vertices has more than
\[
(1+\varepsilon)\left(\frac{n}{r}\right)^r
\]
edges, must it contain a subhypergraph on a growing number of vertices with edge density at least some constant
\[
c_r>r^{-r}?
\]

## Literature
- A denser-regime Erdős theorem gives a guaranteed constant at level $c_r=r^{-r}$ under a stronger global edge hypothesis.
- The sharpened sparse-threshold / improved-constant version remains open in this repository.

## Our Approaches / What Is Proven
- Added standalone exploratory computation (`scripts/ep1075.mjs`) on random $r$-uniform hypergraphs near the threshold.
- For each sampled instance, used greedy peeling to track best induced density
\[
\frac{e(U)}{\binom{|U|}{r}}
\]
over intermediate subset sizes.
- Recorded hit rates above baseline $r^{-r}$ with a growing-size proxy $|U|\ge 2\log_2 n$.

## Computation-Guided Observations
Deep run (`DEPTH=5`) over six scenarios ($r=3,4$) found frequent high finite densities on peeled subsets; all sampled trials exceeded baseline $r^{-r}$ under this heuristic objective.

Important caveat:
- This objective is strongly biased toward small/medium dense pockets,
- random-model finite evidence does not imply the extremal theorem,
- therefore this does **not** constitute proof progress, only exploratory signal.

## Status
- Open.
- Added first standalone deep heuristic experiment for finite threshold behavior, with explicit non-proof caveat.

## Deeper Proof Program (A => B => C)

### Target
Near the threshold
\[
e(H)>(1+\varepsilon)\left(\frac nr\right)^r,
\]
force a growing vertex set \(U\) with induced edge density \(>c_r\), where \(c_r>r^{-r}\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a supersaturation-to-structure lemma: if an \(r\)-graph has edge count above the threshold, then it contains \(\Omega(n)\) vertices whose local codegree profile exceeds random baseline by a fixed margin. From this, derive an induced \(U\) with density \(>r^{-r}+\delta_r\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (average degree forcing)
If \(e(H)>(1+\varepsilon)(n/r)^r\), then average vertex degree exceeds \((1+\varepsilon)\) times the threshold average.

**Proof.**
Average degree is \(r e(H)/n\). Substitute the edge lower bound. \(\square\)

#### Lemma B2 (dense core extraction)
From any \(r\)-graph, iterative deletion of vertices below current average degree terminates in a nonempty induced subgraph with minimum degree at least its own average degree.

**Proof.**
Standard peeling: each deletion increases average degree of remaining graph; process stops when no vertex is below average. \(\square\)

#### Lemma B3 (A plus B2 implies target)
If Lemma A gives a core with degree/profile surplus \(\delta_r\), then the induced density of that core exceeds \(r^{-r}+\delta'_r\) for some \(\delta'_r>0\).

**Proof.**
Translate degree surplus to edge-count surplus through \(e(U)=\frac1r\sum_{v\in U}\deg_U(v)\), then normalize by \(\binom{|U|}{r}\). \(\square\)

### C. Main unresolved point

The missing ingredient is a robust local-structure theorem at this sparse threshold: current methods either need denser global assumptions or lose constants at exactly the range where \(c_r>r^{-r}\) is desired.
