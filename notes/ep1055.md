# EP-1055

## Problem Statement
Recursive prime-class problem based on prime factors of $p+1$: determine long-range behavior (including infinitude in each class).

## Literature
- The recursive class definition is classical; asymptotic infinitude questions remain open.

## Our Approaches / What Is Proven
- Added standalone deep computation up to
\[
p\le 21{,}000{,}000.
\]
- Computed class values recursively, first prime in each class, and population by class.

## Computation-Guided Observations
- This scan reaches class $11$ (first prime found: $8{,}524{,}807$).
- First-prime sequence for classes starts
\[
2,13,37,73,1021,2917,15013,49681,532801,1065601,8524807.
\]
- Finite evidence supports continued class growth, but does not prove infinitude in every class.

- Open.
