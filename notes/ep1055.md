## Problem Statement

## Deeper Proof Program (A => B => C)

### Target
Prove infinitude (or classify obstructions) for every recursive prime class defined via prime factors of \(p+1\).

### A. Concrete Blocking Lemma (Most Critical)

Establish a lifting lemma: if class \(j\) contains infinitely many primes in residues where \(p+1\) has a prime factor in class \(j-1\), then class \(j+1\) is infinite.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (class recursion monotonicity)
If \(q\mid p+1\) and \(q\) is in class \(j\), then \(p\) belongs to class at least \(j+1\) by definition.

**Proof.**
Immediate from recursive class rule based on factors of \(p+1\). \(\square\)

#### Lemma B2 (first-prime certification)
Exhibiting one prime in class \(j\) certifies nonemptiness of all lower classes along its ancestry chain.

**Proof.**
Trace recursive witnesses \(q\mid p+1\) downward through classes. \(\square\)

#### Lemma B3 (A implies infinitude induction)
If A holds for every \(j\), induction gives infinitude of each class.

**Proof.**
Base class is infinite by prime infinitude. Apply A repeatedly to lift infinitude one class at a time. \(\square\)

### C. Main unresolved point

The bottleneck is obtaining distributional control of prime factors of \(p+1\) inside specific recursive classes, beyond finite-window evidence.
