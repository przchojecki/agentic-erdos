# EP-986

## Proof Attempts and Literature Notes

### Working statement
For off-diagonal Ramsey numbers, determine asymptotic lower growth (fixed \(k\), large \(t\)).

### Exact graph-theoretic equivalence (for \(k=3\))
\[
R(3,t)>n
\iff
\exists\ \text{triangle-free graph }G\text{ on }n\text{ vertices with }\alpha(G)<t.
\]

Proof:
- If such \(G\) exists, then \(G\) has no \(K_3\) and no independent set of size \(t\), so by definition \(R(3,t)\) must exceed \(n\).
- Conversely, \(R(3,t)>n\) means there exists a 2-coloring of \(K_n\) with no red triangle and no blue \(K_t\). Taking red edges gives a triangle-free graph with \(\alpha<t\).

So computational construction of triangle-free graphs with small independence number is exactly the right witness mechanism for lower bounds on \(R(3,t)\).

### Computation context
Triangle-free process samples in this project produced witnesses such as \(R(3,12)>30\), consistent with this exact equivalence.

### Status
- General fixed-\(k\) conjectural growth remains open.
- For \(k=3\), witness-based lower bounds and known asymptotic theory are aligned with the above equivalence.

### Density constraint via Caro--Wei
For any graph \(G\) with degrees \(d_v\), Caro--Wei gives
\[
\alpha(G)\ge \sum_{v}\frac1{d_v+1}.
\]
By Jensen/AM-HM on fixed average degree \(\bar d=2m/n\),
\[
\sum_v \frac1{d_v+1}\ge \frac{n}{\bar d+1}
=\frac{n^2}{2m+n}.
\]
Hence any triangle-free witness for \(R(3,t)>n\) (so \(\alpha(G)<t\)) must satisfy
\[
\frac{n^2}{2m+n}<t
\quad\Longrightarrow\quad
m>\frac{n(n-t)}{2t}.
\]

So low-independence witnesses necessarily lie in a quantitatively dense edge regime.

### Asymptotic anchor for \(k=3\)
Unconditionally, there exist absolute constants \(c_1,c_2>0\) such that for all
sufficiently large \(t\),
\[
c_1\frac{t^2}{\log t}\le R(3,t)\le c_2\frac{t^2}{\log t}.
\]
So for \(k=3\), the order of growth is known:
\[
R(3,t)=\Theta\!\left(\frac{t^2}{\log t}\right).
\]

This means computational witness-building is primarily about improving constants
or finite-\(t\) records, not the asymptotic exponent/form for \(k=3\).
