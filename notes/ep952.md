# EP-952

## Proof Attempts and Literature Notes

### Working statement
Does there exist an infinite path through Gaussian primes with uniformly bounded step length?

### Deep standalone computation (`ep952.mjs`)
- Method: finite-window graph on Gaussian primes in `[-R,R]^2`; edge between primes at Euclidean distance `<= D`; BFS from the smallest-norm prime.
- Deep run: `R=1200`, `D in {3,4,5,6,7,8,10,12}`.
- Point count in window: `549,372` Gaussian primes.

### Results
- `D=3`: component size `2,996`, max reached radius `~93.47`, does **not** reach boundary.
- `D=4`: component size `530,604`, reaches boundary.
- `D=5`: component size `548,612`, reaches boundary.
- `D>=6`: full window connectivity from start component (`549,372` visited), reaches boundary.

### Interpretation
Finite-window threshold behavior remains consistent with a moat around step size near `3` and strong connectivity by `4+` in this scale. This is still finite evidence, not an infinite-plane proof.
