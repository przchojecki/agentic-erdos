# EP-963 partial resolution

## Statement
f(n): guaranteed size of a dissociated subset in any n-element set of reals. Ask in
particular if f(n)>=floor(log_2 n).

## What is resolved
Background records greedy lower bound:
f(n)>=floor(log_3 n).

## What remains open in this note
Closing the base-3 to base-2 gap is unresolved.

## Status
- logarithmic lower bound known.
- stronger log_2 threshold remains open.

## Proof details for the base-3 lower bound

### Definitions
A finite set \(D=\{d_1,\dots,d_m\}\subset \mathbb R\) is dissociated if the only relation
\[
\sum_{i=1}^m \varepsilon_i d_i = 0,\qquad \varepsilon_i\in\{-1,0,1\},
\]
is the trivial one \(\varepsilon_i=0\) for all \(i\).

For \(D\), define its \(\{-1,0,1\}\)-span:
\[
\operatorname{Span}_{\pm1}(D)=\left\{\sum_{i=1}^m \varepsilon_i d_i:\ \varepsilon_i\in\{-1,0,1\}\right\}.
\]

### Lemma 1 (distinct coefficient vectors)
If \(D\) is dissociated and \(|D|=m\), then all \(3^m\) coefficient vectors
\((\varepsilon_1,\dots,\varepsilon_m)\in\{-1,0,1\}^m\) give distinct sums.

Proof:
If two vectors gave the same sum, subtracting yields a nontrivial
\(\{-1,0,1\}\)-relation on \(D\), contradiction.

Hence
\[
|\operatorname{Span}_{\pm1}(D)|=3^m.
\]

### Lemma 2 (maximal dissociated subsets span the ambient set in \(\pm1\)-sense)
Let \(A\subset\mathbb R\) be finite, and let \(D\subseteq A\) be maximal dissociated (by inclusion).
Then every \(a\in A\) lies in \(\operatorname{Span}_{\pm1}(D)\).

Proof:
If some \(a\notin \operatorname{Span}_{\pm1}(D)\), then \(D\cup\{a\}\) is still dissociated:
any nontrivial \(\{-1,0,1\}\)-relation involving \(a\) would rearrange to
\(a\in \operatorname{Span}_{\pm1}(D)\), contradiction.
This violates maximality.

So \(A\subseteq \operatorname{Span}_{\pm1}(D)\).

### Proposition (classical lower bound)
For every \(n\), any \(n\)-element set \(A\subset\mathbb R\) contains a dissociated subset of size at least \(\lfloor \log_3 n\rfloor\). Equivalently,
\[
f(n)\ge \lfloor \log_3 n\rfloor.
\]

Proof:
Take maximal dissociated \(D\subseteq A\), \(|D|=m\).
By Lemma 2, \(n=|A|\le |\operatorname{Span}_{\pm1}(D)|\).
By Lemma 1, \(|\operatorname{Span}_{\pm1}(D)|=3^m\).
Thus \(n\le 3^m\), so \(m\ge \log_3 n\), hence \(m\ge \lfloor\log_3 n\rfloor\).
