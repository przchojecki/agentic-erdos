# EP-1109

## Problem Statement
Let $f(N)$ be the largest size of a set $A\subseteq\{1,\dots,N\}$ such that every element of $A+A$ is squarefree.
Estimate $f(N)$.

## Literature
- Known bounds leave a large gap; conjectural direction is very sparse growth (possibly polylogarithmic scale).

## Our Approaches / What Is Proven
- Replaced stub with standalone max-clique formulation (`scripts/ep1109.mjs`):
  - vertices: $x\in[1,N]$ with $2x$ squarefree,
  - edge $(x,y)$ if $x+y$ squarefree,
  - then $f(N)$ is clique number.
- Used branch-and-bound with greedy coloring bounds.
- Reports exact values when fully solved; timed-out rows are certified lower bounds.

## Deeper Proof Program (A => B => C)

### Target
Estimate
\[
f(N)=\max\{|A|:\ A\subseteq[1,N],\ A+A\subseteq\{\text{squarefree}\}\}.
\]

### A. Concrete Blocking Lemma (Most Critical)

Show a modular obstruction theorem: for a suitable set of square moduli \(m_j^2\), any admissible \(A\) must avoid a positive fraction of residue classes modulo each \(m_j^2\), and these constraints combine multiplicatively to force
\[
f(N)\le N^{1/2+o(1)}
\]
or better.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (graph equivalence)
Define graph \(G_N\) on \([1,N]\) with edge \(\{x,y\}\) iff \(x+y\) is squarefree and require \(2x\) squarefree for vertices in admissible sets. Then
\[
f(N)=\omega(G_N).
\]

**Proof.**
Condition \(A+A\) squarefree means every pair sum (including doubles) is squarefree, exactly clique plus vertex self-condition in \(G_N\). \(\square\)

#### Lemma B2 (residue-class forbiddance)
For each prime \(p\), if \(x\equiv a\pmod{p^2}\), then elements congruent to \(-a\pmod{p^2}\) cannot co-occur with \(x\) inside \(A\).

**Proof.**
Otherwise \(x+y\equiv0\pmod{p^2}\), so \(x+y\) is not squarefree. \(\square\)

#### Lemma B3 (density transfer)
If residue restrictions imply upper density \(\delta_M\) modulo \(M\), then \(f(N)\le \delta_M N+O(M)\).

**Proof.**
Count admissible elements by residue classes mod \(M\) plus boundary error. \(\square\)

### C. Main unresolved point

The key gap is optimizing coupled residue exclusions across many \(p^2\) simultaneously; single-modulus bounds are too weak.
