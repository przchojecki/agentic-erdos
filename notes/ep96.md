## Deeper Proof Program (A => B => C)

### Target
Improve the upper bound for the maximum number of unit distances among vertices
of a convex \(n\)-gon, ideally toward linear.

### Step A: Convex-order encoding
Represent unit chords by index differences and crossing constraints in cyclic
order.

### Step B: Forbidden pattern elimination
Show dense unit-chord patterns force local geometric contradictions (angle/length
monotonicity) in convex position.

### Step C: Global counting
Convert forbidden patterns into a linear (or near-linear) extremal bound on
number of unit chords.

### Concrete Blocking Lemma (Most Critical)
Need a sharp combinatorial-geometric lemma controlling coexistence of many unit
chords across multiple index gaps; current arguments leave an \(n\log n\) slack.

## LaTeX Proofs for A => B => C Skeleton

### Proposition V1 (Crossing monotonicity constraint)
In a strictly convex polygon, if unit chords are indexed by endpoints
\((i,j)\), then for fixed \(i\), the set of \(j\) with \(|v_i-v_j|=1\) forms at
most two contiguous intervals along cyclic order.

\[
\textit{Proof sketch.}
\]
Distance from \(v_i\) to vertices along each side of the polygon is unimodal by
strict convexity; level set \(|v_i-v_j|=1\) intersects each monotone branch in at
most one interval.
\(\square\)

### Proposition V2 (Interval model reduction)
If each vertex contributes at most \(g(n)\) admissible index-interval endpoints
for unit chords, then total number \(U(n)\) of unit chords satisfies
\[
U(n)\le \frac12 n\,g(n).
\]

\[
\textit{Proof.}
\]
Sum per-vertex unit incidences and divide by two.
\(\square\)
