# EP-810

## Problem Statement
Does there exist \(\varepsilon>0\) such that for all large \(n\), one can find an \(n\)-vertex graph with at least \(\varepsilon n^2\) edges and an \(n\)-color edge-coloring in which every \(C_4\) has 4 distinct edge colors?

## Literature
- Global asymptotic existence/nonexistence remains open in the current notes.

## Our Approaches / What Is Proven
- Built a random-greedy constructor:
  - process edges in random order,
  - assign one of \(n\) colors only if every newly created \(C_4\) remains 4-distinct.
- Measured best and average achieved densities over multiple runs.

## Computation-Guided Observations
- For \(n=30,40,50\), best densities were around 0.16-0.18 in this heuristic.
- All successful best runs used essentially all \(n\) colors.
- Finite positive-density constructions exist in these ranges under this method, but no asymptotic guarantee follows.
