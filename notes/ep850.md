# EP-850

## Problem Statement
Can there exist distinct integers $x,y$ such that:
- $x,y$ have the same prime supports,
- $x+1,y+1$ have the same prime supports,
- $x+2,y+2$ have the same prime supports?

## Literature
- Erdos-Woods type conjecture.
- For the 2-shift pattern $(x,x+1)$, known examples include the classical pair $(75,1215)$.
- The 3-shift pattern $(x,x+1,x+2)$ remains open.

## Our Approaches / What Is Proven
- Ran a deeper finite collision scan up to $x\le 2\cdot 10^6$.
- Built prime-support signatures for $n$ and compared both:
  - pair keys $(x,x+1)$,
  - triple keys $(x,x+1,x+2)$.

## Computation-Guided Observations
- No collisions found for triple keys $(x,x+1,x+2)$ in this range.
- Pair-key collisions were found, including the known $(75,1215)$ example.
- Finite evidence supports rarity of 3-shift collisions, but does not settle nonexistence.
