# EP-1003 partial attempt

## Route
Computed exact counts of solutions to `phi(n)=phi(n+1)` up to `x=5e6`.

## Evidence from this batch
- `data/ep1003_phi_adjacent_equal_scan.json` found `108` solutions up to `5e6`.
- First solutions include `n=3,15,104,164,...`.

## Hard point
Positive counts in finite ranges do not prove infinitely many solutions.

## Status
No proof of infinitude or counterexample; finite data supports continued occurrence.

## Exact arithmetic reformulation
Using
\[
\varphi(n)=n\prod_{p\mid n}\left(1-\frac1p\right),
\]
the equation
\[
\varphi(n)=\varphi(n+1)
\]
is equivalent to
\[
\frac{n+1}{n}
=
\frac{\prod_{p\mid n}\left(1-\frac1p\right)}
{\prod_{q\mid n+1}\left(1-\frac1q\right)}.
\]
Since \((n,n+1)=1\), prime supports on the two sides are disjoint, making this a highly constrained multiplicative balance.

### Parity lemma
If \(n>1\) and \(\varphi(n)=\varphi(n+1)\), then both values are even.

Proof:
\(\varphi(m)\) is even for every \(m>2\). Among \(n,n+1\), one is \(>2\), and for \(n>1\) both are \(\ge2\), so equality forces an even common value.

### Certificate principle
Each explicit \(n\) with \(\varphi(n)=\varphi(n+1)\) is a rigorous witness; therefore the data file provides certified lower bounds on counting function
\[
A(X):=\#\{n\le X:\varphi(n)=\varphi(n+1)\}.
\]
Current computations establish \(A(5\cdot 10^6)\ge 108\).
