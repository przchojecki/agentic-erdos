# EP-1003 partial attempt

## Route
Computed exact counts of solutions to `phi(n)=phi(n+1)` up to `x=5e6`.

## Evidence from this batch
- `data/ep1003_phi_adjacent_equal_scan.json` found `108` solutions up to `5e6`.
- First solutions include `n=3,15,104,164,...`.

## Hard point
Positive counts in finite ranges do not prove infinitely many solutions.

## Status
No proof of infinitude or counterexample; finite data supports continued occurrence.

## Exact arithmetic reformulation
Using
\[
\varphi(n)=n\prod_{p\mid n}\left(1-\frac1p\right),
\]
the equation
\[
\varphi(n)=\varphi(n+1)
\]
is equivalent to
\[
\frac{n+1}{n}
=
\frac{\prod_{p\mid n}\left(1-\frac1p\right)}
{\prod_{q\mid n+1}\left(1-\frac1q\right)}.
\]
Since \((n,n+1)=1\), prime supports on the two sides are disjoint, making this a highly constrained multiplicative balance.

### Parity lemma
If \(n>1\) and \(\varphi(n)=\varphi(n+1)\), then both values are even.

Proof:
\(\varphi(m)\) is even for every \(m>2\). Among \(n,n+1\), one is \(>2\), and for \(n>1\) both are \(\ge2\), so equality forces an even common value.

### Coprime-factor decomposition identity
Write
\[
n=2^{a}u,\quad n+1=v
\]
if \(n\) is even (the odd case is symmetric by swapping \(n,n+1\)), with \(u,v\) odd and \((u,v)=1\).
Then
\[
\varphi(n)=2^{a-1}\varphi(u),\qquad
\varphi(n+1)=\varphi(v),
\]
so any solution must satisfy
\[
2^{a-1}\varphi(u)=\varphi(v).
\]
Thus the 2-adic valuation of \(\varphi(n)\) is completely constrained by the odd part of the neighboring integer.

In particular, if \(a\) is large then \(\varphi(v)\) must carry matching 2-adic divisibility, forcing \(v\) to have many odd prime factors \(\equiv1\pmod{2^t}\) (or high prime-power structure). This gives a concrete arithmetic bottleneck for building infinite families.

### Certificate principle
Each explicit \(n\) with \(\varphi(n)=\varphi(n+1)\) is a rigorous witness; therefore the data file provides certified lower bounds on counting function
\[
A(X):=\#\{n\le X:\varphi(n)=\varphi(n+1)\}.
\]
Current computations establish \(A(5\cdot 10^6)\ge 108\).

## Deeper Proof Program (A => B => C)

### Target
Decide whether \(\varphi(n)=\varphi(n+1)\) has infinitely many solutions.

### A. Concrete Blocking Lemma (Most Critical)

Prove existence of an infinite parametric family \((n_t)\) such that prime-support conditions on \(n_t\) and \(n_t+1\) force
\[
\frac{\varphi(n_t)}{n_t}=\frac{\varphi(n_t+1)}{n_t+1}\cdot\frac{n_t+1}{n_t},
\]
equivalently \(\varphi(n_t)=\varphi(n_t+1)\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (ratio form)
\[
\varphi(n)=\varphi(n+1)\iff
\frac{n+1}{n}=\frac{\prod_{p\mid n}(1-1/p)}{\prod_{q\mid n+1}(1-1/q)}.
\]

**Proof.**
Substitute Euler product formula for both totients and divide. \(\square\)

#### Lemma B2 (2-adic coupling)
If \(n=2^a u\) with \(u\) odd, then
\[
2^{a-1}\varphi(u)=\varphi(n+1).
\]

**Proof.**
\(\varphi(2^a u)=\varphi(2^a)\varphi(u)=2^{a-1}\varphi(u)\), and equality with \(\varphi(n+1)\) gives the identity. \(\square\)

#### Lemma B3 (A implies infinitude)
If Lemma A produces infinitely many distinct \(n_t\), infinitude follows immediately.

**Proof.**
Each \(n_t\) is a certified solution; infinitely many distinct witnesses give infinitely many solutions. \(\square\)

### C. Main unresolved point

The bottleneck is constructing infinite neighboring integers with sufficiently synchronized prime-factor profiles to satisfy the exact multiplicative balance.
