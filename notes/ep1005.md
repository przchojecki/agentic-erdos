# EP-1005

## Proof Attempts and Literature Notes

### Working statement
The problem asks asymptotics of a Farey-sequence extremal quantity \(f(n)\) (similarly ordered run length variant).

### Exact finite-certificate principle
If a concrete Farey-sequence segment of length \(L+1\) is verified to satisfy the defining similarity-order property at level \(n\), then
\[
f(n)\ge L.
\]
So each explicit witness segment from computation is a rigorous lower-bound certificate.

### Monotonicity observation
Farey sequences are nested (\(F_n\subseteq F_{n+1}\) as sets in increasing order). Any admissible local pattern at level \(n\) persists at level \(n+1\), hence
\[
f(n+1)\ge f(n).
\]
Therefore \(f\) is nondecreasing.

### Computation context
Finite exact values in this project (up to \(n=240\)) are consistent with linear-scale growth, but asymptotic constant determination remains open.
