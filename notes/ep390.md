# EP-390 partial resolution

## Statement
Let `f(n)` be minimal `m` such that

`n! = a_1 ... a_k` with `n < a_1 < ... < a_k = m`.

Question: is there `c>0` with

`f(n)-2n ~ c * n/log n`?

## What is resolved
Background gives (Erdos--Guy--Selfridge, 1982):

`f(n)-2n asymp n/log n`.

So the scale/order of magnitude is known.

## What remains open in this note
The existence/value of the exact asymptotic constant `c` is not settled by the
provided background.

## Status
- Order of growth: resolved.
- Exact asymptotic constant: unresolved.

## Deeper Proof Program (A => B => C)

### Target
Refine known order
\[
f(n)-2n\asymp \frac{n}{\log n}
\]
to an exact asymptotic constant \(c\), if it exists.

### A. Concrete Blocking Lemma (Most Critical)

Prove a sharp local-prime packing theorem near \(2n\): the minimal extension length \(L=f(n)-2n\) is governed by a limiting prime-availability density in \((n,2n+L]\), yielding
\[
L\sim c\,\frac{n}{\log n}.
\]

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (factorization window form)
Condition \(n<a_1<\cdots<a_k=f(n)\) and \(a_1\cdots a_k=n!\) implies all prime valuations of \(n!\) must be supplied by factors in \((n,f(n)]\).

**Proof.**
Each \(a_i>n\); product identity forces exact valuation matching prime-by-prime. \(\square\)

#### Lemma B2 (shift variable)
Set \(L_n=f(n)-2n\). Existing theorem gives
\[
L_n\asymp \frac{n}{\log n}.
\]

**Proof.**
Direct restatement of Erdős-Guy-Selfridge result in shift notation. \(\square\)

#### Lemma B3 (A implies constant law)
If A gives limiting normalized shift \(L_n/(n/\log n)\to c\), then
\[
f(n)-2n\sim c\,\frac{n}{\log n}.
\]

**Proof.**
Unwrap \(L_n\) definition. \(\square\)

### C. Main unresolved point

The key gap is a second-order valuation-balance analysis precise enough to pass from two-sided order bounds to a single limiting constant.
