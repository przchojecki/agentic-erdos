## LaTeX Proofs for A => B => C Skeleton

### Proposition BI1 (Monotonicity in \(\alpha\))
For fixed \(n\), if \(0\le \alpha_1\le \alpha_2\), then
\[
F^{(3)}(n,\alpha_1)\le F^{(3)}(n,\alpha_2).
\]

\[
\textit{Proof.}
\]
Any coloring satisfying the stricter \(\alpha_2\)-condition automatically
satisfies \(\alpha_1\)-condition.
\(\square\)

### Proposition BI2 (Jump-detection reduction)
If one proves that for some \(\alpha_0\), the normalized limits
\[
L_\pm(\alpha_0):=\lim_{n\to\infty}\frac{F^{(3)}(n,\alpha_0\pm\varepsilon_n)}{g(n)}
\]
exist with \(L_-(\alpha_0)<L_+(\alpha_0)\), then a genuine asymptotic jump occurs
at \(\alpha_0\).

\[
\textit{Proof.}
\]
By definition, differing left/right limits imply discontinuity.
\(\square\)
