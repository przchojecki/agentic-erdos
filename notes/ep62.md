# EP-62 partial attempt

## Route
Compared the target common-subgraph claim with known universal subgraphs inside every graph of chromatic number `aleph_1`.

## What is known here
Background says every `aleph_1`-chromatic graph contains all sufficiently large odd cycles (chromatic number `3`).

## Hard point
Upgrading from guaranteed common `3`-chromatic structures to a common subgraph with chromatic number `4` (or `aleph_0`) is a substantial jump.

## Status
Open in this batch; no proof/counterexample.
