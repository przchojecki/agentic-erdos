### Lemma 2 (Monotone lower-bound transfer)
If for parameter \(N\) one constructs admissible object of size \(s\), then
extremal function \(F(N)\ge s\).

\[
\textit{Proof.}
\]
By definition, \(F(N)\) is maximum over admissible objects.
\(\square\)

### Lemma 3 (Iteration reduction)
If there is a one-step map sending size \(s\) to at least \(s+\Delta\) while
increasing parameter by at most \(C\), then
\[
F(N)\ge \frac{\Delta}{C}N-O(1).
\]

\[
\textit{Proof.}
\]
Iterate \(\lfloor N/C\rfloor\) times and apply Lemma 2.
\(\square\)
