# EP-190 partial

## Evidence from this batch
Computed the exact `k=3` finite case (canonical backtracking over colorings):
- script: `scripts/ep190_k3_coloring_exact.mjs`
- data: `data/ep190_k3_coloring_exact.json`

Here one seeks colorings of `[1..N]` where every 3-term AP is neither
monochromatic nor rainbow (equivalently every 3-AP uses exactly 2 colors).

## Result
Exact search gives:
- avoider exists up to `N=8`,
- no avoider at `N=9`.

So for `k=3`, this certifies `H(3)=9`.

## Limit
This is a small-`k` base case only; the main question about growth of `H(k)` as
`k->infinity` remains open.
