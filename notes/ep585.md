# EP-585

## Problem Statement
Determine the correct growth for extremal edge counts under the forbidden configuration of multiple edge-disjoint cycles sharing the same vertex set.

## Literature / Context
- Current status in prior notes: strong upper bounds of the form \(n(\log n)^{O(1)}\), with lower bounds around \(n\log\log n\) scale.
- Remaining gap is mainly polylogarithmic sharpening.

## Our Approaches / What Is Proven
- Implemented an exact cycle-mask detector for the forbidden same-vertex-set disjoint-cycle pattern.
- Ran deeper greedy maximization over all edges for \(n=8,9,10,11\) with many restarts.

## Computation-Guided Observations
- Best found values:
  - \(n=8: 19\),
  - \(n=9: 23\),
  - \(n=10: 26\),
  - \(n=11: 29\).
- Finite-\(n\) normalized ratios stay in the expected slowly growing range, consistent with near-linear times mild-log growth.

## Next Proof Targets
- Prove a sharper upper bound for the forbidden pattern using cycle-space compression on common vertex sets.
- Seek a matching constructive lower bound that tracks the same polylog profile.
