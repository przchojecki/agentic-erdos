# EP-1056

## Problem Statement
For each $k$, ask for primes $p$ admitting many disjoint consecutive intervals whose products are $1\pmod p$ (prefix-factorial collision formulation).

## Literature
- Classical small-$k$ examples are known; arbitrarily large-$k$ behavior remains open.

## Our Approaches / What Is Proven
- Added standalone deep prefix-factorial collision scan for primes up to
\[
p\le 3400.
\]
- For each prime, computed maximal collision multiplicity and earliest prime detecting each $k$ threshold.

- This is finite evidence only; it does not settle existence for all large $k$.

## Deeper Proof Program (A => B => C)

### Target
Show arbitrarily large \(k\) can be achieved by primes \(p\) admitting many disjoint consecutive intervals with product \(1\pmod p\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a multiplicative-collision growth lemma: for infinitely many \(p\), the set of prefix factorial residues
\[
F_j:=j!\pmod p
\]
contains value-collision multiplicity at least \(k\), with collision indices separated enough to realize disjoint intervals.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (interval-product to prefix collisions)
Product on interval \([a+1,b]\) equals \(1\pmod p\) iff \(F_b=F_a\).

**Proof.**
\[
\prod_{t=a+1}^{b} t \equiv b!\,(a!)^{-1}\pmod p,
\]
so equality to \(1\) is equivalent to \(b!\equiv a!\). \(\square\)

#### Lemma B2 (disjointness criterion)
If collision pairs \((a_i,b_i)\) satisfy \(b_i<a_{i+1}\), corresponding intervals are disjoint.

**Proof.**
Ordered endpoints without overlap imply disjoint index sets. \(\square\)

#### Lemma B3 (A implies arbitrarily large \(k\))
If A gives collision multiplicity \(\ge k\) with separable indices for infinitely many \(p\), then required interval family exists for each \(k\).

**Proof.**
Apply B1 to convert collisions into product-1 intervals, then B2 for disjointness. \(\square\)

### C. Main unresolved point

The key difficulty is controlling collision multiplicities of factorial residues modulo primes with enough spacing to ensure interval disjointness.
