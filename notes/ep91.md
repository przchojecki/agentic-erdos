## Deeper Proof Program (A => B => C)

### Target
Classify minimizers (up to similarity) for the number of distinct distances
determined by \(n\)-point planar sets, including uniqueness/non-uniqueness by \(n\).

### Step A: Extremal candidate parametrization
Describe candidate minimizers by low-dimensional geometric templates (lattice
patches, polygons with perturbations, glued motifs).

### Step B: Stability inequality
Show any near-minimizer is close (in Hausdorff/combinatorial sense) to one of
the templates.

### Step C: Exact minimizer extraction
Upgrade stability to exact classification at each fixed \(n\), proving uniqueness
or constructing multiple similarity classes.

### Concrete Blocking Lemma (Most Critical)
Need a quantitative rigidity statement for near-minimizers of distinct distances;
without it, template comparison is not exhaustive.

## LaTeX Proofs for A => B => C Skeleton

### Proposition T1 (Template completeness reduction)
Let \(\mathcal T_n\) be a finite template family. If every minimizer \(P\) of
distinct distances at size \(n\) is similar to some \(T\in\mathcal T_n\), then
classification reduces to checking distances within \(\mathcal T_n\).

\[
\textit{Proof.}
\]
By assumption, every minimizer lies in the union of similarity classes of
\(\mathcal T_n\). Compute each template's distinct-distance count and retain only
minimum-achieving classes.
\(\square\)

### Proposition T2 (Stability implies template completeness)
If one proves: for every \(\varepsilon>0\), sufficiently near-minimizers are
\(\varepsilon\)-close to a template in \(\mathcal T_n\), and template classes
are isolated, then exact minimizers are template-complete.

\[
\textit{Proof sketch.}
\]
Take \(\varepsilon\) smaller than isolation radius of template classes. A true
minimizer is a near-minimizer, hence must lie in one isolated class.
\(\square\)
