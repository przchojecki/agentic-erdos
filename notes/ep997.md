# EP-997

## Proof Attempts and Literature Notes

### Working statement
Investigate equidistribution of
\[
\{\alpha p_n\}\subset[0,1)
\]
for irrational \(\alpha\).

### Exact criterion (Weyl)
The sequence \(\{\alpha p_n\}\) is equidistributed mod \(1\) iff for every nonzero integer \(h\),
\[
\frac1N\sum_{n\le N} e(h\alpha p_n)\to 0.
\]

So EP-997 is exactly an exponential-sum problem along the primes.

### Equivalent discrepancy formulation
Define
\[
D_N(\alpha):=\sup_{I\subset[0,1)}\left|\frac1N\#\{n\le N:\{\alpha p_n\}\in I\}-|I|\right|.
\]
Then
\[
\{\alpha p_n\}\ \text{equidistributed}
\iff
D_N(\alpha)\to 0.
\]

### Simple obstruction lemma (rational case)
If \(\alpha\in\mathbb Q\), then \(\{\alpha p_n\}\) is not equidistributed in \([0,1)\).

Proof:
Write \(\alpha=a/b\) with integers \(a,b\ge1\), \((a,b)=1\).
Then every value \(\{\alpha p_n\}\) lies in the finite set
\[
\left\{0,\frac1b,\frac2b,\dots,\frac{b-1}{b}\right\}.
\]
An equidistributed sequence in \([0,1)\) cannot be supported on finitely many points, so equidistribution fails.

Hence the irrationality hypothesis is necessary.

### Computation context
Finite-window discrepancy scans in this project show nontrivial deviations for tested \(\alpha\), but finite \(N\) cannot decide the all-\(\alpha\) asymptotic statement.
