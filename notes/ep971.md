# EP-971

## Proof Attempts and Literature Notes

### Working statement
For reduced residue classes \(a\pmod d\), let \(p(a,d)\) be the least prime \(\equiv a\pmod d\).
Question: for all sufficiently large \(d\), does there exist \(c>0\) so that a positive proportion of reduced classes satisfy
\[
p(a,d)>(1+c)\varphi(d)\log d?
\]

### Known status from background
- Holds for infinitely many moduli \(d\).
- Uniform “all sufficiently large \(d\)” positive-proportion version remains open.

### Counting lemma for small thresholds
For \(y\ge 2\), define
\[
A_d(y):=\#\{a\in (\mathbb Z/d\mathbb Z)^*:\ p(a,d)\le y\}.
\]
Then
\[
A_d(y)\le \pi(y).
\]

Proof:
If \(p(a_1,d)=p(a_2,d)=p\), then \(a_1\equiv p\equiv a_2\pmod d\), so \(a_1=a_2\).
Thus the map
\(a\mapsto p(a,d)\) is injective on classes counted by \(A_d(y)\), and images are primes \(\le y\).

### Consequence
Taking \(y=\varepsilon\varphi(d)\log d\), this injective bound shows only a controlled fraction of classes can have very small least prime (at most \(\pi(y)/\varphi(d)\)).
So any positive-proportion claim at small scale must respect this counting barrier.

### Interpretation
This does not solve EP-971, but gives a clean structural restriction on how large the exceptional sets can be at any prescribed threshold.
