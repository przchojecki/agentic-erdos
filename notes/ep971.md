# EP-971

## Problem Statement
For reduced residue classes $a\pmod d$, let $p(a,d)$ be the least prime congruent to $a\pmod d$.
Does there exist $c>0$ such that for all sufficiently large $d$,
$$
p(a,d)>(1+c)\varphi(d)\log d
$$
for a positive proportion of reduced classes $a$?

## Literature
- Known for infinitely many moduli in classical work.
- Uniform “all sufficiently large $d$” positive-proportion version remains open.

## Our Approaches / What Is Proven
- Added standalone deep empirical distribution scan for least primes in residue classes.
- For each modulus $d$:
  - computed least prime in every reduced residue class up to a large search bound,
  - measured fraction of classes above thresholds $(1+c)\varphi(d)\log d$ for several $c$.
- Deep run included 1000+ moduli (`d=30,60,...,30000`) plus special moduli like $30030$.

## Computation-Guided Observations
- Example `d=30030` ($\varphi(d)=5760$):
  - fraction above $(1+0.5)\varphi(d)\log d$ was about `0.1165`,
  - fraction above $(1+1.0)\varphi(d)\log d$ was about `0.0630`.
- Similar positive proportions above moderate thresholds appear across many tested moduli.
- Finite evidence is compatible with the conjectural positive-proportion phenomenon, but does not prove the uniform all-large-$d$ claim.
