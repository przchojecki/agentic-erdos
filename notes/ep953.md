# EP-953

## Proof Attempts and Literature Notes

### Working statement
For measurable sets in a disk avoiding integer distances, determine maximal possible measure as a function of radius.

### Standalone finite analog computation (`ep953.mjs`)
- Method: lattice discretization (`(1/q)Z^2` points in disk), forbid exact integer distances on the grid, then run many randomized greedy independent-set searches.
- This gives **lower-bound heuristics** for a discretized model, not a proof for the continuous problem.

### Deep run
- Seed: `20260307`
- Configs `(R:q:rounds)`: `4:10:80`, `6:10:80`, `8:10:80`, `10:10:80`
- Runtime: `~10.6s`

### Results
Best independent-set sizes (and scaled estimates `|A|/q^2`):
- `R=4`: best `440`, scaled `4.40`
- `R=6`: best `670`, scaled `6.70`
- `R=8`: best `894`, scaled `8.94`
- `R=10`: best `1111`, scaled `11.11`

Observed selected density in the discretized grid decreases with `R`, while the scaled absolute size grows.

### Interpretation
The discretized model supports nontrivial large constructions but does not close the known continuous upper/lower-gap for EP-953.
