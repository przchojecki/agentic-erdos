# EP-783

## Problem Statement
Fix a constant \(C>0\), let \(N\) be large, and let \(A\subseteq\{2,\dots,N\}\) be pairwise coprime:
\[
(a,b)=1 \ \text{for all distinct } a,b\in A,
\qquad
\mu(A):=\sum_{a\in A}\frac1a \le C.
\]
Which \(A\) minimizes the number of integers \(m\le N\) not divisible by any \(a\in A\)?
Erdos suggested the minimizer should be the prime tail \(A=\{q_1,\dots,q_k\}\), where
\(q_1>q_2>\cdots\) are consecutive primes \(\le N\), and \(k\) is maximal with
\[
\sum_{i=1}^k \frac1{q_i}\le C.
\]

## Literature
- The problem page remains open.
- For \(C\le \log 2\), the prime-tail extremizer case is recorded as solved.
- Tao's February 2026 note establishes the weak asymptotic form
  \[
  \Sigma(N,C)\ge \rho(e^C)+o(1)
  \]
  for general pairwise-coprime \(A\), via reduction to the prime case.
- Chojecki's follow-up establishes asymptotic prime-tail rigidity and related stability structure, while noting that exact finite-\(N\) optimization is harder.
- The discussion thread also indicates Saias gives an essentially exact finite-\(N\) evaluation of the prime-tail upper bound, while boundary-precision matching lower bounds remain open.

## Our Approaches / What Is Proven
We do not currently have a verified proof of the full finite-\(N\) statement ("for all sufficiently large \(N\)").

Useful sharpening added here:

### Composite-tail negligibility proposition
Let \(A\subseteq\{2,\dots,N\}\) be pairwise coprime with \(\mu(A)\le C\). For any \(T\ge 2\),
\[
\sum_{\substack{a\in A\setminus\mathbb P\\a\ge T}}\frac1a \ll T^{-1/2}.
\]
Hence, if \(A_{\mathrm{comp}}^{(\ge T)}:=\{a\in A\setminus\mathbb P:a\ge T\}\), then
\[
\left|\sigma_N(A)-\sigma_N\!\left(A\setminus A_{\mathrm{comp}}^{(\ge T)}\right)\right|
\ll T^{-1/2}.
\]

Sketch:
- Tao observes that the number of composite elements in \(A\cap[1,x]\) is \(O(\sqrt x)\).
- On each dyadic block \([X,2X)\), this gives
  \[
  \sum_{\substack{a\in A\setminus\mathbb P\\X\le a<2X}}\frac1a
  \ll X^{-1/2}.
  \]
- Summing dyadically for \(X\ge T\) yields \( \ll T^{-1/2}\).
- Then use the Lipschitz bound
  \[
  |\sigma_N(A)-\sigma_N(A')|\le \mu(A\triangle A').
  \]

If \(w=e^C\), \(y=N^{1/w}\), and \(T=y/(\log N)^M\), then
\[
T^{-1/2}
= N^{-1/(2w)}(\log N)^{M/2}
= o\!\left((\log N)^{-B}\right)
\]
for every fixed \(B>0\). So composites near the prime-tail boundary are too sparse to affect the \(1/\log^2 N\) scale.

## Refined Bottleneck
The remaining obstacle is now sharper:
- Need a prime-case lower bound at true boundary precision (matching upper expansion up to \(o(1/\log^2 N)\)).
- Need a general-to-prime reduction with error \(o(1/\log^2 N)\), whereas current Bonferroni/floor losses are at \(O(1/\log N)\).

Once such boundary-scale estimates are available, the stability collapse should be straightforward from strict log-concavity of \(\rho\).

## Status
- Weak asymptotic form is proved in current literature.
- Exact finite-\(N\) extremal theorem is still open.
