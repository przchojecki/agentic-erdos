# EP-601

## Problem Statement
For limit ordinal \(\alpha\), if a graph has vertex set \(\alpha\), must it contain either an infinite path or an independent set of order type \(\alpha\)?

## Literature / Context
- Known positive ranges in prior notes:
  - ZFC below \(\omega_1^{\omega+2}\),
  - under MA, up to \(2^{\aleph_0}\).
- Boundary and full ZFC classification remain open.

## Our Approaches / What Is Proven
- Implemented finite proxy experiments:
  - random graphs on \(n=40,60\),
  - estimate lower bounds for longest path ratio and independence ratio,
  - track \(\min\) sampled value of \(\max(\text{path ratio},\text{independence ratio})\).
- This is heuristic finite evidence, not an ordinal proof.

## Computation-Guided Observations
- Across tested densities, sampled \(\max(\text{path ratio},\text{independence ratio})\) remains strongly bounded away from \(0\), often close to \(1\) due to long paths.
- At sparser setting (\(n=60,p=0.1\)), sampled minimum of this max-ratio is about \(0.717\), still linear-scale.

## Next Proof Targets
- Replace random finite proxies by structured transfinite constructions reflecting ordinal order types.
- Identify which finite combinatorial lemmas plausibly lift toward the unresolved ZFC boundary.
