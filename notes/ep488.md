## Deeper Proof Program (A => B => C)

### Target
Prove universal inequality
\[
\frac{|B\cap[1,m]|}{m} < 2\,\frac{|B\cap[1,n]|}{n}
\]
for \(m>n\ge\max(A)\), and identify sharpness mechanism.

### A. Concrete Blocking Lemma (Most Critical)

Show a monotone-density-doubling lemma for sets of multiples:
for \(B=\bigcup_{a\in A} a\mathbb N\), density profile \(D(x)=|B\cap[1,x]|/x\) satisfies
\[
\sup_{m>n}\frac{D(m)}{D(n)}\le 2-\eta(n,A),
\]
with \(\eta\ge0\) and \(\eta\to0\) only for near-singleton extremizers.

### B. Proven Lemmas (this note)

#### Lemma B1 (inclusion-exclusion form)
\[
|B\cap[1,x]|=\sum_{\emptyset\neq S\subseteq A}(-1)^{|S|+1}\left\lfloor\frac{x}{\operatorname{lcm}(S)}\right\rfloor.
\]

**Proof.**
Standard inclusion-exclusion over divisibility by elements of \(A\). \(\square\)

#### Lemma B2 (density approximation)
\[
D(x)=\sum_{\emptyset\neq S\subseteq A}\frac{(-1)^{|S|+1}}{\operatorname{lcm}(S)}+O_A(1/x).
\]

**Proof.**
Replace floors in B1 by real parts plus \(O(1)\), then divide by \(x\). \(\square\)

#### Lemma B3 (Singleton Case Is True)
For \(A=\{a\}\), \(a\ge2\), and any \(m>n\ge a\),
\[
\frac{|B\cap[1,m]|}{m}<2\frac{|B\cap[1,n]|}{n},
\quad
B=a\mathbb N.
\]

**Proof.**
Here
\[
\frac{|B\cap[1,x]|}{x}=\frac{\lfloor x/a\rfloor}{x}.
\]
Set \(u=\lfloor n/a\rfloor\ge1\). Then:
1. \(\lfloor m/a\rfloor/m\le 1/a\).
2. \(n<a(u+1)\), hence
\[
2\frac{\lfloor n/a\rfloor}{n}

=2\frac{u}{n}
>
2\frac{u}{a(u+1)}.
\]
If \(u\ge2\), then \(2u/(u+1)>1\), so RHS \(>1/a\), proving strict inequality.
If \(u=1\), then \(n\in[a,2a-1]\), so
\[
2\frac{\lfloor n/a\rfloor}{n}=2/n\ge 2/(2a-1)>1/a\ge \lfloor m/a\rfloor/m.
\]
Thus inequality holds in all cases. \(\square\)

#### Lemma B4 (Sharpness Constant 2 From Singleton Family)
For \(A=\{a\}\), choosing \(n=2a-1\), \(m=2a\),
\[
\frac{ \frac{|B\cap[1,m]|}{m} }{ \frac{|B\cap[1,n]|}{n} }
=\frac{(1/a)}{(1/(2a-1))}
=2-\frac1a\to2.
\]

**Proof.**
\(|B\cap[1,2a]|=2\), \(|B\cap[1,2a-1]|=1\). Substitute directly. \(\square\)

#### Lemma B5 (Eventual-In-\(n\) Inequality For Fixed \(A\))
Let \(A\) be fixed, \(k=|A|\), \(C_A:=2^k-1\), \(a_{\max}:=\max(A)\), and
\[
\delta_A=\sum_{\emptyset\neq S\subseteq A}\frac{(-1)^{|S|+1}}{\mathrm{lcm}(S)}.
\]
Then for all
\[
n\ge \max\{a_{\max},\,3C_A/\delta_A\},
\]
and all \(m>n\),
\[
\frac{|B\cap[1,m]|}{m}<2\frac{|B\cap[1,n]|}{n}.
\]

**Proof.**
From inclusion-exclusion,
\[
M_A(x):=|B\cap[1,x]|=\delta_A x + E_A(x),
\]
where each floor contributes an error in \((-1,0]\), hence
\[
|E_A(x)|\le C_A.
\]
Therefore
\[
\frac{M_A(m)}{m}\le \delta_A+\frac{C_A}{m}\le \delta_A+\frac{C_A}{n},
\]
and
\[
\frac{M_A(n)}{n}\ge \delta_A-\frac{C_A}{n}.
\]
So it suffices to show
\[
\delta_A+\frac{C_A}{n}<2\left(\delta_A-\frac{C_A}{n}\right),
\]
equivalent to \(\delta_A>3C_A/n\), i.e. \(n>3C_A/\delta_A\). The stated threshold gives strict inequality. \(\square\)

#### Corollary B6 (Finite-Range Reduction)
For each fixed finite \(A\), EP-488 reduces to checking finitely many \(n\), namely
\[
\max(A)\le n<3C_A/\delta_A,
\]
since all larger \(n\) are covered by Lemma B5.

**Proof.**
Immediate from Lemma B5. \(\square\)

#### Lemma B7 (Exact sparse-edge case \(B(n)=|A|\))
Let \(A\subset \{2,\dots,n\}\) be finite and
\[
B=\bigcup_{a\in A} a\mathbb N,\qquad |B\cap[1,n]|=|A|=:k.
\]
Then for every \(m>n\),
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}.
\]

**Proof.**
The condition \(|B\cap[1,n]|=|A|\) means that, up to \(n\), each \(a\in A\) contributes only its first multiple.
Hence \(2a>n\) for all \(a\in A\), i.e. \(a>n/2\).
Now for \(m>n\),
\[
|B\cap[1,m]|
\le \sum_{a\in A}\left\lfloor\frac{m}{a}\right\rfloor
< \sum_{a\in A}\frac{m}{a}
< \sum_{a\in A}\frac{2m}{n}
=\frac{2km}{n}.
\]
Divide by \(m\), and use \(|B\cap[1,n]|=k\):
\[
\frac{|B\cap[1,m]|}{m}<\frac{2k}{n}
=2\,\frac{|B\cap[1,n]|}{n}.
\]
\(\square\)

#### Lemma B8 (Near-extremal structure at small denominator)
If \(n\ge\max(A)\) and \(|B\cap[1,n]|=1\), then \(A=\{a\}\) and \(n\in[a,2a-1]\), so EP-488 follows from Lemma B3.

**Proof.**
Since \(n\ge\max(A)\), every \(a\in A\) lies in \([1,n]\cap B\). Thus \(|A|\le |B\cap[1,n]|=1\), so \(A=\{a\}\).
Also \(|B\cap[1,n]|=\lfloor n/a\rfloor=1\), equivalent to \(a\le n\le 2a-1\).
Apply Lemma B3. \(\square\)

#### Lemma B9 (Case \(|B\cap[1,n]|=2\) is fully resolved)
If \(n\ge\max(A)\), \(|B\cap[1,n]|=2\), and \(m>n\), then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}.
\]

**Proof.**
Write \(t:=|B\cap[1,n]|=2\).
Since each \(a\in A\) belongs to \(B\cap[1,n]\), we have \(|A|\le2\).

If \(|A|=1\), this is the singleton case (Lemma B3), done.

If \(|A|=2\), say \(A=\{a,b\}\), then \(B\cap[1,n]\) has only the two base elements \(a,b\), so there is no second multiple of either within \([1,n]\). Thus \(n<2a\) and \(n<2b\), equivalently \(a,b>n/2\). Therefore
\[
|B\cap[1,m]|
\le \left\lfloor\frac{m}{a}\right\rfloor+\left\lfloor\frac{m}{b}\right\rfloor
< \frac{m}{a}+\frac{m}{b}
< \frac{2m}{n}+\frac{2m}{n}
=\frac{4m}{n}.
\]
Divide by \(m\):
\[
\frac{|B\cap[1,m]|}{m}<\frac{4}{n}
=2\cdot\frac{2}{n}
=2\,\frac{|B\cap[1,n]|}{n}.
\]
\(\square\)

#### Lemma B10 (Case \(|B\cap[1,n]|=3\) is fully resolved)
If \(n\ge\max(A)\), \(|B\cap[1,n]|=3\), and \(m>n\), then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}
=\frac{6}{n}.
\]

**Proof.**
Set \(t:=|B\cap[1,n]|=3\). Since each \(a\in A\) lies in \(B\cap[1,n]\), \(|A|\le3\).

If \(|A|=1\), apply Lemma B3.

If \(|A|=3\), then \(|B\cap[1,n]|=|A|\), so apply Lemma B7.

It remains \(|A|=2\), say \(A=\{a,b\}\) with \(a<b\). Then
\[
|B\cap[1,m]|
\le \left\lfloor\frac{m}{a}\right\rfloor+\left\lfloor\frac{m}{b}\right\rfloor
< \frac{m}{a}+\frac{m}{b},
\]
so it is enough to show
\[
\frac1a+\frac1b<\frac6n.
\]
Write \(u:=\lfloor n/a\rfloor\in\{1,2,3\}\), since \(u\ge1\) (as \(a\le n\)) and \(u\le3\) (otherwise \(a,2a,3a,4a\le n\) would already force at least 4 elements of \(B\cap[1,n]\)).

Case \(u=1\): then \(a>n/2\), hence \(1/a<2/n\). Also \(b>a>n/2\), so \(1/b<2/n\). Therefore \(1/a+1/b<4/n<6/n\).

Case \(u=2\): then \(a>n/3\), so \(1/a<3/n\). Also \(b>a>n/3\), so \(1/b<3/n\). Hence \(1/a+1/b<6/n\).

Case \(u=3\): multiples of \(a\) up to \(n\) are \(\{a,2a,3a\}\). Since \(|B\cap[1,n]|=3\), every multiple of \(b\) up to \(n\) must lie in \(\{a,2a,3a\}\). In particular \(b\in\{2a,3a\}\), so \(b\ge2a\). Now \(a>n/4\) (from \(u=3\)), thus \(1/a<4/n\), and \(b\ge2a>n/2\), so \(1/b<2/n\). Therefore \(1/a+1/b<6/n\).

In all cases, \(|B\cap[1,m]|/m<6/n=2|B\cap[1,n]|/n\). \(\square\)

#### Lemma B11 (Divisibility reduction)
Let \(A\) be finite and
\[
B=\bigcup_{a\in A} a\mathbb N.
\]
If \(a_1,a_2\in A\) with \(a_1\mid a_2\), then removing \(a_2\) does not change \(B\).
Hence there is a reduced set \(A_{\min}\subseteq A\) (no two elements divide each other) with
\[
\bigcup_{a\in A_{\min}} a\mathbb N = B.
\]

**Proof.**
If \(a_1\mid a_2\), then \(a_2\mathbb N\subseteq a_1\mathbb N\), so \(a_2\) is redundant in the union. Repeating finitely many times yields \(A_{\min}\). \(\square\)

#### Lemma B12 (Case \(|B\cap[1,n]|=4\) is fully resolved)
If \(n\ge\max(A)\), \(|B\cap[1,n]|=4\), and \(m>n\), then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}
=\frac{8}{n}.
\]

**Proof.**
Replace \(A\) by the reduced generator set \(G:=A_{\min}\) from Lemma B11; this does not change \(B\). Let \(r:=|G|\). Since every \(g\in G\) belongs to \(B\cap[1,n]\), we have \(1\le r\le4\).

If \(r=1\), apply Lemma B3.

If \(r=4\), then \(|B\cap[1,n]|=|G|\), so apply Lemma B7.

It remains \(r=2\) or \(r=3\). Write \(G=\{g_1<\cdots<g_r\}\), and \(u_i:=\lfloor n/g_i\rfloor\).

For any \(i\), \(u_i\le4\), otherwise \(g_i,2g_i,3g_i,4g_i,5g_i\le n\) would already give at least 5 points in \(B\cap[1,n]\), contradiction.

**Case \(r=2\).**
Suppose \(u_1=4\). Then \(g_1,2g_1,3g_1,4g_1\in B\cap[1,n]\). Since \(G\) is reduced, \(g_2\) is not a multiple of \(g_1\), so \(g_2\notin\{g_1,2g_1,3g_1,4g_1\}\), forcing at least 5 points in \(B\cap[1,n]\), contradiction. Hence \(u_1\le3\), so \(g_1>n/4\). Also \(g_2>g_1>n/4\). Therefore
\[
\frac{|B\cap[1,m]|}{m}
\le \frac{\lfloor m/g_1\rfloor+\lfloor m/g_2\rfloor}{m}
< \frac1{g_1}+\frac1{g_2}
< \frac4n+\frac4n
=\frac8n.
\]

**Case \(r=3\).**
Claim: at most one \(u_i\) equals 2. Indeed, if \(u_i=u_j=2\) with \(i\ne j\), then \(2g_i,2g_j\le n\) are two distinct extra points beyond the three base points \(g_1,g_2,g_3\), so \(|B\cap[1,n]|\ge5\), contradiction.
Also no \(u_i\ge3\): if \(u_i\ge3\), then \(2g_i,3g_i\le n\) give two distinct extra points beyond the three bases, again contradiction.
Thus one has either:
- all \(u_i=1\), hence each \(g_i>n/2\); or
- exactly one \(u_i=2\) and the other two are 1, hence one generator \(>n/3\) and two \(>n/2\).
In both subcases,
\[
\sum_{i=1}^3 \frac1{g_i}
< \frac3n+\frac2n+\frac2n
=\frac7n
<\frac8n.
\]
Therefore
\[
\frac{|B\cap[1,m]|}{m}
\le \sum_{i=1}^3 \frac{\lfloor m/g_i\rfloor}{m}
< \sum_{i=1}^3 \frac1{g_i}
<\frac8n.
\]

All cases give \(|B\cap[1,m]|/m<8/n=2|B\cap[1,n]|/n\). \(\square\)

#### Lemma B13 (All reduced two-generator sets are fully resolved)
Let \(G=\{a,b\}\) with \(2\le a<b\), \(a\nmid b\), and
\[
B=a\mathbb N\cup b\mathbb N.
\]
Then for every \(n\ge b\) and every \(m>n\),
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}.
\]

**Proof.**
Set
\[
u:=\Big\lfloor\frac{n}{a}\Big\rfloor,\quad
v:=\Big\lfloor\frac{n}{b}\Big\rfloor,\quad
w:=\Big\lfloor\frac{n}{\mathrm{lcm}(a,b)}\Big\rfloor.
\]
Then
\[
|B\cap[1,n]|=u+v-w.
\]
Also
\[
\frac{|B\cap[1,m]|}{m}
\le \frac{\lfloor m/a\rfloor+\lfloor m/b\rfloor}{m}
<\frac1a+\frac1b.
\]
Now \(u\le n/a<u+1\), \(v\le n/b<v+1\), hence
\[
\frac1a+\frac1b<\frac{u+1}{n}+\frac{v+1}{n}=\frac{u+v+2}{n}.
\]
So it is enough to show
\[
u+v+2\le 2(u+v-w),
\]
equivalently
\[
2\le u+v-2w. \tag{*}
\]

Because \(a\nmid b\), with \(g=\gcd(a,b)\) we have \(a/g\ge2\), so
\[
\mathrm{lcm}(a,b)=\frac{ab}{g}=b\frac{a}{g}\ge 2b.
\]
Therefore
\[
w\le \Big\lfloor\frac{n}{2b}\Big\rfloor\le \Big\lfloor\frac{v}{2}\Big\rfloor.
\]
Hence
\[
u+v-2w\ge u+v-2\Big\lfloor\frac v2\Big\rfloor \in\{u,u+1\}.
\]
If \(u\ge2\), this is at least 2. If \(u=1\), then \(n<2a<2b\), so \(v=1\), and then
\[
u+v-2w\ge 1+1-2\lfloor 1/2\rfloor=2.
\]
Thus \((*)\) always holds. Consequently
\[
\frac{|B\cap[1,m]|}{m}<\frac{u+v+2}{n}\le \frac{2(u+v-w)}{n}
=2\,\frac{|B\cap[1,n]|}{n}.
\]
\(\square\)

#### Lemma B14 (Case \(|B\cap[1,n]|=5\) is fully resolved)
If \(n\ge\max(A)\), \(|B\cap[1,n]|=5\), and \(m>n\), then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}
=\frac{10}{n}.
\]

**Proof.**
Replace \(A\) by reduced generators \(G=A_{\min}\) (Lemma B11), and let \(r:=|G|\). Since each \(g\in G\) lies in \(B\cap[1,n]\), \(1\le r\le5\).

If \(r=1\), use Lemma B3.  
If \(r=2\), use Lemma B13.  
If \(r=5\), then \(|B\cap[1,n]|=|G|\), so use Lemma B7.

It remains \(r=3\) or \(r=4\). Write \(G=\{g_1<\cdots<g_r\}\), and \(u_i:=\lfloor n/g_i\rfloor\).
As before,
\[
\frac{|B\cap[1,m]|}{m}
\le \sum_{i=1}^r \frac{\lfloor m/g_i\rfloor}{m}
< \sum_{i=1}^r \frac1{g_i}
< \sum_{i=1}^r \frac{u_i+1}{n}
=\frac{(\sum_i u_i)+r}{n}.
\]
So it is enough to prove
\[
\sum_i u_i \le 10-r. \tag{*}
\]

Subcase \(r=3\): need \(\sum u_i\le7\).

- \(u_1\le3\): if \(u_1\ge4\), then \(g_1,2g_1,3g_1,4g_1\in B\cap[1,n]\), and \(g_2,g_3\) are distinct non-multiples of \(g_1\), hence outside this 4-set; so \(|B\cap[1,n]|\ge6\), contradiction.
- \(u_2\le2\): if \(u_2\ge3\), then the sets \(\{g_1,2g_1,3g_1\}\) and \(\{g_2,2g_2,3g_2\}\) have intersection size at most 1, so their union has at least 5 points. Also \(g_3\) is not divisible by \(g_1\) or \(g_2\), hence is outside that union, forcing at least 6 points, contradiction.

Thus \(u_1\le3\), \(u_2\le2\), \(u_3\le u_2\le2\), so \(\sum u_i\le3+2+2=7\), proving \((*)\). Hence
\[
\frac{|B\cap[1,m]|}{m}<\frac{7+3}{n}=\frac{10}{n}.
\]

Subcase \(r=4\): need \(\sum u_i\le6\).

- \(u_1\le2\): if \(u_1\ge3\), then \(g_1,2g_1,3g_1\) plus \(g_2,g_3,g_4\) (all outside \(g_1\mathbb N\) by reducedness) give at least 6 points, contradiction.
- At most one \(u_i\) equals 2: if \(u_i=u_j=2\) with \(i\neq j\), then \(g_i,2g_i,g_j,2g_j\) are at least 4 distinct points (equalities like \(2g_i=g_j\) are forbidden by reducedness), and adding the two other generators gives at least 6 points, contradiction.

So \(\sum u_i\le2+1+1+1=5\), stronger than \((*)\), and
\[
\frac{|B\cap[1,m]|}{m}<\frac{5+4}{n}=\frac{9}{n}<\frac{10}{n}.
\]

All cases yield the target inequality. \(\square\)

#### Lemma B15 (Case \(|B\cap[1,n]|=6\): resolved for reduced size \(r=4,5\))
Let \(n\ge\max(A)\), \(m>n\), \(|B\cap[1,n]|=6\), and let \(G=A_{\min}\) be the reduced generator set (Lemma B11), with \(r:=|G|\).
If \(r\in\{4,5\}\), then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}=\frac{12}{n}.
\]

**Proof.**
Write \(G=\{g_1<\cdots<g_r\}\), \(u_i:=\lfloor n/g_i\rfloor\). As in earlier lemmas,
\[
\frac{|B\cap[1,m]|}{m}
\le \sum_{i=1}^r \frac{\lfloor m/g_i\rfloor}{m}
< \sum_{i=1}^r \frac1{g_i}
< \sum_{i=1}^r \frac{u_i+1}{n}
=\frac{(\sum_i u_i)+r}{n}.
\]
So it suffices to bound \(\sum_i u_i\).

Let \(E:=|B\cap[1,n]|-r=6-r\), the number of non-base points in \(B\cap[1,n]\).

For each \(i\) with \(u_i\ge2\), the point \(2g_i\le n\) is a non-base point. These \(2g_i\) are pairwise distinct, so
\[
s:=\#\{i:u_i\ge2\}\le E. \tag{1}
\]
Also, if some \(u_i\ge4\), then \(2g_i,3g_i,4g_i\) are three distinct non-base points, forcing \(E\ge3\), impossible for \(r=4,5\). Hence
\[
u_i\le3\quad\text{for all }i. \tag{2}
\]

Case \(r=5\): then \(E=1\). By (1), \(s\le1\). Since \(|B\cap[1,n]|=6>5=r\), we must have \(s=1\). By (2), the unique index with \(u_i\ge2\) has \(u_i\in\{2,3\}\); but \(u_i=3\) would already give two non-base points \(2g_i,3g_i\), contradicting \(E=1\). So that index has \(u_i=2\), and all others have \(u_i=1\). Therefore
\[
\sum_i u_i=2+1+1+1+1=6.
\]
Hence
\[
\frac{|B\cap[1,m]|}{m}<\frac{6+5}{n}=\frac{11}{n}<\frac{12}{n}.
\]

Case \(r=4\): then \(E=2\). By (1), \(s\le2\), and since \(6>4\), we have \(s\ge1\).

- If \(s=1\), then one \(u_i\in\{2,3\}\), others are 1, so \(\sum_i u_i\le3+1+1+1=6\).
- If \(s=2\), then two indices have \(u\in\{2,3\}\), others are 1. They cannot both be 3, because then the two progressions contribute non-base points \(\{2g_i,3g_i\}\) and \(\{2g_j,3g_j\}\), which contain at least three distinct numbers (since \(2g_i\neq2g_j\), and both \(3g_i,3g_j\) cannot simultaneously coincide with \(\{2g_i,2g_j\}\)); this would force \(E\ge3\), contradiction. So the pair is at most \((3,2)\), and
\[
\sum_i u_i\le3+2+1+1=7.
\]

Thus in both subcases, \(\sum_i u_i\le7\), and
\[
\frac{|B\cap[1,m]|}{m}<\frac{7+4}{n}=\frac{11}{n}<\frac{12}{n}.
\]

Therefore the claim holds for \(r=4,5\). \(\square\)

#### Lemma B21 (Case \(|B\cap[1,n]|=6\): resolved for reduced size \(r=3\))
Let \(n\ge\max(A)\), \(m>n\), \(|B\cap[1,n]|=6\), and let \(G=A_{\min}=\{a<b<c\}\) (Lemma B11).  
Then
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}=\frac{12}{n}.
\]

**Proof.**
Set
\[
u_1:=\Big\lfloor\frac{n}{a}\Big\rfloor,\quad
u_2:=\Big\lfloor\frac{n}{b}\Big\rfloor,\quad
u_3:=\Big\lfloor\frac{n}{c}\Big\rfloor.
\]
As usual,
\[
\frac{|B\cap[1,m]|}{m}
\le \sum_{g\in G}\frac{\lfloor m/g\rfloor}{m}
< \sum_{g\in G}\frac1g
< \frac{u_1+u_2+u_3+3}{n}. \tag{1}
\]
So it suffices to prove \(u_1+u_2+u_3\le8\).

First, \(u_1\le4\): if \(u_1\ge5\), then \(a,2a,3a,4a,5a\le n\), giving at least 5 points from \(a\mathbb N\) in \([1,n]\), plus \(b\) and \(c\) (not multiples of \(a\), by reducedness), hence at least 7 points in \(B\cap[1,n]\), contradiction.

Now claim \(u_2\le2\). Suppose \(u_2\ge3\). Then \(u_1\ge u_2\ge3\), so
\[
2a,3a\le n,\qquad 2b,3b\le n.
\]
Since \(|B\cap[1,n]|=6\) and we already have 3 base points \(a,b,c\), there are exactly 3 non-base points total. Therefore the four numbers \(2a,3a,2b,3b\) cannot be all distinct; at least one equality must hold between one of \(\{2a,3a\}\) and one of \(\{2b,3b\}\).
The only possible equality is
\[
3a=2b,
\]
because \(2a=2b\), \(3a=3b\) force \(a=b\), and \(2a=3b\) gives \(b< a\), impossible.
Hence \(b=\frac{3a}{2}\), so \(u_2\ge3\) implies
\[
n\ge 3b=\frac{9a}{2},
\]
thus \(u_1=\lfloor n/a\rfloor\ge4\), so \(4a\le n\).
Now
\[
2a,\ 3a(=2b),\ 4a,\ 3b\le n
\]
are four distinct non-base points (indeed \(3b=\frac{9a}{2}\neq 2a,3a,4a\)). This contradicts that only 3 non-base points exist. Therefore \(u_2\le2\).

Since \(u_3\le u_2\), we get
\[
u_1+u_2+u_3\le 4+2+2=8.
\]
Applying (1):
\[
\frac{|B\cap[1,m]|}{m}<\frac{8+3}{n}=\frac{11}{n}<\frac{12}{n}.
\]
So the target inequality holds. \(\square\)

### Failed shortcut (now explicit)

The tempting bound
\[
\frac{|B\cap[1,m]|}{m}\le \frac{|B\cap[1,n]|+1}{n}\qquad(m>n)
\]
is false in general.
Concrete counterexample (found in standalone check):
\[
A=\{3,4,22,38,46,54,65\},\quad n=74,\quad m=81,
\]
for which
\[
\frac{|B\cap[1,m]|}{m}=\frac{45}{81}\approx 0.55556
>
\frac{40+1}{74}=\frac{41}{74}\approx 0.55405.
\]
So this route cannot close the problem.

### C. Main unresolved point

The remaining gap is now sharply localized:

- By Lemmas B3/B8/B9/B10/B12/B14, all cases with \(|B\cap[1,n]|\le5\) are done.
- By Lemma B13 (plus divisibility reduction, Lemma B11), all problems with reduced generator size \(\le2\) are done for all \(n\).
- By Lemmas B15 and B21, the full \(|B\cap[1,n]|=6\) layer is done (all reduced sizes \(r\le6\)).
- By Lemma B5, all sufficiently large \(n\) (for fixed \(A\)) are done.

So the unsolved core is the finite-window regime with
\[
7\le |B\cap[1,n]|<\infty
\]
and reduced generator size \(\ge3\), below the explicit B5 threshold.

### D. General Techniques (to finish full EP-488)

The layer-by-layer closure is useful but likely not sufficient by itself for a full theorem. The global routes are:

1. Sieve-weight majorant with sharp finite-\(n\) error:
\[
M_A(x)=\delta_A x+E_A(x),\qquad E_A(x)=\sum_{\emptyset\neq S\subseteq A}(-1)^{|S|+1}\Bigl(\Big\lfloor\frac{x}{\mathrm{lcm}(S)}\Big\rfloor-\frac{x}{\mathrm{lcm}(S)}\Bigr).
\]
Need a sign-sensitive bound on \(E_A(m)-2E_A(n)\), not just \(|E_A|\le C_A\).

2. Extremal reduction to minimal antichains:
after Lemma B11, reduce to \(A_{\min}\) and prove a universal extremal principle:
near-equality in the factor \(2\) inequality forces singleton-like structure.
This is a stability theorem, analogous to Tao-style near-extremizer rigidity.

3. LP/duality formulation:
for fixed \(n\), maximize
\[
\sup_{m>n}\frac{|B\cap[1,m]|/m}{|B\cap[1,n]|/n}
\]
over reduced multiplicative set-systems.
If dual certificate gives value \(<2\), full theorem follows.

4. Finite-window compactness:
combine B5 with a structural bound showing that for fixed \(t:=|B\cap[1,n]|\), only finitely many reduced patterns of first-hit multiplicities \((u_i)\) are extremal; then solve each pattern class symbolically.

### E. Concrete Missing Lemmas (Global)

- **GL1 (signed discrepancy transport):**
  prove \(E_A(m)\le 2E_A(n)+\Theta_A(n,m)\) with \(\Theta_A\) summably small/uniformly negative in the critical window.
- **GL2 (near-extremizer rigidity):**
  if
  \[
  \frac{|B\cap[1,m]|/m}{|B\cap[1,n]|/n}>2-\varepsilon,
  \]
  then \(A_{\min}\) has a dominant generator \(a_*\) and all others contribute \(O_\varepsilon(1/n)\) to density.
- **GL3 (pattern finiteness at fixed \(t\)):**
  for fixed \(t\), reduced \(A\) with \(|B\cap[1,n]|=t\) has bounded admissible profile complexity (independent of arithmetic size of elements), enabling full symbolic elimination.

### F. GL1 Progress: Signed Discrepancy Transport (executed)

We now make GL1 explicit in a weighted-lcm basis.

For fixed finite \(A\), let \(\mathcal L(A)\) be the set of distinct lcm-values
\[
\mathcal L(A):=\{\operatorname{lcm}(S):\emptyset\neq S\subseteq A\},
\]
and define coefficients
\[
\lambda_d:=\sum_{\substack{\emptyset\neq S\subseteq A\\ \operatorname{lcm}(S)=d}}(-1)^{|S|+1},
\qquad d\in\mathcal L(A).
\]
Then
\[
M_A(x)=\sum_{d\in\mathcal L(A)} \lambda_d \Big\lfloor\frac{x}{d}\Big\rfloor,
\qquad
\delta_A=\sum_{d\in\mathcal L(A)}\frac{\lambda_d}{d}.
\]

Define
\[
W_+(A):=\sum_{\lambda_d>0}\lambda_d,\qquad
W_-(A):=\sum_{\lambda_d<0}(-\lambda_d).
\]

#### Lemma B16 (Exact transport identity)
For \(m>n\ge\max(A)\),
\[
\frac{M_A(m)}{m}-2\frac{M_A(n)}{n}
=
-\delta_A
+\sum_{d\in\mathcal L(A)}\lambda_d\!\left(\frac{2}{n}\Big\{\frac{n}{d}\Big\}
-\frac{1}{m}\Big\{\frac{m}{d}\Big\}\right).
\]

**Proof.**
Use \(\lfloor x/d\rfloor=x/d-\{x/d\}\) in \(M_A(x)=\sum_d \lambda_d\lfloor x/d\rfloor\), divide by \(x\), and subtract \(2M_A(n)/n\) from \(M_A(m)/m\). \(\square\)

#### Lemma B17 (Signed GL1 upper bound)
For \(m>n\ge\max(A)\),
\[
\frac{M_A(m)}{m}-2\frac{M_A(n)}{n}
\le
-\delta_A+\frac{2W_+(A)}{n}+\frac{W_-(A)}{m}
\le
-\delta_A+\frac{2W_+(A)+W_-(A)}{n}.
\]

**Proof.**
From Lemma B16, bound each term with \(0\le\{x/d\}<1\):

- if \(\lambda_d>0\), then
  \[
  \lambda_d\!\left(\frac{2}{n}\{n/d\}-\frac1m\{m/d\}\right)\le \frac{2\lambda_d}{n};
  \]
- if \(\lambda_d<0\), write \(\lambda_d=-\mu_d\), \(\mu_d>0\), then
  \[
  -\mu_d\!\left(\frac{2}{n}\{n/d\}-\frac1m\{m/d\}\right)\le \frac{\mu_d}{m}.
  \]

Sum over \(d\in\mathcal L(A)\). \(\square\)

#### Corollary B18 (GL1 finite-\(n\) criterion)
If
\[
n>\frac{2W_+(A)+W_-(A)}{\delta_A},
\]
then for all \(m>n\),
\[
\frac{|B\cap[1,m]|}{m}<2\,\frac{|B\cap[1,n]|}{n}.
\]

**Proof.**
By Lemma B17, RHS is \(<0\) under the stated inequality, hence the target inequality follows. \(\square\)

**Interpretation.**
This is a genuine signed transport inequality: positive and negative lcm-weights contribute asymmetrically (\(2/n\) vs \(1/m\)). It sharpens the crude \(3C_A/\delta_A\) criterion whenever cancellation in lcm-layers makes \(2W_++W_-\ll 3(2^{|A|}-1)\).

### G. GL2 Progress: Near-Extremizer Rigidity (partial, rigorous)

Let
\[
R_A(n,m):=\frac{M_A(m)/m}{M_A(n)/n},\qquad m>n\ge \max(A).
\]

#### Lemma B19 (Near-factor-2 forces a finite window)
Fix finite \(A\), and \(0<\varepsilon<\delta_A\).  
If
\[
R_A(n,m)>2-\varepsilon,
\]
then necessarily
\[
n<\frac{2W_+(A)+W_-(A)}{\delta_A-\varepsilon}.
\]

**Proof.**
From \(R_A(n,m)>2-\varepsilon\),
\[
\frac{M_A(m)}{m}-2\frac{M_A(n)}{n}
>
-\varepsilon\frac{M_A(n)}{n}
\ge -\varepsilon
\]
since \(M_A(n)/n\le1\).  
By Lemma B17,
\[
\frac{M_A(m)}{m}-2\frac{M_A(n)}{n}
\le
-\delta_A+\frac{2W_+(A)+W_-(A)}{n}.
\]
Combining,
\[
-\varepsilon<-\delta_A+\frac{2W_+(A)+W_-(A)}{n},
\]
so
\[
n<\frac{2W_+(A)+W_-(A)}{\delta_A-\varepsilon}.
\]
\(\square\)

#### Corollary B20 (Uniform exclusion of near-extremizers at large \(n\))
For fixed \(A\) and \(\varepsilon\in(0,\delta_A)\), all pairs \(m>n\) with
\[
n\ge \frac{2W_+(A)+W_-(A)}{\delta_A-\varepsilon}
\]
satisfy
\[
R_A(n,m)\le 2-\varepsilon.
\]

**Proof.**
Contrapositive of Lemma B19. \(\square\)

**What this gives for GL2.**
- We now have a rigorous near-extremizer rigidity in the \(n\)-direction: ratio \(>2-\varepsilon\) can occur only in an explicit finite initial window.
- The remaining GL2 gap is structural inside that finite window: prove that any such near-extremizer must be singleton-dominant (or classify all exceptional reduced patterns).

### H. Global Reduction (non-finite, theorem-level)

Let \(G:=A_{\min}\) (Lemma B11), and define
\[
\mu_G:=\sum_{g\in G}\frac1g.
\]
Since
\[
\frac{M_A(m)}{m}
\le \sum_{g\in G}\frac{\lfloor m/g\rfloor}{m}
< \sum_{g\in G}\frac1g
=\mu_G,
\]
we get the following exact reduction.

#### Lemma B22 (Single-time reduction)
If
\[
\frac{M_A(n)}{n}\ge \frac{\mu_G}{2},
\]
then for every \(m>n\),
\[
\frac{M_A(m)}{m}<2\,\frac{M_A(n)}{n}.
\]

**Proof.**
From \(M_A(m)/m<\mu_G\le 2M_A(n)/n\). \(\square\)

So EP-488 is reduced to proving
\[
\frac{M_A(n)}{n}\ge \frac12\sum_{g\in A_{\min}}\frac1g
\quad(n\ge \max A). \tag{R}
\]

#### Lemma B23 (Bonferroni sufficient condition for (R))
Let
\[
\Omega_G(n):=\sum_{\substack{g,h\in G\\ g<h}}\frac{\lfloor n/\mathrm{lcm}(g,h)\rfloor}{n}.
\]
If
\[
\Omega_G(n)\le \frac{\mu_G}{2}-\frac{|G|}{n},
\]
then (R) holds.

**Proof.**
By two-term inclusion-exclusion,
\[
\frac{M_A(n)}{n}
\ge
\sum_{g\in G}\frac{\lfloor n/g\rfloor}{n}
-\Omega_G(n).
\]
Also \(\lfloor n/g\rfloor/n\ge 1/g-1/n\), so
\[
\frac{M_A(n)}{n}\ge \mu_G-\frac{|G|}{n}-\Omega_G(n)\ge \frac{\mu_G}{2}.
\]
Thus (R) follows. \(\square\)

**Interpretation (new global bottleneck).**
The hard part is now explicit and non-finite:
prove a uniform overlap bound of the shape
\[
\Omega_G(n)\le \frac{\mu_G}{2}-\frac{|G|}{n},
\]
for reduced divisor antichains \(G\). This is a sieve/overlap-control statement, not a finite pattern check.
This condition is a **sufficient criterion**, not currently proved as a universal identity.

#### Lemma B24 (Near-extremizer overlap coercion)
Keep notation
\[
S_1(n):=\sum_{g\in G}\frac{\lfloor n/g\rfloor}{n},\qquad
\Omega_G(n):=\sum_{g<h}\frac{\lfloor n/\mathrm{lcm}(g,h)\rfloor}{n}.
\]
If for some \(m>n\ge \max(A)\),
\[
R_A(n,m)>2-\varepsilon\qquad(0<\varepsilon<1),
\]
then
\[
\Omega_G(n)>S_1(n)-\frac{\mu_G}{2-\varepsilon}.
\]
In particular, since \(S_1(n)\ge \mu_G-|G|/n\),
\[
\Omega_G(n)>
\mu_G\frac{1-\varepsilon}{2-\varepsilon}
-\frac{|G|}{n}.
\]

**Proof.**
From \(R_A(n,m)>2-\varepsilon\),
\[
\frac{M_A(m)}{m}>(2-\varepsilon)\frac{M_A(n)}{n}.
\]
But \(M_A(m)/m<\mu_G\), hence
\[
\frac{M_A(n)}{n}<\frac{\mu_G}{2-\varepsilon}. \tag{1}
\]
Also by two-term inclusion-exclusion,
\[
\frac{M_A(n)}{n}\ge S_1(n)-\Omega_G(n). \tag{2}
\]
Combining (1) and (2),
\[
S_1(n)-\Omega_G(n)<\frac{\mu_G}{2-\varepsilon},
\]
which is the first claim. Using \(S_1(n)\ge \mu_G-|G|/n\) gives the second inequality. \(\square\)

**GL2 consequence.**
Any near-factor-2 configuration must exhibit pair-overlap mass at (roughly) half of \(\mu_G\). This turns GL2 into a structural theorem about high-overlap reduced divisor families.

#### Lemma B25 (Anchor overlap mass)
Let
\[
\widetilde\Omega_G:=\sum_{g<h}\frac{1}{\mathrm{lcm}(g,h)}.
\]
Then there exists \(g_*\in G\) such that
\[
\sum_{h\in G\setminus\{g_*\}}\frac{1}{\mathrm{lcm}(g_*,h)}
\ge
\frac{2\widetilde\Omega_G}{|G|}.
\]

**Proof.**
Define
\[
T(g):=\sum_{h\in G\setminus\{g\}}\frac{1}{\mathrm{lcm}(g,h)}.
\]
Then
\[
\sum_{g\in G}T(g)=2\widetilde\Omega_G.
\]
By averaging, some \(g_*\) satisfies \(T(g_*)\ge 2\widetilde\Omega_G/|G|\). \(\square\)

#### Corollary B26 (Near-extremizer anchor from B24)
Assume \(R_A(n,m)>2-\varepsilon\) \((0<\varepsilon<1)\).  
Set \(r:=|G|\), \(G=A_{\min}\). Then there exists \(g_*\in G\) with
\[
\sum_{h\in G\setminus\{g_*\}}\frac{1}{\mathrm{lcm}(g_*,h)}
>
\frac{2}{r}\left(\mu_G\frac{1-\varepsilon}{2-\varepsilon}-\frac{|G|}{n}\right).
\]

**Proof.**
By B24,
\[
\Omega_G(n)>
\mu_G\frac{1-\varepsilon}{2-\varepsilon}
-\frac{|G|}{n}.
\]
Since \(\lfloor x\rfloor\le x\), we have
\[
\Omega_G(n)\le \widetilde\Omega_G.
\]
Apply Lemma B25. \(\square\)

#### Lemma B27 (Threshold core around anchor)
Let \(g_*\in G\), \(r:=|G|\), and
\[
T_*:=\sum_{h\in G\setminus\{g_*\}}\frac{1}{\mathrm{lcm}(g_*,h)}.
\]
For any \(Y>0\), define
\[
N_Y:=\#\{h\in G\setminus\{g_*\}:\ \mathrm{lcm}(g_*,h)\le Y\}.
\]
Then
\[
N_Y\ge \frac{Y\,T_*-(r-1)}{Y-1}.
\]

**Proof.**
Split the sum \(T_*\) into terms with lcm \(\le Y\) and \(>Y\):
\[
T_*
\le N_Y\cdot 1 + (r-1-N_Y)\frac1Y
=\frac{r-1}{Y}+N_Y\!\left(1-\frac1Y\right).
\]
Rearranging yields
\[
N_Y\ge \frac{Y T_*-(r-1)}{Y-1},
\]
for \(Y>1\). (For \(0<Y\le1\), the claim is trivial.) \(\square\)

**Structural takeaway (GL2 bridge).**
Near-extremizers force an anchor \(g_*\) with many low-lcm pairs \((g_*,h)\).  
The remaining step to full GL2 is to show that, in a reduced divisor antichain, such a large low-lcm anchor core can only occur in singleton-dominant (or otherwise explicitly classifiable) configurations.

#### Lemma B28 (Anchor factorization identity)
Fix \(g_*\in G\). For each \(h\in G\setminus\{g_*\}\), define
\[
d_h:=\gcd(g_*,h),\qquad
u_h:=\frac{h}{d_h}=\frac{\mathrm{lcm}(g_*,h)}{g_*}.
\]
Then \(u_h\in\mathbb N\), \(u_h\ge2\), and
\[
T_*:=\sum_{h\ne g_*}\frac1{\mathrm{lcm}(g_*,h)}
=\frac1{g_*}\sum_{h\ne g_*}\frac1{u_h}.
\]

**Proof.**
By definition, \(\mathrm{lcm}(g_*,h)=g_*u_h\), giving the identity.
If \(u_h=1\), then \(h=d_h\mid g_*\), impossible in reduced \(G\). So \(u_h\ge2\). \(\square\)

#### Corollary B29 (Universal anchor upper bound)
With notation of B28,
\[
T_*\le \frac{r-1}{2g_*}.
\]

**Proof.**
From B28, \(u_h\ge2\), hence each term \(1/u_h\le1/2\). \(\square\)

#### Corollary B30 (Near-extremizer quantitative constraint)
If \(R_A(n,m)>2-\varepsilon\) \((0<\varepsilon<1)\), then for the anchor \(g_*\) from B26,
\[
\mu_G\frac{1-\varepsilon}{2-\varepsilon}
<
\frac{r(r-1)}{4g_*}+\frac{r}{n}.
\]

**Proof.**
From B26,
\[
T_*>
\frac{2}{r}\left(\mu_G\frac{1-\varepsilon}{2-\varepsilon}-\frac{r}{n}\right).
\]
From B29, \(T_*\le (r-1)/(2g_*)\). Combine and rearrange. \(\square\)

**Interpretation.**
Near-extremizers require an anchor \(g_*\) small enough (relative to \(\mu_G,r,n\)) to satisfy B30.  
This is a genuine structural restriction and narrows GL2 to controlling reduced families with such a forced anchor scale.

#### Lemma B31 (Anchor gcd-decomposition)
Fix \(g\in G\), and write
\[
\mu_{-g}:=\sum_{h\in G\setminus\{g\}}\frac1h,\qquad
H_{\mathrm{nc}}(g):=\sum_{\substack{h\in G\setminus\{g\}\\ \gcd(g,h)>1}}\frac1h.
\]
Then
\[
T(g):=\sum_{h\ne g}\frac1{\mathrm{lcm}(g,h)}
\le
\frac{\mu_{-g}}{g}+\Bigl(1-\frac1g\Bigr)H_{\mathrm{nc}}(g).
\]

**Proof.**
Using \(\mathrm{lcm}(g,h)=gh/\gcd(g,h)\),
\[
\frac1{\mathrm{lcm}(g,h)}=\frac{\gcd(g,h)}{gh}.
\]
If \(\gcd(g,h)=1\), this equals \(1/(gh)\).  
If \(\gcd(g,h)>1\), then \(\gcd(g,h)\le g\), so
\[
\frac{\gcd(g,h)}{gh}\le \frac1h.
\]
Summing over coprime and non-coprime parts gives
\[
T(g)\le \frac{\mu_{-g}-H_{\mathrm{nc}}(g)}{g}+H_{\mathrm{nc}}(g),
\]
which is the claim. \(\square\)

#### Corollary B32 (Near-extremizer forces non-coprime anchor mass)
Assume \(R_A(n,m)>2-\varepsilon\) and let \(g_*\) be the anchor from B26.  
Define
\[
L_*:=\frac{2}{r}\left(\mu_G\frac{1-\varepsilon}{2-\varepsilon}-\frac{r}{n}\right),
\quad
\mu_*:=\mu_G-\frac1{g_*}.
\]
Then
\[
H_{\mathrm{nc}}(g_*)
\ge
\frac{L_*-\mu_*/g_*}{1-1/g_*}.
\]

**Proof.**
B26 gives \(T(g_*)>L_*\).  
Apply Lemma B31 with \(g=g_*\): if \(T(g_*)>L_*\), then
\[
L_*<
\frac{\mu_*}{g_*}+\Bigl(1-\frac1{g_*}\Bigr)H_{\mathrm{nc}}(g_*),
\]
and rearrange. \(\square\)

**GL2 refinement.**
Near-extremizers do not only require a small anchor; they require a quantitatively large harmonic mass of generators sharing factors with that anchor.  
This pushes the remaining GL2 step toward a prime-factor incidence theorem on reduced divisor antichains.

#### Lemma B33 (Prime-fiber reduction from anchor mass)
Let \(g_*\in G\), and let \(P(g_*)\) be the set of primes dividing \(g_*\).
For each \(p\in P(g_*)\), define
\[
H_p(g_*):=\sum_{\substack{h\in G\setminus\{g_*\}\\ p\mid h}}\frac1h.
\]
Then
\[
H_{\mathrm{nc}}(g_*)\le \sum_{p\mid g_*} H_p(g_*).
\]
Consequently, there exists \(p_*\mid g_*\) such that
\[
H_{p_*}(g_*)\ge \frac{H_{\mathrm{nc}}(g_*)}{\omega(g_*)},
\]
where \(\omega(g_*)\) is the number of distinct prime factors of \(g_*\).

**Proof.**
If \(\gcd(g_*,h)>1\), then \(h\) is divisible by at least one prime \(p\mid g_*\). Thus
\[
1_{\gcd(g_*,h)>1}\le \sum_{p\mid g_*}1_{p\mid h}.
\]
Multiply by \(1/h\), sum over \(h\in G\setminus\{g_*\}\), getting the first inequality.  
The second is the pigeonhole principle over at most \(\omega(g_*)\) prime fibers. \(\square\)

#### Corollary B34 (Near-extremizer heavy prime fiber)
Under the near-extremizer hypothesis \(R_A(n,m)>2-\varepsilon\), there exists \(p_*\mid g_*\) such that
\[
H_{p_*}(g_*)
\ge
\frac{1}{\omega(g_*)}
\cdot
\frac{L_*-\mu_*/g_*}{1-1/g_*},
\]
with \(L_*,\mu_*\) as in B32.

**Proof.**
Combine B32 and B33. \(\square\)

**Current GL2 bottleneck (fully explicit).**
To finish GL2, it now suffices to prove a nontrivial upper bound on \(H_p(g)\) for reduced divisor antichains \(G\), strong enough to contradict B34 in the near-extremizer regime.

#### Lemma B35 (Prime anchor exclusion in near-extremizers)
Assume \(R_A(n,m)>2-\varepsilon\) and let \(g_*\) be the anchor from B26.
If \(g_*\) is prime, then necessarily
\[
L_*\le \frac{\mu_*}{g_*},
\]
where \(L_*,\mu_*\) are as in B32.
Hence, whenever \(L_*>\mu_*/g_*\), the anchor \(g_*\) cannot be prime.

**Proof.**
If \(g_*=p\) prime and \(h\in G\setminus\{g_*\}\) satisfies \(\gcd(g_*,h)>1\), then \(p\mid h\), so \(g_*=p\mid h\), impossible because \(G\) is reduced (no divisibility among distinct generators). Therefore
\[
H_{\mathrm{nc}}(g_*)=0.
\]
Applying B32:
\[
0=H_{\mathrm{nc}}(g_*)\ge \frac{L_*-\mu_*/g_*}{1-1/g_*},
\]
which implies \(L_*\le\mu_*/g_*\). \(\square\)

**Structural consequence.**
Near-extremizer analysis can be restricted to composite anchors \(g_*\) (and, more specifically, to anchors with enough prime-factor incidence to support the B32 lower bound).

#### Lemma B36 (Prime-power anchor exclusion criterion)
Assume \(R_A(n,m)>2-\varepsilon\), and let \(g_*=p^k\) (\(p\) prime, \(k\ge2\)) be the anchor.
Then
\[
L_*\le \frac{\mu_*}{p}
\quad\Longrightarrow\quad
\text{no contradiction from B32,}
\]
while
\[
L_*>\frac{\mu_*}{p}
\quad\Longrightarrow\quad
g_*=p^k\ \text{is impossible}.
\]

**Proof.**
For any \(h\in G\setminus\{g_*\}\),
\[
\frac1{\mathrm{lcm}(g_*,h)}
=\frac{\gcd(g_*,h)}{g_*h}
\le \frac{1}{h}.
\]
Because \(G\) is reduced, \(h\) cannot be a multiple of \(g_*=p^k\), so \(\gcd(g_*,h)\) is a proper divisor of \(p^k\), hence
\[
\gcd(g_*,h)\le p^{k-1}=g_*/p.
\]
Therefore
\[
\frac1{\mathrm{lcm}(g_*,h)}\le \frac1{ph},
\]
and summing over \(h\in G\setminus\{g_*\}\):
\[
T(g_*)\le \frac{\mu_*}{p}.
\]
By B26/B32, near-extremizer lower bound gives \(T(g_*)>L_*\). So \(L_*>\mu_*/p\) is impossible. \(\square\)

**Note.**
This criterion reduces the remaining GL2 branch to composite anchors where \(L_*\le \mu_*/p\) for every prime \(p\mid g_*\), i.e. anchors with sufficiently rich multi-prime incidence.

#### Lemma B37 (Unified anchor contraction bound)
Let \(g\in G\), and let
\[
p_{\min}(g):=\min\{p\ \text{prime}: p\mid g\},\qquad
\mu_{-g}:=\sum_{h\in G\setminus\{g\}}\frac1h.
\]
Then
\[
T(g):=\sum_{h\ne g}\frac1{\mathrm{lcm}(g,h)}
\le
\frac{\mu_{-g}}{p_{\min}(g)}.
\]

**Proof.**
For \(h\in G\setminus\{g\}\), reducedness implies \(g\nmid h\), hence
\[
\gcd(g,h)<g.
\]
Any proper divisor of \(g\) is at most \(g/p_{\min}(g)\), so
\[
\gcd(g,h)\le \frac{g}{p_{\min}(g)}.
\]
Therefore
\[
\frac1{\mathrm{lcm}(g,h)}
=\frac{\gcd(g,h)}{gh}
\le \frac{1}{p_{\min}(g)\,h}.
\]
Summing over \(h\ne g\) gives the claim. \(\square\)

#### Corollary B38 (Unified near-extremizer anchor exclusion)
Under \(R_A(n,m)>2-\varepsilon\), for the anchor \(g_*\) from B26,
\[
L_*\le \frac{\mu_*}{p_{\min}(g_*)}.
\]
Hence any parameter regime forcing
\[
L_*>\frac{\mu_*}{p_{\min}(g_*)}
\]
is impossible.

**Proof.**
B26 gives \(T(g_*)>L_*\), while B37 gives \(T(g_*)\le \mu_*/p_{\min}(g_*)\). \(\square\)

**Update of GL2 bottleneck.**
The remaining branch is exactly where
\[
L_*\le \frac{\mu_*}{p_{\min}(g_*)}
\]
and simultaneously B34 forces a large prime-fiber mass.  
Finishing GL2 amounts to proving these two constraints are incompatible except in singleton-dominant structure.

### I. Counterexample Profile (exact-\(2\) regime)

Assume, for contradiction, there is a violating pair \(m>n\ge\max(A)\) with
\[
\frac{M_A(m)}{m}\ge 2\frac{M_A(n)}{n}. \tag{CE}
\]
Then all prior near-extremizer lemmas apply with \(\varepsilon=0\).

Set \(r:=|G|\), \(G=A_{\min}\), and choose anchor \(g_*\) from B26.
Then
\[
L_*=\frac{2}{r}\left(\frac{\mu_G}{2}-\frac{r}{n}\right)=\frac{\mu_G}{r}-\frac{2}{n}.
\]

#### Lemma B39 (Necessary anchor inequalities for a true counterexample)
Under (CE), the anchor \(g_*\) must satisfy
\[
\frac{\mu_G}{r}-\frac{2}{n}
\le
\frac{\mu_G-\frac1{g_*}}{p_{\min}(g_*)}, \tag{N1}
\]
\[
\mu_G\frac12
\le
\frac{r(r-1)}{4g_*}+\frac{r}{n}, \tag{N2}
\]
and there exists \(p_*\mid g_*\) with
\[
H_{p_*}(g_*)
\ge
\frac{1}{\omega(g_*)}
\cdot
\frac{\left(\frac{\mu_G}{r}-\frac{2}{n}\right)-\frac{\mu_G-\frac1{g_*}}{g_*}}
{1-1/g_*}. \tag{N3}
\]

**Proof.**
(N1) is B38 at \(\varepsilon=0\).  
(N2) is B30 at \(\varepsilon=0\).  
(N3) is B34 at \(\varepsilon=0\). \(\square\)

#### Corollary B40 (Asymptotic shape of any counterexample sequence)
For any sequence of hypothetical counterexamples with \(n\to\infty\), one must have eventually
\[
\frac{\mu_G}{r}\lesssim \frac{\mu_G}{p_{\min}(g_*)},
\]
hence
\[
p_{\min}(g_*)\lesssim r.
\]

**Proof.**
From (N1), as \(n\to\infty\), correction \(2/n\to0\), and \((\mu_G-1/g_*)/p_{\min}\le \mu_G/p_{\min}\). \(\square\)

**Meaning.**
Any true counterexample is forced into a narrow arithmetic corridor:
- anchor must be small enough to satisfy (N2),
- smallest anchor prime factor cannot be too large relative to \(r\) (B40),
- and one prime-fiber harmonic mass must be large (N3).
The remaining proof task is to show this corridor is empty for reduced antichains.

#### Lemma B42 (Quantitative exclusion when \(p_{\min}(g_*)>r\))
Assume the exact-counterexample regime (CE), and let \(g_*\) be the anchor from B39 with
\[
p:=p_{\min}(g_*),\qquad p>r.
\]
Then
\[
n<\frac{2}{\mu_G\left(\frac1r-\frac1p\right)}.
\]
In particular, if \(p\ge r+1\), then
\[
n<\frac{2r(r+1)}{\mu_G}.
\]

**Proof.**
From (N1),
\[
\frac{\mu_G}{r}-\frac{2}{n}\le \frac{\mu_G-\frac1{g_*}}{p}\le \frac{\mu_G}{p}.
\]
Hence
\[
\mu_G\left(\frac1r-\frac1p\right)\le \frac{2}{n},
\]
which gives the first claim.  
For \(p\ge r+1\),
\[
\frac1r-\frac1p\ge \frac1r-\frac1{r+1}=\frac1{r(r+1)},
\]
yielding the second bound. \(\square\)

**Consequence.**
Any infinite family of counterexamples must lie in the branch
\[
p_{\min}(g_*)\le r
\]
eventually; the \(p_{\min}(g_*)>r\) branch is quantitatively finite.

#### Lemma B43 (Prime-fiber packing upper bound)
Let \(G\) be reduced, \(g\in G\), and let \(p\mid g\) be prime. Then
\[
H_p(g):=\sum_{\substack{h\in G\setminus\{g\}\\ p\mid h}}\frac1h
\le \frac{r-1}{2p},
\qquad r:=|G|.
\]

**Proof.**
If \(h\in G\setminus\{g\}\) and \(p\mid h\), then \(h\neq p\): otherwise \(h=p\mid g\), contradicting reducedness (distinct generators cannot divide each other). Hence \(h\ge 2p\), so each term \(1/h\le 1/(2p)\). There are at most \(r-1\) such terms. \(\square\)

#### Corollary B44 (Residual-branch necessary inequality)
In the exact-counterexample regime (CE), for the prime \(p_*\mid g_*\) from B34,
\[
\frac{1}{\omega(g_*)}
\cdot
\frac{\left(\frac{\mu_G}{r}-\frac{2}{n}\right)-\frac{\mu_G-\frac1{g_*}}{g_*}}
{1-1/g_*}
\le
\frac{r-1}{2p_*}.
\]

**Proof.**
Lower bound from B34, upper bound from B43. \(\square\)

**Residual branch now.**
Together with B42, any hypothetical infinite counterexample family must satisfy simultaneously:
\[
p_{\min}(g_*)\le r,\quad \text{and}\quad \text{B44}.
\]
So the last step is a structural contradiction theorem for reduced antichains in this low-prime, bounded-incidence corridor.

### J. Density Split (global simplification)

Write
\[
D(x):=\frac{M_A(x)}{x}.
\]

#### Lemma B41 (Dense-\(n\) case is automatic)
If \(D(n)\ge \frac12\), then for every \(m>n\),
\[
D(m)<2D(n).
\]

**Proof.**
Since \(1\notin A\), not every integer is divisible by an element of \(A\), hence \(D(m)<1\) for all \(m\).
If \(D(n)\ge1/2\), then \(2D(n)\ge1\), so \(D(m)<1\le2D(n)\). \(\square\)

So EP-488 reduces to the sparse regime
\[
D(n)<\frac12. \tag{S}
\]

#### Reduction target in sparse regime
Let \(G=A_{\min}\), \(r:=|G|\), and
\[
I(n):=\sum_{g\in G}\Big\lfloor\frac{n}{g}\Big\rfloor,\qquad M:=M_A(n).
\]
Because
\[
\sum_{g\in G}\frac1g
\le \sum_{g\in G}\frac{\lfloor n/g\rfloor+1}{n}
=\frac{I(n)+r}{n},
\]
a sufficient condition for EP-488 (via Lemma B22) is
\[
I(n)+r\le 2M. \tag{K}
\]
Indeed then \(\mu_G\le 2M/n\), and for all \(m>n\),
\[
\frac{M_A(m)}{m}<\mu_G\le 2\frac{M_A(n)}{n}.
\]

**Final bottleneck reformulated.**
After B41, it suffices to prove (K) for all reduced \(G\) in the sparse regime (S).  
This is a purely combinatorial incidence inequality and would complete the full theorem.

#### Lemma B45 (Equivalent forms of the sparse bottleneck)
Let
\[
\deg_G(x):=\#\{g\in G:\ g\mid x\},
\]
and define
\[
\mathrm{Dup}:=\sum_{1\le x\le n,\ \deg_G(x)\ge1}\bigl(\deg_G(x)-1\bigr),
\qquad
N_{\mathrm{nb}}:=M_A(n)-r.
\]
Then the condition (K)
\[
I(n)+r\le 2M_A(n)
\]
is equivalent to each of:
\[
\mathrm{Dup}\le N_{\mathrm{nb}}, \tag{K1}
\]
\[
\sum_{x\le n,\ \deg_G(x)\ge1}\deg_G(x)\le 2M_A(n)-r. \tag{K2}
\]

**Proof.**
By definition
\[
I(n)=\sum_{x\le n}\deg_G(x),
\qquad
\mathrm{Dup}=I(n)-M_A(n).
\]
Hence
\[
I(n)+r\le 2M_A(n)
\iff I(n)-M_A(n)\le M_A(n)-r
\iff \mathrm{Dup}\le N_{\mathrm{nb}}.
\]
This also gives (K2). \(\square\)

#### Lemma B46 (Pair-overlap sufficient criterion for (K))
Define pair-overlap incidence
\[
P(n):=\sum_{g<h}\Big\lfloor\frac{n}{\mathrm{lcm}(g,h)}\Big\rfloor
=\sum_{x\le n}\binom{\deg_G(x)}{2}.
\]
If
\[
P(n)\le N_{\mathrm{nb}},
\]
then (K) holds.

**Proof.**
For each \(d\ge1\), \(\binom d2\ge d-1\). Summing over \(x\le n\):
\[
P(n)=\sum_x \binom{\deg_G(x)}2
\ge \sum_x (\deg_G(x)-1)_+
=\mathrm{Dup}.
\]
Thus \(P(n)\le N_{\mathrm{nb}}\) implies \(\mathrm{Dup}\le N_{\mathrm{nb}}\), i.e. (K1), hence (K). \(\square\)

**Important correction.**
The condition \(P(n)\le M_A(n)-r\) is only sufficient, not necessary, and it is false in general even under sparse density \(D(n)<1/2\).  
Counterexample:
\[
G=\{4,6,9,10,14,15\},\quad n=60,
\]
for which
\[
D(n)=\frac{27}{60}<\frac12,\qquad
P(n)=23,\qquad
M_A(n)-r=27-6=21.
\]
So \(P(n)>M_A(n)-r\), yet EP-488 still holds for this instance.

Therefore the true remaining theorem target is (K) itself:
\[
I(n)+r\le 2M_A(n)
\]
in sparse regime, without replacing it by the stronger \(P\)-bound.

#### Lemma B47 (Order-slack identity for (K))
Fix any ordering \(G=\{g_1,\dots,g_r\}\). Let
\[
u_i:=\Big\lfloor\frac{n}{g_i}\Big\rfloor,\qquad
t_i:=\#\Bigl(g_i\mathbb N\cap[1,n]\setminus \bigcup_{j<i} g_j\mathbb N\Bigr).
\]
Then
\[
I(n)=\sum_{i=1}^r u_i,\qquad
M_A(n)=\sum_{i=1}^r t_i,
\]
and
\[
2M_A(n)-I(n)-r
=
\sum_{i=1}^r\bigl(2t_i-u_i-1\bigr). \tag{S}
\]
Hence (K) is equivalent to nonnegativity of the total slack in (S).

**Proof.**
Immediate from definitions of \(I,M\). \(\square\)

#### Corollary B48 (Local-to-global sufficient criterion)
If there exists an ordering with
\[
t_i\ge \frac{u_i+1}{2}\quad\text{for all }i,
\]
then (K) holds.

**Proof.**
Each summand \(2t_i-u_i-1\ge0\), so (S) is nonnegative. \(\square\)

**Deep remaining target (current best).**
Prove existence of such an ordering (or a weaker averaged variant implying (S)\(\ge0\)) for reduced antichains in sparse regime \(D(n)<1/2\).  
This would finish EP-488 from B41.

#### Open Lemma OL1 (Order-slack existence, exact form)
For every reduced \(G\) and \(n\ge \max G\) with \(D(n)<1/2\), there exists an ordering \(g_1,\dots,g_r\) such that
\[
\sum_{i=1}^r(2t_i-u_i-1)\ge0
\]
(equivalently \(I(n)+r\le 2M_A(n)\)).

This is exactly equivalent to the remaining sparse-case theorem.

#### Why current inequalities still fall short
Current global constraints (B39–B44) force a narrow counterexample corridor but do not yet imply OL1:
- B38 gives anchor contraction \(L_*\le \mu_*/p_{\min}(g_*)\),
- B44 gives a prime-fiber incidence inequality,
- B42 makes \(p_{\min}>r\) quantitatively finite.

These are necessary constraints, but not yet contradictory in the residual branch \(p_{\min}(g_*)\le r\).

#### Next-proof directions from OL1
1. Prove OL1 via a deterministic ordering rule (descending/greedy maximum new mass) with an averaged slack estimate.
2. Prove a direct inequality \(\mathrm{Dup}\le N_{\mathrm{nb}}\) from divisor-incidence decomposition by prime fibers.
3. Strengthen B44 with a sharper \(H_p\)-upper bound that uses reducedness + sparse density simultaneously (not only \(h\ge 2p\)).
