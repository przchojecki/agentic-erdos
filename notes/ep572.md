# EP-572

## Problem Statement
For each \(k\ge 3\), prove
\[
\operatorname{ex}(n;C_{2k})\gg n^{1+1/k}.
\]

## Literature / Context
- The target exponent is known in specific cases (\(k=3,5\), attributed in prior notes to Benson-type constructions).
- General lower bounds exist but with weaker exponent than \(1+1/k\).

## Our Approaches / What Is Proven
- Construction direction tracked: randomized-greedy \(C_{2k}\)-avoidance to probe finite-\(n\) constants.
- This does not prove the asymptotic exponent, but helps identify which \(k\)-ranges are numerically consistent with the conjectural scale.
- Current theorem-level status in this repo: partial (special \(k\) cases noted), full statement open.

## Computation-Guided Observations
- Standalone deeper run records best found \(C_{2k}\)-free edge counts for \(k=3,4,5\) and moderate \(n\).
- Output is used as empirical support only; not a substitute for explicit algebraic constructions.
- Latest deep run (about \(98\)s) gives normalized values near order \(1\) for \(k=4,5\) at tested scales, while \(k=3\) shows stronger finite-size drift.

## Next Proof Targets
- Convert the strongest observed finite constructions into explicit deterministic templates (incidence/finite-geometry style).
- Separate tasks by \(k\): close \(k=4\) first, then bootstrap families for larger \(k\).
