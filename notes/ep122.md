## Deeper Proof Program (A => B => C)

### Target
Characterize arithmetic functions \(f\) for which local overcrowding of values
\(n+f(n)\) in windows \((x,x+F(x))\) must diverge under broad growth assumptions
on \(F\).

### Step A: Multiplicity decomposition
Write local counts via fibers of \(n\mapsto n+f(n)\) and separate smooth/rough
input regimes.

### Step B: Mean-square control
Estimate second moments of local multiplicities to detect forced peaks.

### Step C: Function-class criterion
Extract necessary/sufficient conditions on distribution of \(f(n+1)-f(n)\) or
inverse-image growth.

### Concrete Blocking Lemma (Most Critical)
Need a general inverse-image bound: for most \(y\), control of
\(\#\{n: n+f(n)=y\}\) with error uniform enough for moving windows.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AJ1 (Second-moment forcing)
Let \(M_x(y)\) count \(n\) with \(x<n+f(n)\le x+F(x)\) and value \(y\). Then
\[
\max_y M_x(y)\ge \frac{\left(\sum_y M_x(y)\right)^2}{\sum_y M_x(y)^2}.
\]

\[
\textit{Proof.}
\]
Since \(\sum_y M_x(y)^2 \le (\max_y M_x(y))\sum_y M_x(y)\), rearrange.
\(\square\)

### Proposition AJ2 (Reduction)
If one proves \(\sum_y M_x(y)\gg F(x)\) and
\(\sum_y M_x(y)^2\ll F(x)/\omega(x)\) for some \(\omega(x)\to\infty\), then
\[
\max_y M_x(y)\to\infty.
\]

\[
\textit{Proof.}
\]
Apply AJ1 to obtain \(\max_y M_x(y)\gg \omega(x)\).
\(\square\)
