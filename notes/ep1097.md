# EP-1097

## Problem Statement
Given a set $A$ of $n$ integers, let
\[
D(A)=\{d>0:\exists x\ \text{with}\ x,x+d,x+2d\in A\}.
\]
How large can $|D(A)|$ be? Is $|D(A)|=O(n^{3/2})$ always?

## Literature
- Problem links to Bourgain’s sums-differences/Kakeya-type exponent.
- Current known exponent window (as recorded in background) is between about $1.77898$ and $11/6$.
- Whether the $3/2$ target is achievable as an upper bound remains open.

## Our Approaches / What Is Proven
- Replaced quick batch sampling with standalone deep local optimization (`scripts/ep1097.mjs`):
  - random restarts,
  - swap-based hill-climbing / annealing,
  - exact recomputation of distinct 3-AP differences at each move.
- Ran time-budgeted search across multiple $n$ values in dense intervals $[1,M]$ with $M=12n$.

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- $n\in\{44,56,68,80\}$,
- 30 seconds per $n$ (total 120 seconds),
- objective evaluations per size in the range $146{,}983$ to $432{,}394$.

Best values found:
- $n=44$: $|D|=114$,
- $n=56$: $|D|=163$,
- $n=68$: $|D|=212$,
- $n=80$: $|D|=267$.

Normalized profile from this run:
\[
\frac{|D|}{n^{3/2}}\approx 0.391,\ 0.389,\ 0.378,\ 0.373,
\]
with slowly decreasing trend in this window.

These are finite lower-bound constructions only; they do not determine the true asymptotic exponent.

## Deeper Proof Program (A => B => C)

### Target
For \(A\subset\mathbb Z\), \(|A|=n\), bound
\[
|D(A)|,\quad D(A):=\{d>0:\exists x,\ x,x+d,x+2d\in A\},
\]
ideally by \(O(n^{3/2})\).

### A. Concrete Blocking Lemma (Most Critical)

Prove an incidence-energy inequality:
\[
|D(A)|^{2}\ll E_+(A)\,n^{-1}
\]
with \(E_+(A)=|\{a+b=c+d\}|\), together with a universal bound \(E_+(A)\ll n^{4-\delta}\) in the extremal regime forcing \(|D(A)|\ll n^{3/2-\delta/2}\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (3-AP count decomposition)
Let \(T_d=\#\{x:\ x,x+d,x+2d\in A\}\). Then
\[
\sum_{d>0} T_d = \#\{\text{3-APs in }A\}.
\]

**Proof.**
Each nontrivial 3-AP has unique positive common difference \(d\), so summing over \(d\) partitions all 3-APs. \(\square\)

#### Lemma B2 (support-size inequality)
If \(S=\{d:T_d>0\}=D(A)\), then
\[
|D(A)|\le \frac{\left(\sum_d T_d\right)^2}{\sum_d T_d^2}.
\]

**Proof.**
Cauchy-Schwarz on \((T_d)_{d\in S}\): \((\sum_d T_d)^2\le |S|\sum_d T_d^2\). Rearrangement gives the claim. \(\square\)

#### Lemma B3 (A plus additive-energy control implies target)
If \(\sum_d T_d^2\) admits a lower bound in terms of \(\sum_d T_d\) and additive energy as in A, then \(|D(A)|=O(n^{3/2})\) follows.

**Proof.**
Insert the structural lower bound for \(\sum_d T_d^2\) into Lemma B2 and simplify exponents. \(\square\)

### C. Main unresolved point

The key obstacle is a sharp universal relation between 3-AP difference multiplicities \(T_d\) and additive energy in the worst-case extremal sets.
