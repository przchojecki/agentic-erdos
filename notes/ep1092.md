# EP-1092

## Problem Statement
For $r=2$, let $f_2(n)$ be maximal such that if every $m$-vertex subgraph $H$ of a graph $G$ can be written as
\[
H = B \cup E_0
\]
with $B$ bipartite and $|E_0|\le f_2(m)$, then $\chi(G)\le 3$.
Is $f_2(n)\gg n$? More generally, is $f_r(n)\gg_r n$?

## Literature
- The $r=2$ linear-growth claim is known false (Rödl's construction).
- The broader general-$r$ behavior remains open in this repository.

## Our Approaches / What Is Proven
- Added standalone deep finite proxy computation (`scripts/ep1092.mjs`) based on
\[
b(H)=|E(H)|-\operatorname{MaxCut}(H),
\]
the exact minimum number of edge deletions needed to make $H$ bipartite.
- Evaluated induced subgraphs sampled from high-chromatic Mycielski graphs:
  - Mycielski($C_5$), $n=11$, $\chi\ge 4$,
  - second Mycielski lift, $n=23$, $\chi\ge 5$.
- For each $m$, sampled 1200 induced subgraphs and computed exact max-cut (thus exact $b(H)$) per sample.

## Computation-Guided Observations
Deep run (`DEPTH=4`) over
\[
m\in\{6,7,8,9,10,11,12\}
\]
showed finite sampled worst-case values up to:
- on $n=11$ graph: best found $b(H)=4$ (at $m=11$),
- on $n=23$ graph: best found $b(H)=4$ (at $m=11,12$).

Corresponding sampled ratios $b(H)/m$ are around $0.14$ to $0.36$ in this range.
These are finite sampled envelopes only; they do not determine asymptotics for $f_2(m)$.

## Status
- First subquestion ($r=2$ linear lower bound): already false in literature.
- General-$r$ behavior: open.
- Added deep standalone finite diagnostic on exact edge-bipartization profiles in explicit high-chromatic constructions.
