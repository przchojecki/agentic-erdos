# EP-137 partial attempt

## Route
Performed a sliding-window exact factor-exponent scan for products of `k` consecutive integers over broad ranges.

## Evidence from this batch
- `data/ep137_powerful_consecutive_scan.json`: no powerful product witness found for `3 <= k <= 12` and `m <= 300000`.

## Hard point
Nonexistence for all `m` and fixed `k>=3` is a global arithmetic statement; large finite scans cannot prove impossibility.

## Status
No counterexample found in tested range; conjectural nonexistence remains open.

## Deeper Proof Program (A => B => C)

### Target
For fixed \(k\ge 3\), prove the product of \(k\) consecutive integers is never
powerful (or classify exceptional \(k\) if any).

### Step A: \(p\)-adic valuation profile
Track \(v_p\!\left(\prod_{i=0}^{k-1}(m+i)\right)\) by counting hits of \(p,p^2\)
inside short intervals.

### Step B: Forced single-prime occurrence
Show existence of a prime \(p\) dividing exactly one factor \(m+i\) to odd/low
exponent, forcing non-powerfulness.

### Step C: Uniform \(k\)-dependent closure
Prove the forcing argument for all \(m\) once \(k\) is fixed.

### Concrete Blocking Lemma (Most Critical)
Need a short-interval prime-power distribution lemma ensuring at least one prime
contributes valuation exactly \(1\) in every block of length \(k\).

## LaTeX Proofs for A => B => C Skeleton

### Proposition AR1 (Powerful criterion)
An integer \(N\) is powerful iff for every prime \(p\mid N\), \(v_p(N)\ge 2\).

\[
\textit{Proof.}
\]
This is the definition of powerful numbers.
\(\square\)

### Proposition AR2 (Reduction for consecutive products)
Let
\[
P_{m,k}:=\prod_{i=0}^{k-1}(m+i).
\]
If one can show that for every \(m\) there exists a prime \(p\) with
\(v_p(P_{m,k})=1\), then \(P_{m,k}\) is not powerful for any \(m\).

\[
\textit{Proof.}
\]
Immediate from AR1.
\(\square\)
