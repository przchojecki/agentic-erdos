# EP-1087 deeper attempt

## Statement
Estimate extremal number of "degenerate" 4-point subsets (with repeated pairwise distance)
in `n`-point planar sets.

## Computation in this attempt
I sampled random integer point sets and counted degenerate quadruples (literal repeated-distance
interpretation).

Data file:
- `data/ep1087_random_quadruple_scan.json`

For sampled sets up to `n=32`, all sampled counts were `0`.

## Interpretation
This is only typical-case evidence under random generic placement. The problem is extremal and
asks for worst-case scaling, so this does not resolve the conjectured bound.
