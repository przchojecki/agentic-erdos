# EP-1087 deeper attempt

## Statement
Estimate extremal number of "degenerate" 4-point subsets (with repeated pairwise distance)
in `n`-point planar sets.

## Computation in this attempt
I sampled random integer point sets and counted degenerate quadruples (literal repeated-distance
interpretation).

Data file:
- `data/ep1087_random_quadruple_scan.json`

For sampled sets up to `n=32`, all sampled counts were `0`.

## Interpretation
This is only typical-case evidence under random generic placement. The problem is extremal and
asks for worst-case scaling, so this does not resolve the conjectured bound.

## Deeper Proof Program (A => B => C)

### Target
Estimate the extremal number of 4-point subsets in planar \(n\)-point sets that exhibit a repeated pairwise distance (distance-degenerate quadruples).

### A. Concrete Blocking Lemma (Most Critical)

Show that every such degenerate quadruple can be injected (up to bounded multiplicity) into an incidence configuration between points and equal-distance loci, yielding
\[
Q(n)\ll n^{2+\varepsilon}
\]
for all \(\varepsilon>0\) (or a matching lower-scale target).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (distance graph reformulation)
For each distance value \(r\), let \(G_r\) be graph on \(P\) with edges \(\{x,y\}\) at distance \(r\). A quadruple with repeated distance exists iff some \(G_r\) contributes at least two edges on the same 4-vertex set.

**Proof.**
Repeated distance means two distinct pairs share the same length \(r\). Those pairs are edges in \(G_r\), both contained in the chosen 4-set. Converse is immediate. \(\square\)

#### Lemma B2 (energy upper reduction)
If \(m_r=|E(G_r)|\), then
\[
Q(n)\ll \sum_r m_r^2
\]
up to absolute constants.

**Proof.**
Choose the repeated distance \(r\), then choose an ordered pair of edges of \(G_r\); each choice determines at most constant-many quadruples after quotienting symmetries/overcounting. \(\square\)

#### Lemma B3 (A implies global bound)
If Lemma A gives \(\sum_r m_r^2\ll n^{2+\varepsilon}\), then \(Q(n)\ll n^{2+\varepsilon}\).

**Proof.**
Direct from Lemma B2. \(\square\)

### C. Main unresolved point

The bottleneck is obtaining sharp control of the distance-energy sum \(\sum_r m_r^2\) in worst-case planar configurations.
