# EP-884

## Problem Statement
For divisors $d_1<\cdots<d_t$ of $n$, is it true that
$$\sum_{1\le i<j\le t}\frac1{d_j-d_i}\ll 1+\sum_{1\le i<t}\frac1{d_{i+1}-d_i},$$
with an absolute implied constant?

## Literature
- The local dataset had malformed metadata around this entry, but the inequality statement itself is available.
- The core question is whether a uniform ratio bound holds over all $n$.

## Our Approaches / What Is Proven
- Performed direct finite scan up to $n\le 200000$.
- Computed ratio
  $$R(n)=\frac{\sum_{i<j}1/(d_j-d_i)}{\sum_i 1/(d_{i+1}-d_i)}$$
for each $n$ with at least 3 divisors.

## Computation-Guided Observations
- Maximum observed ratio in this range:
  $$R(n)\approx 3.56587\quad\text{at }n=166320.$$
- Running maxima stayed in a small-constant range over checkpoints.
- Finite evidence is consistent with the conjectured bounded-ratio phenomenon.
