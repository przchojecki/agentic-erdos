# EP-616

## Problem Statement
For \(r\)-uniform hypergraphs with a local transversal constraint on small induced subhypergraphs, determine the best global bound \(t\) for \(\tau(G)\).

## Literature / Context
- Prior notes record linear-in-\(r\) bounds:
  \[
  \frac{3}{16}r+\frac{7}{8}\le t\le \frac{1}{5}r.
  \]
- Exact constant/structure remains open.

## Our Approaches / What Is Proven
- Implemented exact small-\(n\) finite proxy search:
  - random \(r\)-uniform hypergraphs,
  - enforce local condition via exact induced-subfamily transversal checks (using a local \(\tau\le 2\) proxy),
  - compute global \(\tau(G)\) exactly.

## Computation-Guided Observations
- Accepted local-\(\tau\le2\) instances are frequent for tested small \(r,n\).
- Observed max global \(\tau\) in accepted samples is \(4\) (for \(r=3,4\) tested), with average values around \(3.16\)–\(3.76\).
- Finite data supports linear-scale behavior but does not identify sharp asymptotic coefficient.

## Next Proof Targets
- Align the local condition exactly with the intended statement normalization, then rerun exact finite search.
- Seek constructive families saturating either side of the known linear bounds.
