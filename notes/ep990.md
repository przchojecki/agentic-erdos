# EP-990 partial resolution

## Statement
Discrepancy of root arguments for sparse polynomials: target bound scales like
sqrt(n log M) with n nonzero coefficients.

## What is resolved
Background says Erdos-Turan proved such a bound with n replaced by degree d.

## What remains open in this note
Replacing degree d by sparsity n in the same scale is not settled here.

## Status
- classical degree-based bound known.
- sparsity-refined bound unresolved.

## Classical proof backbone (Erdos-Turan form)
Let \(z_1,\dots,z_d\) be roots (with multiplicity) and \(\theta_j=\arg z_j/(2\pi)\in\mathbb R/\mathbb Z\).
Discrepancy of root arguments on arcs can be bounded via trigonometric moments:
\[
D_d \ll \frac1H + \sum_{h=1}^H \frac1h\left|\frac1d\sum_{j=1}^d e(h\theta_j)\right|.
\]
This is the Erd\H{o}s--Tur\'an discrepancy inequality.

For polynomial roots, the Fourier sums \(\sum_j e(h\theta_j)\) are controlled by coefficients (via logarithmic height / Jensen-type arguments), yielding the classical degree-based discrepancy bound.

### Relevance to sparsity version
To replace degree \(d\) by number of nonzero coefficients \(n\), one needs the same Fourier-discrepancy mechanism with bounds depending on sparsity rather than ambient degree. So the core technical gap is coefficient-to-Fourier control in sparse form.
