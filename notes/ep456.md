# EP-456 partial

## Statement
Compare `m_n` (smallest `m` with `n|phi(m)`) and `p_n` (smallest prime `1 mod n`):
is `m_n<p_n` for almost all `n`? does `p_n/m_n->infty` for almost all `n`?

## Attempt in this batch
I computed exact `(m_n,p_n)` for `2<=n<=1500`.

Data file:
- `data/ep456_mn_vs_pn_scan.json`

## Result
- In this range, `m_n < p_n` for 358 out of 1499 values (`~23.9%`).
- Equality `m_n=p_n` occurs for `~76.1%` in this range.
- Power-of-two odd-exponent samples (`n=2^(2k+1)`) consistently satisfy strict
  inequality in tested cases with potentially large ratios (e.g. `n=512`,
  `p_n/m_n ~ 9.96`).

## Hard point
Finite frequencies up to 1500 are far from proving almost-all behavior.

## Deeper Proof Program (A => B => C)

### Target
Establish almost-all inequalities between
\[
m_n:=\min\{m:\ n\mid \phi(m)\},\qquad
p_n:=\min\{p\equiv1\pmod n,\ p\ \text{prime}\},
\]
including \(m_n<p_n\) and \(p_n/m_n\to\infty\) for almost all \(n\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a dense family construction: for density-1 set of \(n\), one can build \(m\) with \(n\mid\phi(m)\) using many small prime-power factors, giving
\[
m_n \le n\,(\log n)^{O(1)},
\]
while \(p_n\) is typically much larger on the same set.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (trivial comparison baseline)
\[
m_n\le p_n
\]
always, since \(\phi(p_n)=p_n-1\equiv0\pmod n\).

**Proof.**
Definition of \(p_n\) gives \(n\mid (p_n-1)=\phi(p_n)\), so \(p_n\) is admissible for \(m_n\). \(\square\)

#### Lemma B2 (strict inequality criterion)
If there exists \(m<p_n\) with \(n\mid\phi(m)\), then \(m_n<p_n\).

**Proof.**
By minimality of \(m_n\) among admissible \(m\). \(\square\)

#### Lemma B3 (A implies almost-all ratio divergence)
If A gives \(m_n\le n(\log n)^C\) on density-1 set and known lower behavior gives \(p_n\) larger by superpolylog factor there, then \(p_n/m_n\to\infty\) a.e.

**Proof.**
Take ratio on density-1 set and use comparative growth bounds. \(\square\)

### C. Main unresolved point

The bottleneck is proving a uniform dense-family upper bound for \(m_n\) strong enough to beat almost-all behavior of the least prime \(1\bmod n\).
