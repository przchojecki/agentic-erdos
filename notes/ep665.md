# EP-665

## Problem Statement
Does there exist a constant \(C>0\) such that for all large \(n\), there is a pairwise balanced design on \(n\) points with every block size strictly larger than \(n^{1/2}-C\)?

## Literature
- Erd\H{o}s--Larson gave unconditional \(h(n)\ll n^{1/2-c}\) and conditional \(h(n)\ll (\log n)^2\) under Cram\'er-type prime-gap assumptions.
- Shrikhande--Singhi proved a conditional negative result under the projective-plane prime-power conjecture framework.

## Our Approaches / What Is Proven
- Ran a projective-plane parameter sweep over many prime-power orders \(q\), using the canonical design parameters:
  \(n=q^2+q+1\), minimum block size \(q+1\).
- Computed explicit gap proxy \(\sqrt n-(q+1)\) and scaling summaries.

## Computation-Guided Observations
- For projective-plane points, \(q+1\) exceeds \(\sqrt n\) by about \(0.5\), i.e. the gap proxy is around \(-0.5\).
- This finite structural profile is compatible with constant-level proximity to \(\sqrt n\) along such special \(n\), but does not resolve unrestricted all-large-\(n\) existence.

## Status
- Open unconditionally.
- Current computation clarifies parameter geometry of the projective-plane model.
