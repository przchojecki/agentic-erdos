# EP-805

## Problem Statement
For which functions \(g(n)\) with \((\log n)^2\le g(n)<n\) does there exist an \(n\)-vertex graph such that every induced subgraph on \(g(n)\) vertices contains both:
- a clique of size at least \(\log n\), and
- an independent set of size at least \(\log n\)?
In particular, can one take \(g(n)=(\log n)^3\)?

## Literature
- The threshold window for \(g(n)\) remains open.

## Our Approaches / What Is Proven
- Ran deeper induced-subgraph local-Ramsey scans on random graphs.
- For each \((n,g)\), sampled many host graphs and many induced \(g\)-subsets, testing both clique and independent-set thresholds exactly on the sampled subsets.

## Computation-Guided Observations
- Clear finite transition is visible:
  - at small \(g\), pass fraction is near 0,
  - over a narrow \(g\)-window, pass fraction jumps toward 1.
- The transition window shifts upward as \(n\) increases (e.g. around \(g\sim 48\to72\) for \(n=192,256\)).
