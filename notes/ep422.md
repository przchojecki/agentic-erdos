## Deeper Proof Program (A => B => C)

### Target
Resolve global well-definedness of Hofstadter recursion and infinitude of missed values.

### A. Concrete Blocking Lemma (Most Critical)

Prove a margin lemma:
\[
n-f(n-1)\ge1,\qquad n-f(n-2)\ge1
\]
for all \(n\), with quantitative slack on a density-1 set to prevent argument collapse.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (well-definedness criterion)
Recursion is well-defined at \(n\) iff both indices \(n-f(n-1)\) and \(n-f(n-2)\) are positive integers already defined.

**Proof.**
Direct from recursive definition. \(\square\)

#### Lemma B2 (monotone upper envelope)
If \(f(n)\le n\) for all \(n\), then B1 positivity holds automatically.

**Proof.**
Then \(n-f(n-1)\ge1\) and similarly for \(n-2\). \(\square\)

#### Lemma B3 (A implies global recursion)
If A proves B1 inequalities for all \(n\), recursion is globally well-defined; combined with separate growth-gap arguments, one can deduce infinitely many missed integers.

**Proof.**
Well-definedness from B1; missed-value infinitude follows if image growth lags behind index growth on infinitely many scales. \(\square\)

### C. Main unresolved point

The key obstacle is proving uniform index-positivity without relying on finite-window computation, while simultaneously controlling value-set density.
