# EP-949 partial resolution

## Statement
If `S subset R` has no solutions to `a+b=c`, must there exist continuum-sized
`A subseteq R\S` with `A+A subseteq R\S`?

## What is resolved
Background gives a positive result for the Sidon-restricted variant:
if `S` is Sidon, then such a continuum-sized set `A` exists.

## What remains open in this note
The full statement for arbitrary sum-free `S` is not settled by the provided
background.

## Status
- Sidon `S` variant: resolved.
- general sum-free `S`: unresolved.

## Equivalent reformulation
The target asks for a set \(A\subseteq \mathbb R\setminus S\) of size continuum such that
\[
A+A\subseteq \mathbb R\setminus S.
\]
Equivalently,
\[
(A+A)\cap S=\varnothing.
\]
So one seeks a large set whose pairwise sumset avoids \(S\).

### Simple sufficient condition
If there exists an interval \(I\subset\mathbb R\setminus S\) such that
\[
I+I\subset\mathbb R\setminus S,
\]
then taking \(A=I\) solves the problem immediately (and \(|A|=\mathfrak c\)).

So the difficult cases are those where \(S\) is sum-free but still intersects every candidate interval-sum region \(I+I\), forcing more delicate non-interval constructions.

## Structural sufficient criteria
Each item below yields a continuum-size witness \(A\subseteq \mathbb R\setminus S\)
with \(A+A\subseteq \mathbb R\setminus S\).

1. Additive subgroup criterion:
if there exists an additive subgroup \(G\le \mathbb R\) with \(|G|=\mathfrak c\)
and \(G\cap S=\varnothing\), then \(A:=G\) works since \(G+G=G\).

2. Translation of subgroup criterion:
if \(t+G\subseteq \mathbb R\setminus S\) for some \(t\in\mathbb R\) and subgroup
\(G\) of size \(\mathfrak c\), then \(A:=t+G\) gives
\[
A+A=(2t)+G\subseteq \mathbb R\setminus S
\]
provided \((2t)+G\) also avoids \(S\). So one can reduce to finding a coset pair
disjoint from \(S\).

3. Open-interval criterion (already noted):
if \(I\subseteq \mathbb R\setminus S\) and \(I+I\subseteq \mathbb R\setminus S\),
then \(A:=I\) works.

These criteria isolate where the hard core of EP-949 lies: sum-free sets \(S\)
that intersect every plausible interval or large additive-coset sum region.
