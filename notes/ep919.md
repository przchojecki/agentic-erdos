# EP-919

## Problem Statement
Is there a graph $G$ with vertex set $\omega_2^2$ and chromatic number $\aleph_2$ such that every subgraph whose vertices have lesser type has chromatic number $\leq\aleph_0$?
What if instead we ask for $G$ to have chromatic number $\aleph_1$?

## Literature
- Erdos--Hajnal gave the classical $\omega_1^2$-type lexicographic construction with a strong global/local separation pattern.
- Related constructions on $\omega_2^2$ with weaker local bound ($\leq\aleph_1$) are known.
- The stronger local collapse in this problem remains open.

## Our Approaches / What Is Proven
- Built finite analogue of the lex-square construction on $[m]\times[m]$ with lexicographic order and edge rule
$$
(a_1,b_1)\sim(a_2,b_2)\iff a_1<a_2\ \text{and}\ b_1<b_2.
$$
- Studied chromatic number of initial segments (finite proxy for “lesser type” substructures).
- Derived explicit prefix formula and verified it exactly by coloring computations for $m=4,\dots,10$.

## Computation-Guided Observations
- Exact checks for all tested prefixes matched the formula
$$
\chi(\text{prefix of size }t)=q+\mathbf 1_{r\ge q+1},
$$
where $t=(q)m+r$ in row-wise lex decomposition.
- In particular, in this finite model:
  - full graph has chromatic number $m$,
  - every proper initial segment has chromatic number at most $m-1$.
- Large-scale profile up to $m=400$ shows the largest proper initial segment has ratio $\chi/m=399/400$.
