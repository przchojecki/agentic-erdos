# EP-74 partial resolution

## Statement
Given `f(n)->infinity`, is there an infinite-chromatic graph where every
`n`-vertex subgraph can be made bipartite by deleting at most `f(n)` edges?

## What is resolved
Background gives:
- Rodl construction for graphs of chromatic number `aleph_0` when
  `f(n)=epsilon*n` (any fixed `epsilon>0`).
- Hypergraph analogue proved.

## What remains open in this note
For graphs, sublinear regimes remain open; background explicitly says open even
for `f(n)=sqrt(n)`.

## Status
- linear regime (`epsilon n`) for countable chromatic case: resolved.
- sublinear regime (for example `sqrt(n)`): unresolved.

## Deeper Proof Program (A => B => C)

### Target
Construct an infinite-chromatic graph \(G\) such that every \(n\)-vertex
subgraph \(H\subseteq G\) satisfies
\[
\tau_{\mathrm{odd}}(H)\le f(n),
\]
with \(f(n)\to\infty\) sublinear (e.g. \(n^{1/2}\)).

### Step A: Layered random construction
Build \(G\) as a union of sparse random layers \(G_t\) with carefully tuned
densities so odd cycles are globally spread but locally repairable.

### Step B: Local repair lemma
Prove every \(n\)-vertex induced \(H\) has odd-cycle transversal
\(\tau_{\mathrm{odd}}(H)\le C n^\alpha\) for some \(\alpha<1\).

### Step C: Global chromatic explosion
Show the layered design forces \(\chi(G)=\aleph_0\) (or higher) despite local
near-bipartiteness.

### Concrete Blocking Lemma (Most Critical)
Need a uniform concentration/containers statement controlling odd-cycle packing
in all \(n\)-vertex traces simultaneously; current probabilistic tools are not
tight enough in the sublinear regime.

## LaTeX Proofs for A => B => C Skeleton

### Proposition H1 (Local repair + global obstruction gives solution)
Suppose an infinite graph \(G\) satisfies:
\[
\forall n,\ \forall H\subseteq G,\ |V(H)|=n:\ \tau_{\mathrm{odd}}(H)\le Cn^\alpha
\quad(\alpha<1),
\]
and also every proper coloring of \(G\) uses infinitely many colors. Then \(G\)
is a positive solution with \(f(n)=\lceil Cn^\alpha\rceil\).

\[
\textit{Proof.}
\]
The local inequality is exactly the deletion requirement. The second condition
is \(\chi(G)=\aleph_0\). Hence \(G\) meets both demanded properties.
\(\square\)

### Proposition H2 (Sublinear control is the only missing quantitative step)
If Step C is achieved by construction and Step B is proved with \(\alpha<1\),
then EP-74 is resolved for that exponent profile.

\[
\textit{Proof.}
\]
Immediate from Proposition H1.
\(\square\)
