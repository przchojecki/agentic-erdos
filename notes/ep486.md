# EP-486 partial resolution

## Statement
For forbidden residue sets X_n mod n over n in A, ask whether the uncovered set B
always has logarithmic density.

## What is resolved
Background gives positive result when each X_n={0} (Davenport-Erdos), with an
alternative elementary proof.

## What remains open in this note
The general case with arbitrary residue subsets X_n is unresolved.

## Status
- singleton-residue case solved.
- full multi-residue generalization open.

## New Experiments
- 2026-03-12T10:31:29.160Z: standalone finite log-density profiles
  (`scripts/ep486.mjs`) for several residue-exclusion families.

## One-by-One Cycle
- Replaced placeholder script with explicit construction of `B` and harmonic-mass
  tracking `\sum_{m\in B,m<x} 1/m`.
- In all tested families, normalized profiles by `\log x` show stabilization
  behavior over large finite ranges.
- This supports plausibility of logarithmic-density existence in structured
  finite models, while the full arbitrary `X_n` theorem remains open.
