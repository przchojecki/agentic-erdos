## LaTeX Proofs for A => B => C Skeleton

### Proposition U1 (Pigeonhole link between distances and multiplicity)
For \(n\)-point set \(P\), let \(M(P)\) be maximum multiplicity of a realized
distance and \(D(P)\) number of distinct distances. Then
\[
M(P)\ge \frac{\binom{n}{2}}{D(P)}.
\]

\[
\textit{Proof.}
\]
Partition all \(\binom{n}{2}\) pairs by distance value. Average class size is
\(\binom{n}{2}/D(P)\), so maximum class size is at least this average.
\(\square\)

### Proposition U2 (Reduction to distinct-distance lower bounds)
If \(D(P)\ge L(n)\) for all \(n\)-point sets \(P\), then
\[
M(P)\le C(n)\quad\text{for extremal minimizers only if }C(n)\ge \frac{\binom{n}{2}}{L(n)}.
\]
Thus improving \(L(n)\) immediately improves lower constraints on best-possible
multiplicity scales.

\[
\textit{Proof.}
\]
Immediate from U1.
\(\square\)
