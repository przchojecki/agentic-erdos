## Deeper Proof Program (A => B => C)

### Target
Determine the constant \(c_\alpha\) in
\[
F(n,\alpha)\sim c_\alpha \log n
\]
for fixed \(\alpha\in(0,1/2)\).

### Step A: Sharp upper via discrepancy concentration
Use concentration and union bounds over all subsets to obtain explicit
\(\log n\)-upper constant.

### Step B: Sharp lower via random construction
Show random (or pseudorandom) colorings satisfy the condition up to
\((c_\alpha-o(1))\log n\).

### Step C: Constant matching
Align upper and lower constants to identify \(c_\alpha\).

### Concrete Blocking Lemma (Most Critical)
Need a large-deviation estimate uniform over all subset sizes above threshold
with optimal exponent in \(\alpha\).

## LaTeX Proofs for A => B => C Skeleton

### Proposition BJ1 (First-moment upper template)
If for each subset size \(m\) the failure probability of a random coloring on a
fixed \(m\)-set is at most \(p_m\), then
\[
\mathbb P(\exists\text{ bad }m\text{-set})\le \binom{n}{m}p_m.
\]

\[
\textit{Proof.}
\]
Union bound over all \(m\)-subsets.
\(\square\)

### Proposition BJ2 (Constant extraction reduction)
If one proves matching thresholds
\[
\binom{n}{m}p_m \to 0 \quad\text{for } m\ge (c_\alpha+\varepsilon)\log n,
\]
and corresponding lower witness for
\[
m\le (c_\alpha-\varepsilon)\log n,
\]
then \(F(n,\alpha)\sim c_\alpha\log n\).

\[
\textit{Proof.}
\]
Upper and lower asymptotic inequalities squeeze \(F(n,\alpha)/\log n\) to
\(c_\alpha\).
\(\square\)
