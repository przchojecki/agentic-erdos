## Deeper Proof Program (A => B => C)

### Target
For fixed \(k\), determine whether eventually
\[
F(N+k)\le F(N)+1
\]
for Sidon extremal function \(F(N)\), and identify the true increment law.

### Step A: Stability of extremizers
Show near-extremal Sidon sets in \([1,N]\) have rigid local structure near the
top end.

### Step B: Extension obstruction
Prove adding \(k\) new integers cannot increase Sidon size by more than \(1\)
once \(N\) is large, due to collision constraints.

### Step C: Uniform in fixed \(k\)
Make constants effective to obtain eventual threshold \(N_0(k)\).

### Concrete Blocking Lemma (Most Critical)
Need a local-collision counting theorem controlling how many fresh elements can
be adjoined in a short interval without creating repeated sums.

## LaTeX Proofs for A => B => C Skeleton

### Proposition BD1 (Subadditive-style bound)
For any \(N,k\),
\[
F(N+k)\le F(N)+k.
\]

\[
\textit{Proof.}
\]
Take a Sidon set \(A\subset[1,N+k]\). At most \(k\) elements lie in
\((N,N+k]\); removing them leaves a Sidon subset in \([1,N]\) of size at most
\(F(N)\).
\(\square\)

### Proposition BD2 (Reduction to one-step extension bound)
If one proves for fixed \(k\) and large \(N\):
\[
F(N+j+1)\le F(N+j)+\mathbf 1_{j=0}\quad (0\le j<k),
\]
then \(F(N+k)\le F(N)+1\).

\[
\textit{Proof.}
\]
Sum the inequalities over \(j=0,\dots,k-1\).
\(\square\)
