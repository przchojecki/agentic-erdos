# EP-1071

## Problem Statement
Two-part status in this note context:
1. finite maximal family of pairwise disjoint unit segments in the unit square,
2. existence of a region admitting a countably infinite maximal disjoint family.

## Literature
- First finite subquestion is known to have a positive answer.
- The countably infinite maximal-family variant remains open.

## Our Approaches / What Is Proven
- Replaced stub with a standalone deep geometric-combinatorial experiment for the finite side:
  - random candidate generation of unit segments inside $[0,1]^2$,
  - exact conflict-graph MIS via bounded branch-and-bound,
  - large number of independent trials,
  - sampled extension stress test to probe near-maximality.
- Deep run scale:
  - $4000$ trials,
  - $348$ candidates per trial,
  - runtime about $113$ seconds.

## Computation-Guided Observations
- Best finite packing found in this candidate model was size $26$ among $348$ candidates.
- Across probes, many trial winners were still extendable by additional random segments, indicating this random candidate pool is not globally maximal in geometric space.
- This is consistent with the finite question being positive but does not address the open countably-infinite maximal-family problem.

## Status
- First subquestion: solved in background.
- Second subquestion: open.
- Added deep standalone heavy finite-search profile for the finite side.

## Added Proof Layer (No New Computation)

### Closed Subquestion
The finite maximal-family existence question is solved positively in the cited
background.

### Remaining Open Subquestion
Existence of a region supporting a **countably infinite** maximal disjoint unit
segment family remains open.

### Blocking Lemma
- Need a geometric compactness/extension principle that either forces
  extendability of every infinite disjoint family, or constructs a rigid
  infinite maximal one.

## Deeper Proof Program (A => B => C)

### Target
Resolve whether some region admits a countably infinite maximal family of
pairwise disjoint unit segments.

### Step A: Compactness Dichotomy
Prove an infinite disjoint family in bounded region has either:
1. an accumulation geometry enabling extension, or
2. a rigid limit-configuration class.

### Step B: Extension Criterion
Show every family in class (1) is not maximal (construct one extra disjoint
unit segment).

### Step C: Rigidity Resolution
Either construct explicit maximal example from class (2), or prove class (2)
cannot occur.

### Concrete Blocking Lemma (Most Critical)
Need a local extension lemma: if the complement region around accumulation
directions has positive angular-width corridor, then one can insert an
additional disjoint unit segment.
