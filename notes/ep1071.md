# EP-1071

## Problem Statement
Two-part status in this note context:
1. finite maximal family of pairwise disjoint unit segments in the unit square,
2. existence of a region admitting a countably infinite maximal disjoint family.

## Literature
- First finite subquestion is known to have a positive answer.
- The countably infinite maximal-family variant remains open.

## Our Approaches / What Is Proven
- Replaced stub with a standalone deep geometric-combinatorial experiment for the finite side:
  - random candidate generation of unit segments inside $[0,1]^2$,
  - exact conflict-graph MIS via bounded branch-and-bound,
  - large number of independent trials,
  - sampled extension stress test to probe near-maximality.
- Deep run scale:
  - $4000$ trials,
  - $348$ candidates per trial,
  - runtime about $113$ seconds.

## Computation-Guided Observations
- Best finite packing found in this candidate model was size $26$ among $348$ candidates.
- Across probes, many trial winners were still extendable by additional random segments, indicating this random candidate pool is not globally maximal in geometric space.
- This is consistent with the finite question being positive but does not address the open countably-infinite maximal-family problem.

## Status
- First subquestion: solved in background.
- Second subquestion: open.
- Added deep standalone heavy finite-search profile for the finite side.
