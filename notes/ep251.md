## Deeper Proof Program (A => B => C)

### Target
Prove irrationality (or stronger) of
\[
\sum_{n\ge1}\frac{p_n}{2^n},
\]
where \(p_n\) is the \(n\)-th prime.

### Step A: Prime-growth decomposition
Split into smooth main approximation using \(p_n\sim n\log n\) and fluctuation
error.

### Step B: Binary-expansion irregularity
Show prime fluctuations induce non-periodic binary tail behavior incompatible with
rationality.

### Step C: Diophantine conclusion
Convert tail irregularity into irrationality/transcendence criterion.

### Concrete Blocking Lemma (Most Critical)
Need a quantitative lower bound on binary-tail complexity generated by the prime
sequence that cannot occur for rational numbers.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CW1 (Tail estimate)
For \(T_N=\sum_{n>N} p_n/2^n\),
\[
T_N\ll \frac{N\log N}{2^N}.
\]

\[
\textit{Proof.}
\]
Use \(p_n\ll n\log n\) and compare to weighted geometric tail.
\(\square\)

### Proposition CW2 (Rationality obstruction reduction)
If one proves infinitely many \(N\) such that the first \(m(N)\) bits after
position \(N\) of partial sums cannot match any eventually periodic pattern with
period \(<P(N)\), where \(m(N)\gg \log P(N)\), then the full sum is irrational.

\[
\textit{Proof.}
\]
Rational numbers have eventually periodic binary expansions with fixed period; a
violating infinite sequence of windows contradicts this.
\(\square\)
