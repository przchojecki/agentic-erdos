# EP-1017

## Proof Attempts and Literature Notes

### Working statement
For \(K_4\)-free graphs with edge count above \(n^2/4\), study guaranteed edge-disjoint triangle packing.

### Exact reformulation
Let \(\nu_3(G)\) be the maximum number of edge-disjoint triangles in \(G\).
The target statement in this regime is
\[
e(G)\ge \left\lfloor \frac{n^2}{4}\right\rfloor + q
\quad\Longrightarrow\quad
\nu_3(G)\ge q
\]
for \(K_4\)-free \(G\).

So the problem is exactly a lower bound on \(\nu_3\) as a function of edge excess above the Turán threshold.

### Witness principle
For fixed \(n,q\), any explicit \(K_4\)-free graph \(G\) with
\[
e(G)=\left\lfloor\frac{n^2}{4}\right\rfloor+q
\]
and a certified packing of \(q\) edge-disjoint triangles is a rigorous instance-level confirmation.

### Computation context
The constructions tested in this project satisfy \(\nu_3(G)\ge q\) for sampled \((n,q)\), consistent with the known \(K_4\)-free phenomenon, but not a full general proof.
