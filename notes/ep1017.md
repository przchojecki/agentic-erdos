# EP-1017

## Proof Attempts and Literature Notes

### Working statement
For \(K_4\)-free graphs with edge count above \(n^2/4\), study guaranteed edge-disjoint triangle packing.

### Exact reformulation
Let \(\nu_3(G)\) be the maximum number of edge-disjoint triangles in \(G\).
The target statement in this regime is
\[
e(G)\ge \left\lfloor \frac{n^2}{4}\right\rfloor + q
\quad\Longrightarrow\quad
\nu_3(G)\ge q
\]
for \(K_4\)-free \(G\).

So the problem is exactly a lower bound on \(\nu_3\) as a function of edge excess above the Turán threshold.

### Witness principle
For fixed \(n,q\), any explicit \(K_4\)-free graph \(G\) with
\[
e(G)=\left\lfloor\frac{n^2}{4}\right\rfloor+q
\]
and a certified packing of \(q\) edge-disjoint triangles is a rigorous instance-level confirmation.

### Computation context
The constructions tested in this project satisfy \(\nu_3(G)\ge q\) for sampled \((n,q)\), consistent with the known \(K_4\)-free phenomenon, but not a full general proof.

### Explicit proved construction (deterministic)
Let \(A,B\) be disjoint vertex sets, \(|A|=a, |B|=b\), and start from \(K_{a,b}\).
Choose \(q\) disjoint edges (a matching) inside \(A\), and add exactly those \(q\) edges.
Call the resulting graph \(G_{a,b,q}\).

Assume \(q\le \min(a,b)\).

1. \(G_{a,b,q}\) is \(K_4\)-free.
- A \(K_4\) cannot use 3 vertices from \(A\), since inside \(A\) only matching edges exist.
- A \(K_4\) cannot use 2 vertices from \(B\), since \(B\) is independent.
So no 4-clique exists.

2. Edge count:
\[
e(G_{a,b,q})=ab+q.
\]
If \(a,b\) are balanced (\(a+b=n\), \(ab=\lfloor n^2/4\rfloor\)), then
\[
e(G_{a,b,q})=\left\lfloor\frac{n^2}{4}\right\rfloor+q.
\]

3. Triangle packing lower bound:
Each added matching edge \(u_iv_i\subset A\) together with any \(w\in B\) forms a triangle \(u_iv_iw\).
Choose distinct \(w_i\in B\) for \(i=1,\dots,q\) (possible since \(q\le b\)).
Then triangles
\[
T_i:=u_iv_iw_i
\]
are pairwise edge-disjoint.
Hence
\[
\nu_3(G_{a,b,q})\ge q.
\]

So this gives a fully rigorous infinite family of \(K_4\)-free graphs at edge level \(\lfloor n^2/4\rfloor+q\) with guaranteed \(q\) edge-disjoint triangles.
