## Deeper Proof Program (A => B => C)

### Target
Prove irrationality of
\[
S=\sum_{n\ge2}\frac1{n!-1}.
\]

### Step A: Rational approximation framework
Construct convergents/linear forms
\[
L_M = A_M S-B_M
\]
with explicit denominator control from \(\prod_{n\le M}(n!-1)\).

### Step B: Lower-vs-upper bound separation
Show \(L_M\neq0\) and
\[
0<|L_M|<U_M,\qquad |L_M|\ge L_M^{\text{(arith)}}
\]
with \(U_M/L_M^{\text{(arith)}}\to0\) under rationality assumption.

### Step C: Contradiction
Conclude rationality impossible, hence \(S\) irrational.

### Concrete Blocking Lemma (Most Critical)
Need a gcd-cancellation bound for shifted factorial factors:
\[
\gcd\!\Big(\prod_{n\le M}(n!-1),\,\prod_{n\le M'}(n!-1)\Big)
\]
must be small enough to preserve exponential gap in Step B.

## LaTeX Proofs for A => B => C Skeleton

### Proposition T1 (Interval exclusion implication)
If for every \(Q\) there exists \(M\) and rational interval
\[
I_M=[S_M-\epsilon_M,\ S_M+\epsilon_M]
\]
containing \(S\), such that no rational \(p/q\in I_M\) with \(1\le q\le Q\),
then \(S\) is irrational.

\[
\textit{Proof.}
\]
If \(S=p_0/q_0\) rational, choose \(Q\ge q_0\); then \(p_0/q_0\in I_M\), contradiction.
\(\square\)

### Proposition T2 (Linear-form contradiction schema)
If one constructs nonzero integers \(A_M,B_M\) with
\[
0<|A_MS-B_M|<\eta_M,\qquad \eta_M\to0,
\]
and also a rationality-implied bound
\[
|A_MS-B_M|\ge c_M,\quad c_M/\eta_M\to\infty,
\]
then \(S\) is irrational.

\[
\textit{Proof.}
\]
Under rationality both inequalities must hold simultaneously for large \(M\),
impossible when \(c_M>\eta_M\).
\(\square\)
