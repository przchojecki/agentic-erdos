# EP-117 partial resolution

## Statement
Let `h(n)` be the minimum number of Abelian subgroups needed to cover any group
where every subset of size `>n` contains two commuting distinct elements.
Estimate `h(n)`.

## What is resolved
Background cites Pyber (and earlier lower-bound work) proving constants
`c2>c1>1` with

`c1^n < h(n) < c2^n`.

Hence the growth type is known to be exponential in `n`.

## What remains open in this note
The exact asymptotic rate (for example, sharp exponential base or finer formula)
is not determined by the provided references.

## Status
- exponential-scale growth: resolved.
- sharp asymptotics/constants: unresolved.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AH1 (Graph-theoretic reformulation)
Let \(\Gamma_G\) be the noncommuting graph of a finite group \(G\). Then an
abelian-subgroup cover of \(G\) corresponds to a clique cover of the complement
\(\overline{\Gamma_G}\).

\[
\textit{Proof.}
\]
Vertices commute exactly when adjacent in \(\overline{\Gamma_G}\). Any abelian
subgroup induces a clique in \(\overline{\Gamma_G}\), and covering elements by
abelian subgroups is equivalent to covering vertices by such cliques.
\(\square\)

### Proposition AH2 (Reduction to extremal graph parameter)
If one proves, for all admissible \(G\),
\[
c_1^n \le \theta(\overline{\Gamma_G}) \le c_2^n
\]
with matching base \(c_1=c_2\), then \(h(n)\) has a sharp exponential rate.

\[
\textit{Proof.}
\]
By AH1, \(h(n)\) is exactly the extremal clique-cover number over the class.
Matching exponentials determine the base.
\(\square\)
