# EP-1120 deeper attempt

## Statement
For monic `f` with roots in unit disk, let `E={z:|f(z)|<=1}`. What is the shortest path length
in `E` from `0` to `|z|=1` in worst case over degree `n`?

## Computation in this attempt
Random-root finite model with grid shortest-path proxy.

Data file:
- `data/ep1120_random_path_scan.json`

Observed (degrees 2..12, 120 trials each):
- all trials found a path in the grid model,
- shortest lengths stayed near `~1.02` in this random family.

## Interpretation
This only probes typical random instances and does not target extremal worst-case polynomials.
No asymptotic lower/upper law is proved here.

## Deeper Proof Program (A => B => C)

### Target
Determine the worst-case shortest path length in
\[
E=\{z:\ |f(z)|\le1\}
\]
from \(0\) to \(|z|=1\) for monic degree-\(n\) polynomials with roots in \(\overline{\mathbb D}\).

### A. Concrete Blocking Lemma (Most Critical)

Establish a geometric bottleneck lemma: any extremal \(E\) contains a quasigeodesic channel from \(0\) to \(\partial\mathbb D\) with length bounded by \(C\) times a conformal modulus quantity that is uniformly controlled in \(n\).

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (sublevel connectedness near 0)
Since \(0\in E\) in the normalized setup, the connected component \(E_0\) containing \(0\) governs feasibility of paths to \(|z|=1\).

**Proof.**
Any admissible path from \(0\) lies in \(E_0\); if \(E_0\) intersects \(\partial\mathbb D\), a path exists by connectedness. \(\square\)

#### Lemma B2 (length via conformal metric bound)
If \(E_0\) admits conformal map to a standard domain with controlled distortion, Euclidean path length is bounded by integrating pullback metric along a canonical curve.

**Proof.**
Length transforms by \(|\phi'|\); distortion bounds give comparability between canonical and pulled-back lengths. \(\square\)

#### Lemma B3 (A implies uniform upper law)
If Lemma A gives uniform modulus control over extremal candidates, then worst-case shortest-path length is uniformly bounded (or has the predicted asymptotic law).

**Proof.**
Apply Lemma A channel bound and optimize over all admissible \(f\). \(\square\)

### C. Main unresolved point

The missing piece is a sharp extremal description of worst-case lemniscate geometry under root-in-disk constraints.
