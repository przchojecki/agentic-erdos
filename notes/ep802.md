# EP-802

## Problem Statement
Is it true that every \(K_r\)-free graph on \(n\) vertices with average degree \(t\) has an independent set of size
\[
\gg_r \frac{\log t}{t}n?
\]

## Literature
- Special cases are known (notably \(r=3\) and stronger local-structure hypotheses).
- Full general \(K_r\)-free case remains open.

## Our Approaches / What Is Proven
- Built finite proxy generator for \(K_r\)-free graphs (random + pruning), then measured:
  - average degree \(t\),
  - greedy independent-set size \(\alpha\),
  - ratio \(\alpha / ((\log t)/t\cdot n)\).
- Ran for \(r=4\), \(n=40,60,80\), multiple density seeds.

## Computation-Guided Observations
- In all tested settings, the proxy ratio stayed comfortably above 1 (often around 1.5-2.3), supporting the target scale in these finite experiments.
- Very small \(t\) regimes are numerically unstable for this ratio because \(\log t\) is small.
- This is heuristic finite evidence only, not a proof.
