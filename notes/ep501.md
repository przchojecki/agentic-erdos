## Statement
Given a map `x \mapsto A_x` with each `A_x` bounded and of outer measure `<1`,
must there exist an infinite independent set (or at least one of size `3` under
additional closedness assumptions)?

## What is resolved from background
For the closed-case variant, a stronger theorem is cited: existence of an
infinite independent set, hence in particular of size `3`.

## What remains open in this note
The unrestricted formulation is set-theoretically delicate in the cited
background (CH-model obstruction vs positive closed-case theorem).

## Deeper Proof Program (A => B => C)

### Target
Strengthen from closed-case theorem to broader classes, and isolate exact obstruction for unrestricted formulation.

### A. Concrete Blocking Lemma (Most Critical)

Prove a compactness-transfer lemma: if every finite subfamily of closed bounded sets admits an independent set of size \(r\), then the full family admits an infinite independent set via diagonal compactness.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (finite-to-infinite reduction template)
Suppose for each \(m\) there exists an independent set of size \(m\). Then one can extract an infinite independent sequence if constraints are closed under coordinatewise limits.

**Proof.**
Choose \(m\)-tuples with increasing \(m\), pass to convergent subsequences coordinatewise using boundedness/compactness, and use closedness to preserve avoidance constraints. \(\square\)

#### Lemma B2 (size-3 corollary)
An infinite independent set immediately implies existence of independent set of size \(3\).

**Proof.**
Take any three distinct points from the infinite set. \(\square\)

#### Lemma B3 (A implies closed-case theorem route)
If A holds for the closed bounded family, B1 and B2 recover the stated positive closed-case conclusion.

**Proof.**
Apply A to get finite independent sets of all sizes; use B1 to pass to infinite, then B2. \(\square\)

### C. Main unresolved point

The bottleneck is extending compactness-based arguments beyond closed/bounded families where limit preservation may fail.
