# EP-1095

## Problem Statement
Let $g(k)>k+1$ be the smallest $n$ such that every prime divisor of
\[
\binom{n}{k}
\]
is greater than $k$.
Estimate $g(k)$.

## Literature
- Classical bounds show very rapid growth and strong irregularity.
- Heuristic viewpoint suggests
\[
\log g(k)\asymp \frac{k}{\log k}
\]
with substantial oscillation.

## Our Approaches / What Is Proven
- Replaced stub with standalone exact search (`scripts/ep1095.mjs`) using Lucas-theorem divisibility tests:
  - for each prime $p\le k$, test whether $p\mid\binom{n}{k}$ via base-$p$ digit comparison,
  - accept the first $n$ where all such tests are nondivisibility.
- This is exact for all found values within the chosen cap.

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- $K_{\max}=90$,
- search cap $N_{\max}=2{,}500{,}000$,
- per-$k$ budget 14s.

Workload scale:
- about $121{,}782{,}980$ tested $n$-values,
- about $137{,}642{,}288$ Lucas prime checks.

Results summary:
- found exact $g(k)$ values for 42 values of $k\le 90$,
- unresolved within cap for 47 values,
- unresolved list starts at $k=41$ and includes most larger $k$ values in this window.

Observed finite profile remains highly irregular in $k$, consistent with known oscillatory heuristics.
