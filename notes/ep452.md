## Deeper Proof Program (A => B => C)

### Target
Determine maximal interval length in \([x,2x]\) on which \(\omega(n)>\log\log n\) for all \(n\).

### A. Concrete Blocking Lemma (Most Critical)

Prove a two-sided cluster theorem: local fluctuations of \(\omega(n)\) around \(\log\log n\) admit Gaussian-like block behavior down to interval length \(L(x)\), yielding both lower and upper bounds for maximal run length.

### B. Reduction Lemmas (with proofs)

#### Lemma B1 (threshold event definition)
Let \(E_x(n):=\mathbf 1_{\omega(n)>\log\log x}\) for \(n\in[x,2x]\). Maximal run length is longest consecutive block of 1s in \(E_x\).

**Proof.**
Direct encoding of condition into binary sequence. \(\square\)

#### Lemma B2 (density baseline)
Known density of \(E_x=1\) is asymptotically \(1/2\).

**Proof.**
By Erdős-Kac type normal order centered at \(\log\log x\), strict-above-center probability tends to \(1/2\). \(\square\)

#### Lemma B3 (A implies extremal-run asymptotic)
If A gives dependence-decay and tail estimates for \(E_x\) blocks, maximal run length follows by standard extreme-value analysis.

**Proof.**
Apply block maxima method to Bernoulli-like process with controlled dependence. \(\square\)

### C. Main unresolved point

The bottleneck is quantifying short-range dependence of \(\omega(n)\) strongly enough to control longest-run extremes, not just average density.
