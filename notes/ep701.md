# EP-701

## Problem Statement
Let \(\mathcal F\) be a set family closed under subsets (a downset): if \(B\subseteq A\in\mathcal F\), then \(B\in\mathcal F\).
Is there always an element \(x\) such that every intersecting subfamily \(\mathcal F'\subseteq\mathcal F\) satisfies
\[
|\mathcal F'|\le |\{A\in\mathcal F:x\in A\}|?
\]

## Literature
- This is a Chv\'atal-type star-dominance question for hereditary/downset families.
- Important special cases are known; the full statement remains open.

## Our Approaches / What Is Proven
- Built exact branch-and-bound solver for maximum intersecting subfamily size in finite downsets.
- Generated many random downsets (controlled family size), then compared:
  - exact \(\max\{|\mathcal F'|: \mathcal F'\text{ intersecting}\}\),
  - largest star \(\max_x |\{A\in\mathcal F:x\in A\}|\).

## Computation-Guided Observations
- Deep run: \(n=5\) to \(9\), 900 samples per \(n\), max family size 36.
- No violations found (all tested families had star size \(\ge\) exact max intersecting size).
- This is positive finite evidence only; it is not a proof of the full infinite statement.
