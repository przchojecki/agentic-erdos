## Deeper Proof Program (A => B => C)

### Target
For fixed finite \(n\), prove
\[
\mathfrak c \to (\beta,n)^3_2
\]
for every countable ordinal \(\beta\).

### Step A: Canonization on a large family
Given a 2-coloring of \([\mathbb R]^3\), extract a size-\(\mathfrak c\) subset
on which color classes are controlled by a finite list of order/topological
types.

### Step B: Transfinite homogeneous build
Inside one canonical class, recursively build a countable ordinal pattern with
all triples of one color while preserving enough branching for the finite side
condition \(n\).

### Step C: Lift to arbitrary countable \(\beta\)
Use countable decomposition \(\beta=\sup_m \beta_m\) and diagonal fusion to
assemble the full \(\beta\)-type monochromatic set.

### Concrete Blocking Lemma (Most Critical)
Need a 3-uniform canonization lemma at size \(\mathfrak c\) strong enough to
support transfinite fusion; finite hypergraph regularization is too weak.

## LaTeX Proofs for A => B => C Skeleton

### Proposition G1 (Canonization implies transfinite construction)
Assume Step A provides \(X\subseteq\mathbb R\), \(|X|=\mathfrak c\), and a
canonical description of the coloring on \([X]^3\) with a branch class
\(\mathcal C\) closed under countable initial-segment extensions. Then for each
finite \(n\) there exists \(Y\subseteq X\) of order type \(\omega\cdot n\) such
that all triples from \(Y\) in \(\mathcal C\) are monochromatic.

\[
\textit{Proof sketch.}
\]
Build \(Y\) by blocks \(Y_0,\dots,Y_{n-1}\), each of type \(\omega\), using
closure of \(\mathcal C\) to preserve color constraints for mixed triples across
previous blocks. Countable support at each stage ensures availability of choices.
\(\square\)

### Proposition G2 (Fusion yields all countable ordinals)
Assume Proposition G1 holds uniformly for all finite block lengths, and the
construction is monotone under end-extension. Then
\[
\forall \beta<\omega_1,\quad \mathfrak c\to(\beta,n)^3_2.
\]

\[
\textit{Proof sketch.}
\]
Choose an increasing sequence \((\beta_m)\) of finite-block approximants with
\(\sup_m\beta_m=\beta\). Apply G1 at stage \(m\), fuse by end-extension
monotonicity, and take the union.
\(\square\)
