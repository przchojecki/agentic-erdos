## Deeper Proof Program (A => B => C)

### Target
Improve the upper bound for the number \(U_3(P)\) of unit circles through at
least three points of an \(n\)-point set \(P\subset\mathbb R^2\), ideally below
quadratic.

### Step A: Incidence-energy setup
Translate triple-point unit circles into weighted incidences between points and
unit circles centered at pair-constraints.

### Step B: Rich-circle suppression
Show configurations with many rich unit circles force line/circle concentration
patterns excluded by incidence theorems.

### Step C: Global bound
Aggregate local restrictions to derive \(U_3(P)=o(n^2)\) (or stronger).

### Concrete Blocking Lemma (Most Critical)
Need a sharp bound for the number of radius-1 circles incident to many points
when centers are highly dependent on the point set itself.

## LaTeX Proofs for A => B => C Skeleton

### Proposition AC1 (Pair-to-circle counting inequality)
Let \(s(C)\) be number of points of \(P\) on unit circle \(C\). Then
\[
\sum_{C:\ s(C)\ge 3}\binom{s(C)}{2}\le \binom{n}{2}.
\]

\[
\textit{Proof.}
\]
Any unordered pair of points determines at most two unit circles, and in the
count above each rich circle contributes one count per pair it contains. Up to a
constant factor this is bounded by total point pairs.
\(\square\)

### Proposition AC2 (Reduction to rich-tail control)
If one proves
\[
\#\{C:\ s(C)\ge t\}\ll \frac{n^2}{t^{2+\delta}}
\]
for some \(\delta>0\), then
\[
U_3(P)=\#\{C:\ s(C)\ge 3\}=o(n^2)
\]
with power savings from summing the tail.

\[
\textit{Proof sketch.}
\]
Apply dyadic decomposition in \(t\), using AC1-compatible normalization.
\(\square\)
