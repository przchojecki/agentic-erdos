# EP-640

## Problem Statement
Does there exist a function \(f\) such that for all \(k\ge 3\), every finite graph
with \(\chi(G)\ge f(k)\) contains an odd cycle whose vertex set spans a subgraph of
chromatic number at least \(k\)?

## Literature
- This is an odd-cycle/chromatic amplification question: high global chromatic number versus high chromatic complexity already visible on vertices of one odd cycle.
- No definitive function \(f\) is recorded in the local dataset entry.

## Our Approaches / What Is Proven
- Used Mycielski graphs \(M^t(C_5)\), which are triangle-free with known chromatic numbers \(3+t\), as structured stress tests.
- For each \(M^t(C_5)\), extracted an odd cycle and computed chromatic upper bounds of the induced subgraph on its vertices.

## Computation-Guided Observations
- For \(t=0\) to \(5\), global chromatic level rises from 3 to 8 as expected.
- In all tested cases, found odd cycles had 5 vertices and induced chromatic upper bound 3.
- This finite evidence points to potential separation: global \(\chi\) can be large while cycle-vertex-induced chromatic complexity stays low on sampled odd cycles.

## Status
- Open.
- The script gives explicit high-\(\chi\) family probes but not a proof for or against existence of such \(f\).
