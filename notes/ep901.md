# EP-901

## Problem Statement
Let $m(n)$ denote the minimum number of edges in an $n$-uniform hypergraph that is not 2-colorable. Determine the correct growth/order of $m(n)$.

## Literature
- Classical lower bound: $m(n)\ge 2^{n-1}$ (probabilistic coloring argument).
- Known upper bounds are exponential in $n$ up to polynomial/exponential-factor refinements; exact asymptotic remains open.

## Our Approaches / What Is Proven
- Built a deeper multirestart greedy constructor for non-2-colorable $n$-uniform hypergraphs.
- For each $n\in\{4,5,6,7\}$, scanned several vertex counts $v$ and ran many randomized restarts with large candidate-edge sampling.
- Verified non-2-colorability by explicit elimination of all 2-colorings when a witness is found.

## Computation-Guided Observations
- Best witnesses found in this run:
  - $n=4$: $28$ edges (at $v=9$), improving earlier in-repo batch witness $30$.
  - $n=5$: $67$ edges (at $v=11$), improving earlier in-repo batch witness $92$.
  - $n=6$: $321$ edges (at $v=13$), slightly improving earlier in-repo batch witness $327$.
- For $n=7$, this run did not find a full witness under the current search budget (best remaining colorings still positive), so deeper search is still needed there.
- Ratios against $2^n$ in this run: about $1.75$ ($n=4$), $2.09$ ($n=5$), $5.02$ ($n=6$).
