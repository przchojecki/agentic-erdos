# EP-679

## Problem Statement
For \(\omega(n)\) (number of distinct prime factors), ask:
1. For each fixed \(\varepsilon>0\), are there infinitely many \(n\) such that for all sufficiently large \(k<n\),
\[
\omega(n-k) < (1+\varepsilon)\frac{\log k}{\log\log k}?
\]
2. Is the stronger bound
\[
\omega(n-k) < \frac{\log k}{\log\log k} + O(1)
\]
false?

## Literature
- Background/comments report a strong disproof pattern for the second stronger version: for all large \(n\), some \(k<n\) gives a positive secondary-term excess.

## Our Approaches / What Is Proven
- Implemented large-scale numerical stress test (up to \(n=1.2\times 10^6\)) with \(k\ge 100\), scanning many \(k\)-values per \(n\).
- Measured
\[
\max_k \left(\omega(n-k)-\frac{\log k}{\log\log k}\right).
\]

## Computation-Guided Observations
- In all tested large-\(n\) samples, the maximum gap is clearly positive (around \(2.6\) to \(3.0\) in scan).
- This numerically aligns with failure of the stronger \(+O(1)\) formulation.
- It does not settle the first, weaker \((1+\varepsilon)\)-factor question.

## Status
- Stronger second subquestion: resolved as false in background.
- First subquestion: open.

## Added Proof Layer (No New Computation)

### Closed Part
The stronger bound
\[
\omega(n-k) < \frac{\log k}{\log\log k}+O(1)
\]
for all sufficiently large \(n\) (uniformly in large \(k<n\)) is false, as
recorded in the background discussion.

### Remaining Open Part
The weaker \((1+\varepsilon)\)-factor statement remains open:
for fixed \(\varepsilon>0\), existence of infinitely many \(n\) for which all
sufficiently large \(k<n\) satisfy
\[
\omega(n-k)<(1+\varepsilon)\frac{\log k}{\log\log k}.
\]

### Blocking Lemma
- Need a uniform upper-tail control theorem for \(\omega(n-k)\) in long shifted
  windows \(k<n\), strong enough to beat the normal order by a fixed
  multiplicative margin on infinitely many \(n\).

## Deeper Proof Program (A => B => C)

### Target
For each fixed \(\varepsilon>0\), find infinitely many \(n\) such that for all
sufficiently large \(k<n\),
\[
\omega(n-k)\le (1+\varepsilon)\frac{\log k}{\log\log k}.
\]

### Step A: Uniform Short-Interval Upper Tail
Prove that for a positive-density set of centers \(n\), the whole interval
\([n-K,n)\) avoids large \(\omega\)-outliers above the \((1+\varepsilon)\)-line,
for \(K\) in a long range.

### Step B: Multi-Scale Patching
Patch Step A over dyadic \(K\)-scales so that one \(n\) controls all sufficiently
large \(k<n\), not just one window length.

### Step C: Infinite Recurrence
Show the good-center set from Step B has infinitely many elements.

### Concrete Blocking Lemma (Most Critical)
Need a shifted almost-sure upper-tail bound:
\[
\sup_{k\in [K,2K]}\left(\omega(n-k)-\frac{\log k}{\log\log k}\right)\le \varepsilon\frac{\log K}{\log\log K}
\]
for infinitely many \(n\), uniformly over large \(K\).

### Method Directions
- Large-sieve / Turán-Kubilius with uniformity in shifted windows.
- Probabilistic model + LLL to suppress simultaneous outliers across many \(k\).
- Entropy compression over prime-factor indicator fields.
