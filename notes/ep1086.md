# EP-1086

## Problem Statement
Let $g(n)$ be the maximum number of equal-area triangles that can be determined by an $n$-point set in $\mathbb{R}^2$.
Estimate $g(n)$.

## Literature
- Classical bounds (Erdős-Purdy and later improvements) place
\[
n^2\log\log n\ll g(n)\ll n^{20/9}
\]
(best listed upper bound due to Raz-Sharir).
- The exact growth exponent and sharp order remain open.

## Our Approaches / What Is Proven
- Replaced the batch stub with a standalone deep randomized search (`scripts/ep1086.mjs`).
- For each fixed $n$, sampled many distinct integer-grid point sets and computed exact area-multiplicity histograms over all nondegenerate triangles.
- Objective: maximize multiplicity of one area value (i.e., many triangles sharing the same area).

## Computation-Guided Observations
Deep run (`DEPTH=4`) used:
- box size $95$,
- per-$n$ time budget $12$ seconds,
- $n\in\{18,22,26,30,34\}$,
- total runtime $60$ seconds,
- total candidates evaluated: $813{,}352$,
- estimated triangles scanned: $1{,}541{,}210{,}476$.

Best multiplicities found in this run:
- $n=18$: 18,
- $n=22$: 16,
- $n=26$: 19,
- $n=30$: 25,
- $n=34$: 28.

Normalized values in this finite range are noisy and non-monotone, so they do not distinguish the asymptotic exponent; they provide only finite lower-bound evidence from explicit constructions.
