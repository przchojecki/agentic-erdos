# EP-1016

## Proof Attempts and Literature Notes

### Working statement
Study the minimal edge excess above \(n\) needed for pancyclic behavior in the EP-1016 formulation.

### Certificate principle (existence-type formulation)
If an explicit \(n\)-vertex pancyclic graph with \(n+h\) edges is exhibited, then the corresponding extremal existence parameter satisfies
\[
h(n)\le h.
\]
So every concrete pancyclic witness from computation is a rigorous upper-bound certificate.

### Monotonicity (safe edge-augmentation form)
If \(G\) is pancyclic, then any supergraph on the same vertex set (adding edges only) is also pancyclic, since all previously present cycles remain.

Hence once a witness exists at edge level \(n+h\), witnesses exist for all larger edge counts on that vertex set.

### Computation context
Random searches in this project produced small-\(n\) pancyclic witnesses with low excess \(h\), giving certified finite upper bounds but not asymptotic proofs.
