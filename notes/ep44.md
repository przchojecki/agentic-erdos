## Deeper Proof Program (A => B => C)

### Target
Given Sidon \(A\subset[1,N]\), determine whether one can always extend inside
\([N+1,3N]\) to a larger Sidon set of target size \((1-\varepsilon)\sqrt{3N}\)
(or the exact EP-44 threshold form).

### Step A: Local extension reservoir
Show many candidates \(x\in[N+1,3N]\) avoid all existing difference conflicts
with \(A\).

### Step B: Iterative nibble with conflict control
Add candidates in rounds while keeping Sidon constraints using sparse conflict
graph estimates.

### Step C: Reach target cardinality
Prove rounds continue until target size is reached.

### Concrete Blocking Lemma (Most Critical)
Need a uniform bound on conflict graph maximum codegree in the extension window,
strong enough for a nibble/LLL step at density close to \(\sqrt N\).

## LaTeX Proofs for A => B => C Skeleton

### Proposition L1 (Extension criterion)
If at each stage with current set \(S\) there exists \(x\in[N+1,3N]\setminus S\)
that preserves Sidon, then iterative extension terminates only at maximal Sidon
superset in the window.

\[
\textit{Proof.}
\]
Each step strictly increases \(|S|\). Finite window implies termination. By
construction, no further admissible \(x\) exists at terminal stage, so it is
maximal.
\(\square\)

### Proposition L2 (Target-size implication)
If one proves existence of admissible \(x\) whenever \(|S|<T\), then maximal set
has size at least \(T\).

\[
\textit{Proof.}
\]
Otherwise terminal maximal set would have size \(<T\), contradicting hypothesis.
\(\square\)
