# EP-196 partial

## Statement
Must every permutation of `N` contain a monotone 4-term arithmetic progression?

## Attempt in this batch
I ran randomized backtracking searches for finite permutations of `[1..N]`
that avoid 4-term AP subsequences in index order.

Data file:
- `data/ep196_permutation_avoidance_search.json`

## Result
- Full avoiding permutations were found for `N=12` and `N=14`.
- In this run, no full witness was found for `N=16,18,20,22` (best prefixes
  reached lengths `14,15,17,17` respectively).

## Hard point
Search failure is not a proof of impossibility.
The infinite permutation question remains far beyond current finite search.

## Deeper Proof Program (A => B => C)

### Target
Decide whether every infinite permutation contains a monotone 4-term arithmetic
progression pattern (in the EP-196 sense).

### Step A: Infinite-to-finite compactness
Show existence/nonexistence in infinite permutations is equivalent to arbitrarily
long finite avoiders.

### Step B: Finite extremal growth
Estimate maximal avoider length \(L(N)\) and detect bounded/unbounded behavior.

### Step C: Infinite conclusion
If \(L(N)\) is bounded, prove inevitability; if unbounded with coherent extension,
build an infinite counterexample.

### Concrete Blocking Lemma (Most Critical)
Need an extension lemma from finite avoiders to longer avoiders with controlled
pattern interactions, or a barrier showing extension must fail.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CB1 (König-lemma reduction)
If for every \(N\) there exists an \(N\)-term avoiding permutation prefix and the
prefix-extension graph is finitely branching, then an infinite avoiding
permutation exists.

\[
\textit{Proof.}
\]
Apply König's infinity lemma to the tree of avoiding prefixes.
\(\square\)

### Proposition CB2 (Inevitability criterion)
If there exists \(N_0\) such that no avoiding prefix of length \(N_0\) exists,
then every infinite permutation contains the forbidden pattern.

\[
\textit{Proof.}
\]
Any infinite permutation has a first \(N_0\)-prefix; by assumption it already
contains the pattern.
\(\square\)
