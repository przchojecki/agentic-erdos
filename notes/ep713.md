# EP-713

## Problem Statement
For every bipartite graph \(G\), is it true that there exist \(\alpha\in[1,2)\), \(c>0\) with
\[
\mathrm{ex}(n;G)\sim c n^{\alpha}?
\]
Must \(\alpha\) be rational?

## Literature
- Many individual bipartite families have known exponents, but a full universal asymptotic classification is open.

## Our Approaches / What Is Proven
- Used deeper greedy constructions in the model case \(G=C_4\) to track finite exponent proxies.
- Computed best found \(C_4\)-free edge counts across larger \(n\) and multiple restarts.

## Computation-Guided Observations
- Deep run with \(n=40\) to \(200\) (36 restarts each) gave \(e/n^{3/2}\) roughly in the 0.90-1.00 range.
- Local log-log slopes stayed around 1.45-1.64, consistent with the classical \(3/2\)-type benchmark for \(C_4\).
- This supports known model behavior but does not address all bipartite \(G\).
