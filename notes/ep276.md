## Deeper Proof Program (A => B => C)

### Target
Construct (or rule out) Lucas-type sequences with all terms composite while
avoiding a trivial global common-factor mechanism.

### Step A: Covering-congruence architecture
Assign congruence classes of indices forcing divisibility by selected primes for
each term \(u_n\).

### Step B: Primitive-prime management
Control emergence of primitive prime divisors so covering remains valid at all
indices.

### Step C: Global gcd collapse check
Verify forced divisibility does not imply a fixed common divisor across all terms.

### Concrete Blocking Lemma (Most Critical)
Need a compatibility lemma between covering sets of indices and Lucas
recurrence residues ensuring every index is covered while \(\gcd(u_1,\dots,u_n)\)
does not stabilize above \(1\).

## Lemma Chain (with Proofs)

### Lemma 1 (Finite cover implies finite compositeness)
If index set \(\{1,\dots,N\}\) is covered by congruence classes each forcing
\(u_n\equiv 0\pmod{q_j}\), then \(u_1,\dots,u_N\) are composite whenever
\(|u_n|>q_j\) on the covered classes.

\[
\textit{Proof.}
\]
For each \(n\le N\), choose a class containing \(n\). Then \(q_j\mid u_n\). If
\(|u_n|>q_j>1\), \(u_n\) is composite.
\(\square\)

### Lemma 2 (Prefix gcd collapse criterion)
If two indices \(m,n\) satisfy \(\gcd(u_m,u_n)=1\), then
\[
\gcd(u_1,\dots,u_{\max\{m,n\}})=1.
\]

\[
\textit{Proof.}
\]
Any common divisor of the whole prefix divides both \(u_m\) and \(u_n\), hence is
\(1\).
\(\square\)

### Lemma 3 (Reduction to infinite construction)
If one can build arbitrarily long prefixes satisfying Lemma 1 and with at least
one coprime pair from Lemma 2 in each prefix, then a diagonal construction gives
an infinite candidate with composite terms and no persistent gcd \(>1\).

\[
\textit{Proof sketch.}
\]
Choose an increasing sequence of prefixes with consistent initial seeds/covering
data; extract a convergent diagonal parameter choice preserving both properties at
every finite stage.
\(\square\)
