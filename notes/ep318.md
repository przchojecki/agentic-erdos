# EP-318 partial resolution

## Statement split
For
`A \subseteq \mathbb{N}` and nonconstant `f:A -> {-1,1}`:
1. If `A` is an infinite arithmetic progression, must there exist finite nonempty
   `S \subset A` with `sum_{n in S} f(n)/n = 0`?
2. What if `A` is an arbitrary set of positive density?
3. What if `A` is the set of squares excluding `1`?

## First question (arithmetic progressions): yes
The background records Sattler's result that every arithmetic progression has this property.

## Second question (positive density): false
As noted in the background, this fails for sets `A` containing exactly one even number.
A concrete obstruction:
- Let `e` be the unique even element of `A`.
- Define `f(e)=+1` and `f(a)=-1` for every odd `a in A`.

Now take any finite nonempty `S \subset A`.
- If `e \notin S`, then `sum_{n in S} f(n)/n < 0`.
- If `e \in S`, then
  `sum_{n in S} f(n)/n = 1/e - sum_{m in T} 1/m`
  with `T` a finite set of odd integers.

`sum_{m in T} 1/m` has odd denominator (in lowest terms), while `1/e` has even denominator,
so equality is impossible. Hence the sum is never `0`.

Therefore the positive-density variant is false.

## Third question (squares excluding 1)
Background says this appears open: Sattler announced a proof, but no proof appeared.

## Status
- First subquestion: resolved (yes, by cited result).
- Second subquestion: resolved (false, explicit parity counterexample family).
- Third subquestion: unresolved.

## Interpretation for the open squares variant
These computations suggest a two-regime finite behavior:
1. short prefixes can be rigid (no exact cancellation for any nonconstant signing
   in the tested range \(m\le 12\));
2. longer prefixes often admit cancellation for random signings.

This does **not** settle the infinite squares question (which asks for all
nonconstant \(f\) on all squares \(>1\)), but it sharpens the search:
potential counterexamples would need highly structured sign patterns that keep
avoiding exact cancellations across arbitrarily long prefixes.

## Added Proof Layer (No New Computation)

### Subquestion 1 (Arithmetic Progressions): Closed
This is settled positively by Sattler's theorem cited in the background.
So for every nonconstant \(f:A\to\{-1,1\}\) on an infinite arithmetic progression
\(A\), there exists finite nonempty \(S\subset A\) with
\[
\sum_{n\in S}\frac{f(n)}{n}=0.
\]

### Subquestion 2 (Positive Density): Closed Negatively
Counterexample family: let \(A\) contain exactly one even number \(e\), and all
other elements odd. Define \(f(e)=1\), \(f(a)=-1\) for odd \(a\in A\).
For finite nonempty \(S\subset A\):
- if \(e\notin S\), sum is negative;
- if \(e\in S\), then
  \[
  \sum_{n\in S}\frac{f(n)}{n}=\frac1e-\sum_{m\in T}\frac1m,\quad T\subset A\cap(2\mathbb Z+1).
  \]
The second term has odd denominator in lowest terms, \(\frac1e\) has even
denominator, so equality to \(0\) is impossible.
Hence this variant is false.

### Subquestion 3 (Squares \(>1\)): Remaining Blocker
Need either:
- a structural theorem forcing an exact reciprocal cancellation for every
  nonconstant signing on squares, or
- an explicit infinite signing that avoids zero for all finite subsets.
This is the single unresolved part of EP-318 in this note.

## Deeper Proof Program (A => B => C) for Squares Variant

### Target
Settle: for every nonconstant \(f\) on \(\{m^2:m\ge2\}\), does some finite
nonempty \(S\) satisfy
\[
\sum_{n\in S}\frac{f(n)}{n}=0\ ?
\]

### Step A: Rational-Lattice Model
Embed reciprocal-square sums into a common denominator lattice on prefixes and
study signed subset-sum coverage defects.

### Step B: Prefix-to-Infinite Transfer
Prove either:
1. every sufficiently long prefix forces zero-sum (implying positive answer), or
2. construct coherent zero-free signings on nested prefixes with compatibility
to yield an infinite counterexample.

### Step C: Compactness/Diagonal Extraction
Use compactness over signings to pass from finite-prefix behavior to infinite
statement (positive or negative).

### Concrete Blocking Lemma (Most Critical)
Need a monotonic extension lemma: if a signing is zero-free on prefix length
\(M\), quantify when it can/cannot be extended to zero-free signing on length
\(M+1\) with deficit propagation control.
