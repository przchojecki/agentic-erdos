# EP-318 partial resolution

## Statement split
For
`A \subseteq \mathbb{N}` and nonconstant `f:A -> {-1,1}`:
1. If `A` is an infinite arithmetic progression, must there exist finite nonempty
   `S \subset A` with `sum_{n in S} f(n)/n = 0`?
2. What if `A` is an arbitrary set of positive density?
3. What if `A` is the set of squares excluding `1`?

## First question (arithmetic progressions): yes
The background records Sattler's result that every arithmetic progression has this property.

## Second question (positive density): false
As noted in the background, this fails for sets `A` containing exactly one even number.
A concrete obstruction:
- Let `e` be the unique even element of `A`.
- Define `f(e)=+1` and `f(a)=-1` for every odd `a in A`.

Now take any finite nonempty `S \subset A`.
- If `e \notin S`, then `sum_{n in S} f(n)/n < 0`.
- If `e \in S`, then
  `sum_{n in S} f(n)/n = 1/e - sum_{m in T} 1/m`
  with `T` a finite set of odd integers.

`sum_{m in T} 1/m` has odd denominator (in lowest terms), while `1/e` has even denominator,
so equality is impossible. Hence the sum is never `0`.

Therefore the positive-density variant is false.

## Third question (squares excluding 1)
Background says this appears open: Sattler announced a proof, but no proof appeared.

## Status
- First subquestion: resolved (yes, by cited result).
- Second subquestion: resolved (false, explicit parity counterexample family).
- Third subquestion: unresolved.

## New deep computation (2026-03-10)

Script: `scripts/ep318.mjs`  
Method: exact meet-in-the-middle subset-sum over scaled integers for
\[
\sum_{n\in S}\frac{f(n)}{n}=0,\quad n\in\{2^2,3^2,\dots,(m+1)^2\}.
\]
For each signing on a fixed prefix, we test whether a nonempty subset gives zero.

### Exhaustive finite-prefix scan (all nonconstant signings)
For each \(m=2,\dots,12\), every nonconstant signing on
\(\{2^2,\dots,(m+1)^2\}\) was checked exactly.

Result:
- number of nonconstant signings: \(2^m-2\) (as expected),
- signings with a zero-subset: **0 for every \(m\le 12\)**.

So, up to 13 terms (\(2^2\) through \(13^2\)), there is no finite-prefix
guarantee of a zero-sum subset.

### Random deeper-prefix scan
For larger prefixes, random nonconstant signings were sampled (120 each):
- \(m=18\): \(0/120\) had a zero-subset.
- \(m=24\): \(62/120\) (\(0.5167\)).
- \(m=30\): \(72/120\) (\(0.6000\)).
- \(m=36\): \(87/120\) (\(0.7250\)).
- \(m=40\): \(99/120\) (\(0.8250\)).

Runtime: \(82.923\) seconds (exact arithmetic).

## Interpretation for the open squares variant
These computations suggest a two-regime finite behavior:
1. short prefixes can be rigid (no exact cancellation for any nonconstant signing
   in the tested range \(m\le 12\));
2. longer prefixes often admit cancellation for random signings.

This does **not** settle the infinite squares question (which asks for all
nonconstant \(f\) on all squares \(>1\)), but it sharpens the search:
potential counterexamples would need highly structured sign patterns that keep
avoiding exact cancellations across arbitrarily long prefixes.

## Counterexample-seed search extension (2026-03-10, deeper)

We added targeted witness search for long prefixes:
- \(m=36,40,42\): found zero-free nonconstant signings (all within first trial
  in this run).
- \(m=44,46\): again found zero-free nonconstant signings (first trial in this run).
- \(m=48\): in 12 exact random trials, no zero-free signing was found.

So there exist explicit signings on \(\{2^2,\dots,48^2\}\) with **no** nonempty
subset satisfying
\[
\sum_{n\in S}\frac{f(n)}{n^2}=0.
\]

Interpretation:
- this is strong finite evidence in favor of possible counterexample mechanisms;
- current boundary search suggests the landscape becomes harder by \(m\approx 48\),
  though this may be a sampling artifact;
- but it is still finite-prefix evidence only, not an infinite counterexample.
