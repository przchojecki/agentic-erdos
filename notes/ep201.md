# EP-201 partial

## Statement
Let `G_k(N)` be the largest guaranteed size of a `k`-AP-free subset inside any
`N`-element integer set. Compare `G_k(N)` with `R_k(N)` (max size in `[1..N]`).

## Attempt in this batch
I used an exact small surrogate for `k=3`:
for each `N` and `M`, exhaust all `N`-subsets of `[1..M]`, compute their largest
3-AP-free subset, and compare that worst case against exact `R_3(N)` on `[1..N]`.

Data file:
- `data/ep201_interval_model_scan.json`

## Result
For tested configurations:
- `N=6,7,8`: surrogate upper bound is `4`, exact `R_3(N)=4`.
- `N=9`: surrogate upper bound is `5`, exact `R_3(9)=5`.
So in this model and range, ratio `R_3(N)/G_3_model(N)=1`.

## Hard point
The scan is restricted (small `N`, interval model, `k=3` only).
It does not settle unrestricted `G_k(N)` or asymptotic relations to `R_k(N)`.

## Deeper Proof Program (A => B => C)

### Target
Determine the asymptotic relation between \(G_k(N)\) (guaranteed AP-free subset
inside any \(N\)-set) and \(R_k(N)\) (interval extremal benchmark).

### Step A: Worst-case set geometry
Characterize \(N\)-element integer sets minimizing largest \(k\)-AP-free subset.

### Step B: Transfer from intervals
Show interval extremizers are universal up to constant/distortion factors.

### Step C: Ratio stabilization
Prove \(G_k(N)\asymp R_k(N)\) or identify a separating family.

### Concrete Blocking Lemma (Most Critical)
Need a Freiman-type compression result preserving large AP-free subsets when
passing from arbitrary sets to near-interval models.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CE1 (Trivial comparison)
For each \(k,N\),
\[
G_k(N)\le R_k(N),
\]
since \([1,N]\) is one admissible \(N\)-element set in the minimization defining
\(G_k(N)\).

\[
\textit{Proof.}
\]
Take minimum over all \(N\)-element sets.
\(\square\)

### Proposition CE2 (Reduction to reverse inequality)
If one proves \(G_k(N)\ge c_k R_k(N)\) for some \(c_k>0\) and all large \(N\),
then \(G_k(N)\asymp R_k(N)\).

\[
\textit{Proof.}
\]
Combine with CE1.
\(\square\)
