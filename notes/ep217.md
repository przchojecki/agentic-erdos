# EP-217 partial attempt

## Route
Ran randomized geometric search on integer grids under the explicit constraints (no 3 collinear, no 4 concyclic), checking the exact multiplicity pattern `1,2,...,n-1`.

## Evidence from this batch

## Hard point
Lattice random search is a narrow finite model and does not capture full Euclidean configuration space or prove nonexistence for large `n`.

## Status
No new witness or impossibility proof; open as stated.

## Deeper Proof Program (A => B => C)

### Target
Decide existence for large \(n\) of point sets whose distance multiplicities are
exactly \(1,2,\dots,n-1\) under the stated general-position constraints.

### Step A: Multiplicity-sum consistency
Use total pair count and multiplicity profile identities to derive necessary
arithmetic conditions.

### Step B: Geometric realizability filters
Impose no-3-collinear/no-4-cocircular constraints and show they conflict with
high-multiplicity requirements beyond some scale.

### Step C: Construction or obstruction
Either build an infinite family meeting all constraints or prove finite cutoff.

### Concrete Blocking Lemma (Most Critical)
Need a realizability theorem showing prescribed full multiplicity spectrum is
incompatible with general position past a threshold.

## LaTeX Proofs for A => B => C Skeleton

### Proposition CK1 (Pair-count identity)
If multiplicities are exactly \(1,\dots,n-1\), then number of distance values is
\(n-1\), and
\[
\binom n2=\sum_{j=1}^{n-1} j = \frac{n(n-1)}2.
\]

\[
\textit{Proof.}
\]
Immediate check of consistency with total pair count.
\(\square\)

### Proposition CK2 (Reduction)
To disprove existence for large \(n\), it suffices to show any general-position
configuration with \(n-1\) distinct distances has some multiplicity missing from
\(\{1,\dots,n-1\}\).

\[
\textit{Proof.}
\]
Contradicts required exact multiplicity multiset.
\(\square\)
