## LaTeX Proofs for A => B => C Skeleton

### Proposition BP1 (Class reduction)
If every set in a class \(\mathcal C\) is Euclidean Ramsey and every Euclidean
Ramsey set belongs to \(\mathcal C\), then \(\mathcal C\) is exactly the class of
Euclidean Ramsey sets.

\[
\textit{Proof.}
\]
This is equality of classes by two inclusions.
\(\square\)

### Proposition BP2 (Counterexample-minimality reduction)
If sufficiency is proved for \(\mathcal C\), then any failure of full
characterization must come from a set outside \(\mathcal C\). Thus searching for
minimal outside-\(\mathcal C\) obstructions is a complete remaining program.

\[
\textit{Proof.}
\]
Immediate from class inclusion.
\(\square\)
