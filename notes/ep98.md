## LaTeX Proofs for A => B => C Skeleton

### Proposition W1 (Energy lower bound via Cauchy)
If \(D(P)\) is number of distinct distances for \(|P|=n\), and \(m_t\) are
distance multiplicities, then
\[
\sum_t m_t^2\ge \frac{\binom{n}{2}^2}{D(P)}.
\]

\[
\textit{Proof.}
\]
Cauchy-Schwarz on \(\sum_t m_t\).
\(\square\)

### Proposition W2 (Reduction criterion)
If one can prove for general-position \(P\):
\[
\sum_t m_t^2\le \frac{n^3}{\omega(n)}
\]
for some \(\omega(n)\to\infty\), then
\[
D(P)\ge \omega(n)\,n,
\]
hence \(h(n)/n\to\infty\).

\[
\textit{Proof.}
\]
Combine W1 with \(\binom{n}{2}\asymp n^2\).
\(\square\)
